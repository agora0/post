「小白自学系列」是我创建的另一个自我反省自我学习提升自我认知的标签，这个标签和「小白刷题」系列的本质区别就是：

  • 「小白刷题」：只针对 leetcode 著名算法题总结分析，对码工参加应试有用，对实际编程作用不大。
  • 「小白自学系列」：涉及 leetcode 算法题以外的所有范围，涉及主题广泛，更具实际操作价值。


「小白自学系列」前世今生

「小白自学系列」比 「小白刷题」更早诞生。所以虽然「小白自学系列」目前只有 15 篇内容，但基于实用性和主题广泛性，关注度还略高于「小白刷题」。

「小白自学系列」诞生之初还是本着「好记性不如烂笔头」原理，把一些遇到过的技术问题整理成文加深印象。

后来发现即便整理成文后，过段时间还是会忘的一干二净，需要自己查看自己写的整理笔记。为了方便查找，于是「小白自学系列」就诞生了。


「小白自学系列」涵盖主题

目前系列除了涉及码工界入门须知 REST API 及其相关 GraphQL 外，还有区块链自建节点挖矿教程（包含前传 linux 操作系统和 docker 基础入门介绍），前端测试框架介绍，还有各种云服务部署（涉及轻量级 heroku 和业界知名 AWS 及 Azure 介绍）

目前云服务部署篇章占去了「小白自学系列」半壁江山，主要原因：部署流程比较杂，需要记忆的点比较多，所以需要特别整理成文以供未来节省时间。本着互帮互助原理，利人利己。


「小白自学系列」经营方向

未来「小白自学系列」会推出更丰富的内容，代码届的一切入门级小白问题都有可能入选「小白自学系列」。

世界本没有路，走的人多了就有路了……Matters 本没有「小白自学系列」，需要自学的内容多了就有系列了……

万物开源，一切皆可自学……以上即「小白自学系列」的存在意义。
