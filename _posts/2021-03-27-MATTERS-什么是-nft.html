我不责怪美国的知识分子对科学和机器的时代怀有敌意。敌意会起到正面的建设性作用，而机器时代的冲击有太多需要行动和智识上抵抗的东西了。但我的确会责怪一个人对机器时代缺乏兴趣。——诺伯特·维纳，The Human Use of Human Beings

Non-Fungible Token （NFT），可以翻译成“不可替代代币”，其实存在了很长时间，但最近却因为一些新闻而成为了热点。比如世界上第一条推特的 NFT 拍卖出了三百万美元。每一次区块链/加密货币登上新闻时，必然会带来对它们的批评，以及对批评的批评，无论是来自程序员社区本身的，还是来自左翼的。不幸的是，很多批评者本身似乎不太清楚 NFT 到底是什么。这篇文章将以尽可能通俗的方式介绍一下 NFT ，为未来的讨论提供一些清晰的框架。

现实中和 NFT 最接近的东西，是可交换收藏卡。比如棒球粉丝熟悉的棒球卡牌，或者游戏角色卡牌、STEAM 卡牌，乃至邮票。

这些卡片本身几乎没有任何价值。它的全部价值来自于其他人对它的需求。这种需求源于卡片所代表的东西。比如喜欢一个球队，于是就想要收集球队中所有球员的卡牌。然而这种需求，和球员本身没有任何关系，交易卡牌不是贩卖人口。撕毁一张卡牌不会对球员造成任何影响，卡片易手也不会让球员转会。从根本上来讲，收藏和交易的卡片只是一个「符号」，而不是「符号所代表的东西」。

但并非任何符号，都可以拿来收藏和交易。比如「火星」这个名字，指示了一颗行星。但任何人只需要复制一下，就能创造出完全相同的符号，因此没有人会想要收藏或者买卖「火星」这个名字。所以收藏卡片的第一条特性就是具有一定的稀缺性。实体的东西本身就很难复制，而收藏卡片往往还会带有一定的防伪措施。从买家的角度看来，我不是在买一个可以无限制造的、相同的东西，而是一个限量的东西。这种稀缺构成了收藏、交易的价值的第一条来源。

仅仅具有稀缺性是不够的。毕竟我随手画的涂鸦也具有稀缺性，大概世界上不会有第二个人能画出来了。但并没有人愿意出钱买我的涂鸦。因为它缺乏「被需要」的特性。这种被需要性可能来自于很多因素。比如梵高的画作，人们因为其审美的价值而需要它。因此它可以成为收藏品。对于棒球卡片来说，被需求性往往是球员的名气。有的时候，被需求性来自于「收集成套」的欲望。这种欲望可以让稀缺的东西自动具有被需求性。比如我想要集齐一套邮票，而其中一张发行量很小，那我就会愿意为它支付更高的价钱。而对于没有成套的东西（比如每天都会出新的报纸），并不会因为稀缺性而被需要，而会像艺术品一样和内容相关（比如报道了某件历史性事件的报纸可能会更值钱）。

上面所介绍的概念可以说是常识，但记住这些特征对理解 NFT 很有帮助。实际上，NFT 就是任何人都可以发行的，可以代表任何东西的独一无二的卡片。一张卡片所代表的东西，可以看作是卡片上印的图案。而保证卡片独一无二的方式，可以想象成做卡片的人（注意不是做图案的人！）在卡片上签了个名。

这一概念模型有助于解释很多 NFT 相对于其他收藏品来说看似匪夷所思的性质。比如为和电子作品可以随意复制，但却可以用来收藏呢？因为可以复制的是电子作品本身，而不是指向电子作品的卡片。正如球员需要吃东西，但球员卡片不需要吃东西一样，卡片和卡片所指向的东西可以具有完全不同的性质。在这里，作品可以被复制，而卡片不行。人们所说的「我用 NFT 收藏了什么作品」指的实际上是「我收藏了某件作品的 NFT 」。只要卡片具有不可复制的性质，就可以被用来收藏。

另一个奇怪的地方是，你可以创建不属于你的作品的 NFT ，这难道不是侵权吗？或者，为什么创建的时候不检查作者是否具有艺术品的版权呢？接着卡片的比喻，当一个人在贩卖 NFT 的时候，其实并没有贩卖原来的数字艺术作品，而是在贩卖基于它创作的卡片。目前几乎没有版权法规定一个作品到底能不能被人用来 NFT （因为这种使用方式超出了通常的使用范畴）。假设使用作品创建 NFT 不违法，那么任何一个人可以创建关于任何作品的 NFT ，并且相互买卖，即使这个人本身无权买卖作品本身。记住，卡片不等于卡片所指向的东西。我没有权力买卖球员，但我可以买卖持有的球员卡片。

又有人说，如果人们可以创造不属于自己作品的 NFT ，又怎么能保证作者获利呢？上面说创造的卡片，为了让其变得独一无二，卡片的作者会进行签名。而这个签名，正是让基于同一个作品的卡片彼此不同的地方。就拿开头提到的世界上第一个推特来说，任何人都可以复制一个完全相同的图片，然后生成一个 NFT 。但你无法获得推特作者用同样的图片生成的 NFT 。因为他在这个卡片上签名了。而别人愿意出钱买的，正是「具有作者签名的 NFT 」，而不是任何其他人签名的 NFT 。因此，作者的签名完全可能成为 NFT 被需求性的重要依据，从而成为估价的重要依据。

当然，这也不排除另一种情况，就是某些非作者的签名使得 NFT 同样被需求。比如，拿某位艺术家的作品，去找另一位名人签名。那么被签名的 NFT 到底值多少钱呢？这些钱，有多少应该归属原来的作者呢？目前还没有出现这种情况，不过到时候估计会有很多讨论。

到这里，你应该已经了解了 NFT 的本质：任何人都可以制作的指向任何东西的卡片，它上面有卡片作者的签名而独一无二。作者的签名会使卡片更加值钱，相反，你我这样的路人的签名会使它一文不值。
