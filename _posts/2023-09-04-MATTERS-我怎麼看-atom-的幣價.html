前陣子跟薯伯伯閒聊，他問我怎看 ATOM 的幣價，這話題三言兩句說不清，當時我只好把他敷衍過去。現在我來還債了，談談我對 ATOM 幣價的看法。

1cm=1cm；$1≠$1

誰都知道一件事物的價格很重要，就算我經常希望大眾能把目光拉闊，關注幣價以外的層面，也不會否認這個事實。我同樣吃人間煙火，會關注價格，只是不把幣價盯得太緊，好讓自己專注在更重要的事情而已。

除了影響工作，太過關注幣價也很容易會出現盲點。不，我不是要說理念，談價值、意義等形而上的概念，純粹是說，太過把目光集中在幣價，容易忽略數量改變產生的影響，誤判資產總值或投資回報。

誰都知道，資產總值是「單位價格 x 數量」。當我們持有的是黃金、股票等資產，會著眼於單位價格，希望無需買入更多數量，資產總值就會隨著單位價格水漲船高；當我們持有現金，因為單位價格始終是 1 元，只會著眼於數量，希望越多越好。1 cm 就是 1cm，1 kg 就是 1 kg，這種理所當然的物理現象，讓大部分人產生錯覺，以為 1 元這個單位，也跟其他度量衡單位一樣，恆定不變。

「以前啲菜幾蚊斤，宜家要成幾十蚊」，是家母跟我的典型閒話家常。家母因為上述錯覺，誤以為「蚊」（元）這個衡量價值的單位固定不變，於是當菜價高了，代表菜貴了，感覺社會貪婪了，卻不太能理解，即使各種客觀條件保持不變，菜價還是會隨著年月提高，不因為甚麼，純粹是現在的 1 元，再也比不上以前的 1 元。金錢的單位雖然一直稱為「元」，但隨著貨幣增發，1 元的購買力持續減少，而這種變化，由不得你管。

這就是通貨膨脹。這個詞誰都聽過，不過民間智慧一般把通膨簡單理解為「物價提高」，而較少能反過來理解不是物品越來越貴，而是「元」這個單位的購買力持續縮小，畢竟這現象非常違反人類直覺。試想像，如果重量單位 kg 也有同樣特性，1 kg 所含的質量每年遞減，即使你保持苗條，體重數字仍會不斷增加，大家也應該轉行做健身教練了。

通貨膨脹讓分母即貨幣的總量增加，當分子即你擁有的錢數量不變，每一元的購買力就會減少，是現代社會的基本金融邏輯。雖然鈔票不斷增發，但不是發到你的口袋，「單位價格 x 數量」中的兩個參數都沒變，因此通膨不會影響個人資產總值；然而同樣的個人資產總值，購買力卻持續減少。

ATOM 式通膨

若我們把區塊鏈理解為經濟體，公鏈的原生幣就相當於民族國家的法定貨幣，兩者的宏觀經濟有很多可供互相借鏡之處。

在 Cosmos Hub 的設計中，ATOM 也跟一般國家的法定貨幣一樣會持續增發，而且通膨率不低，在 7-20% 的區間浮動，當前為 14.43%。但跟國家政府不一樣的是，持續增發的 ATOM，除了 10% 投放到社群池以公投議案分配於社群發展，其餘 90% 會直接進到人民「cosmonauts」及驗證人的錢包，但有一個先決條件：用戶需要把 ATOM 質押（stake，或稱 delegate）出去。換言之，只要選擇質押手上的幣，資產就大致不會被稀釋。

以下用一些虛構的數字為例，幫助說明：

  • 假設當前 ATOM 總發行量 1 bil，市價 1.2 鎂，即總市值 1.2 bil 鎂

  • 再假設通膨率 20%，質押率（staking ratio）66.7%，即 2/3 的幣有被質押出去，一年後，ATOM 的總發行量會增至 1.2 bil

  • 假設當時 ATOM 總市值維持在 1.2 bil 鎂不變，總市值的算式會變成 1.2 bil x 1 鎂 = 1.2 bil 鎂，即市價會下跌 16.6%

  • 換言之，當幣價上漲，或者下跌少於 16.6%，ATOM 總市值會有所上升

再以假設的數字看看個人資產總值變化：

  • 假設 Alice 在期初持有 1,000 ATOM，質押到驗證人，按比例獲得增發的 ATOM 作回報

  • 一般驗證人會收取幾個百分點的佣金以支付伺服器、存儲和帶寬等成本，為求簡化，以下假設 Alice 的驗證人佣金為 0%，忽略不算

  • 由於增發的 20% 由 66.7% ATOM 的持有人按比例獲得，因此質押回報率為 20%/66.7% = 30%，忽略複合效應，一年後 Alice 手上將會持有 1,300 ATOM

  • 如果當時 ATOM 市價跌 23%（即 1-1/1.3），Alice 的資產總值將為 1,300 x 0.77 鎂 = 1,000 鎂，維持不變

  • 換言之，如果 ATOM 上漲，或者下跌少於 23%，Alice 的資產將會升值

以上只是基本的算術，簡單得很，但大部分人，即使是 ATOM 的持幣者都不太會深究，純粹以幣價作直觀的盈虧印象，而沒持有 ATOM 的局外人，當然就更加只會以 ATOM 的價格圖表，來判斷這種資產的表現了。

這構成另一種錯覺，一種跟前面所述相反的錯覺。對於法定貨幣，我們容易陷入單位價格穩定的錯覺；對於 ATOM，反過來，我們容易陷入數量穩定的錯覺。

持分者的永續博弈

無論是否認同 ATOM 的代幣經濟設計，我們最好先了解它的初衷。

打從一開始，ATOM 就被設定成不設上限地通膨，因此，撇除投機不考慮，除非需求持續增加，或者引入銷毀（burn）機制，否則根據經濟學最基本的供需理論，幣價沒有不斷上漲的理由。不過，由於上述邏輯，只要幣價能持平，即使通膨率處於下限 7%，ATOM 質押者一年下來也會賺 10% 左右。

ATOM 的通膨率會自我調整，以達至 67% 質押率為目標動態浮動，因為提領質押需要 21 天，當質押率達標，代表沒有人能在短時間內買進 34% 的投票權去否決議案或生成新區塊，整條鏈的保安足夠高。當 ATOM 的質押率不足 67%，通膨率會逐漸提高以鼓勵用戶質押，直到上限 20%；反過來，當質押率超過 67%，通膨率將逐漸下跌，直到下限 7% 為止。

區塊鏈的大原則是假定用戶行為取決於自身利益，不能單用愛發電。處身於在高通膨代幣經濟模型，持有 ATOM 的用戶每一刻都在面對博弈，單持有而不作質押的好處是夠靈活，隨時可以變賣，無論是因為上漲而套現，還是因為下跌而逃離，總之講金不講心；但是，壞處也很明顯，就像處身法定貨幣大幅通膨的國家，持分比例每分鐘都持續被稀釋。至於選擇質押的，等同於對社群投下信心一票，是一份承擔，相應獲得的好處是持分比例不斷增加；壞處當然是無論漲跌，都要等候至少三週才可提領出來，錯過突然的升浪套利事小，在極端的情況如 LUNA 死亡螺旋，還沒來得及提領逃難，幣價就已經下跌 99.99%。

不妨粗略跟我的 ATOM 算算帳。ATOM 現時幣價 7.41 USD，是最高點的約 20%，或 2019 年初 Cosmos Hub 創世時幣價的 3 倍左右，這是最基本的參考，但我們要算的是資產總值。我的 ATOM 在 2019、2020 年分批買入，3、4 年間，複合回報讓我的持幣量增加接近一倍。不過，因為經濟需要，近一段時間我已不再以質押回報買入更多 ATOM，改為套現應付生活，錢雖然沒有很多，也勉強足夠吃飯、買書、訂閱，再用剩下的一點餘裕買了兩台掃瞄器。

另外，幾年以來質押 ATOM 也為我帶來很多空投，最大宗的是 OSMO 與 JUNO，高位時直接讓原有 ATOM 資產總值翻倍，雖然現時價格已經相當低殘，但這些都是名副其實的不勞而獲，沒甚麼可抱怨。不過，空投純屬意外，完全不是 ATOM 代幣經濟設計的一部分，有的話當作運氣感恩就好，不應該抱有任何期望，尤其是在空投文化已經式微的今天。

總的來說，過去幾年 ATOM 為我帶來不錯的回報，這是完全沒法從價格歷史圖表看出來的。然而，這是錯重點，真正的重點是 ATOM 以永續博弈的機制，讓持有者在「持幣」與「持份」之間作選擇，願意為生態放棄部分流動性，投下信心一票的持份者，雖然不一定會賺錢，但佔比一定會越來越高，也代表投票權越來越大。從個人的微觀角度，這種變化很可能小得沒法感受得到，但從整個社群的宏觀角度，這代表每生成一個區塊，「持幣者」的佔比都會減少，而「持份者」的佔比都會增加，為長時間「留下來的人」提供優勢。

我不會說 ATOM 的幣價是個偽命題，但除非是投機短炒，幣價必須連帶整個脈絡去考慮，討論才有意義。不同的密碼貨幣採用不同的代幣經濟模型，比特幣刻意設計成通縮，供應量越來越少，理論上價格會越來越高，但你不該期望靠它賺取平穩回報；以太幣沒上限慢速通膨，但從 2021 年起加入銷毀機制，當後者幅度大於前者便會進入通縮，也正是當前的情況，此外，也可通過質押賺取約 5% 年回報。至於 ATOM 的高通膨設計，演繹的是跟兩位大佬截然不同的思路，不一定是投資者恩物，卻把「持份」（stake）的意義，演繹得淋漓盡致。
