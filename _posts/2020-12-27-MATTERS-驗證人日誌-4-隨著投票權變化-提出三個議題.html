現在流通的 LIKE 總量約為 1.01b 顆，33% 是一個基本門檻，而理想的委託率是 67%，委託率的變動狀況如下：

  • 10/15：18.65%
  • 12/06：30.97%
  • 12/27：32.14%

可以看到三週過後，現在委託率又微幅增加了一些，但距離 67% 還有蠻大的差距，可能要仰賴更高的通膨來讓鏈上治理更安全。

委託率固然要夠高，但...

最近也在思考一個問題，LIKE 作為多用途通證，初期的好處固然是接受門檻低，但一旦 67% 的 LIKE 都鎖在委託，流動性便會變低。

以 cosmos 來說，這個幣的用途原本就是為了治理存在，所以大部分都被拿去委託是理所當然，但 LIKE 除了是治理幣，同時也是支付打賞的貨幣，如果「治理」這個用途尚未獲得大家的認可，的確是有可能造成幣值浮動（雖然這不代表幣值一定會變低，而且就算不拿去委託，我也強烈懷疑那些幣不會在市面上流通，只是單純晾在那邊 😅 ）。

或許當未來打賞與訂閱改由 USDC 等穩定幣支付，而基礎建設（交易手續費、鏈上治理）用 LIKE 處理，聽起來會是比較合理的解法，要交易就支付穩定幣，創作者比較不擔心幣值浮動，LIKE 就專注在維持鏈上治理與基礎建設，但這也只是我的憑空想像，相信有更多鏈上治理經驗的專家能夠補充更多面向的看法。

驗證人投票權變化，與隨之帶來的議題

這次沒有詳細的圖表，大略簡述一下變動。

原本投票權第一的 GLOxUDomain，因為其中一個大戶的轉委託，導致投票權掉到 2.5%。相對地 Staky.io 一下子拿到 1500 萬，投票權成為第四（7.5%）；而我的驗證人地址也因此增加了 1000 萬委託，變成第一（10.52%）。

這其中除了少量的 LIKE 收益之外，我感到更多的是憂心，原因有幾個：

第一個議題：投票權仍然集中

投票權還是集中在大戶手上，像我跟 Staky.io 都有過半在這位大戶手上，今天是投票權第一或最後，明天可能就翻盤了。

我的看法：

這個問題的解法可能是鼓勵更多散戶委託，但呼應到前面的 LIKE 一通證多用途問題，很多創作者都必須要動用 LIKE 當做日常開銷，要他們將拿到的 LIKE 委託存起來，恐怕不太現實。

第二個議題：以佣金率主導的委託

委託意向似乎還是受到佣金率主導，我不確定上面那位大戶的情況，但至少 Staky.io 是最低的 30%，而我目前是 40%（第二低的族群），或許多少有關係。

我的看法：

要解決這個問題，可能得先強化治理意願，但目前還沒有強應用場景。換句話說，用大量委託能得到的即時利益，可能就只有在高通膨設計下用來抵抗通膨，這是消極的動機，而積極的動機相對沒有那麼即時。

在現實中如果我對台灣政府不滿意，要移民雖然可行，但非常困難，所以有人選擇站出來參選改變它；但對 LikeCoin 這個虛擬國家不滿意，我大可以賣幣永遠離開這個生態圈，「移民」門檻極低，這是在根本上參與意願的差異。要達到有「積極」的治理動機，我想必須等到未來某天，LikeCoin 真的被大家認為是一個不可或缺的基礎建設，才有可能。

第三個議題：佣金比例目前沒有上下限，只有共識

Staky.io 是第一個將佣金設定為 30% 的驗證人，一開始我很驚訝，後來才發現幾位驗證人（包含我）都徹底搞錯了，目前的 40% ~ 60% 並非任何技術限制，只是源自於一個「Signaling Proposal」，也就是說，技術上我可以設定 0% ~ 100% 的佣金比例。

這個「信號」的來源是在第一項由 Matters 提出的議案中，細節有說明「To allow a range of commission rate between 40%-60%, set by a validator independently.」

這裡用了 Allow 這個詞恐怕不太精確，因為原本就沒有任何限制，這個議案的通過也沒有造成任何實際的參數限制被改變，只是提供一個「大家都同意」的版本。或許可以比喻成是像京都議定書這樣的條約，大家同意遵守，但沒有人限制、也沒有法律制裁。

以當時的提案來說，既然通過了，代表當時的驗證人都同意這個遊戲規則，雖然技術上沒有限制，但假設我是新的驗證人，並且跟原本講好的 40%~60% 佣金比例不同，比較好的作法是提出議案來「更新」這個共識，再做對應改動。

我的看法：

這件事情就我的理解，應該並非惡性的競爭，一個很可能的猜想是 Staky.io 根本沒發現這個「共識」藏在 proposal #1，當然大部分委託人也不知道有這件事情，自然地選擇了高回報的驗證人，簡單來說，問題是共識的傳遞不夠明確。

之前也有人詬病驗證人的議案在他看起來很黑箱，我們也認為一直都缺乏跟公眾交流的場所（Matters 用戶雖多，但也不能代表全體 LikeCoin 持有者），所以開了一個 Discord 伺服器，目的是希望讓更多人參與討論，不管是驗證人還是委託人，以上的問題，以及即將到來的議案 #7 應該都被更積極地拿出來交流，才能發揮共治的精神，也希望你一起來參加。

我的佣金調漲，讓投票權更去中心化

在《驗證人日誌 #3：投票權分布變化與觀察 2020/12/06》當中，我已經預告會逐步調高佣金，目的是希望委託更分散。

但為了不要讓才剛轉移給我的委託人無法轉移委託，預計在 2021/1/1 調整到 45%，因為委託給我的 LIKE 增加得比想像還快，這項改動看起來是勢在必行，如果你在意這 5% 的差距，不妨考慮其他佣金只有 40% 的個人驗證人。
