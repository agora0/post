Pi 2022 年終更新

歡迎來到 Pi Network 2022 年下半年的技術和產品更新！

自 Pi2Day（2022 年 6 月 28 日）以來，Pi Core 團隊在構建主網遷移所需的基礎設施和組件、大規模 KYC、生態系統建設以及最終為下一階段主網做準備方面取得了重大進展。

Pi Network 處於封閉主網開發的重要節點，其中 Pi 能夠專注於兩個關鍵優先事項：實現大規模 KYC 和增強生態系統實用程序建設。兩者都是實現我們建立可行的生態系統以準備開放主網的目標的重要步驟。

今天的更新重點是我們所做的與Pioneers 相關且有趣的工作，並且可能會忽略 Pi Network 在接口、後端、合規性、社區成功和錯誤修復方面的無數努力和部署。

具體項目改進

主網遷移

自 Pi2Day（6 月 28 日）以來，大規模主網遷移已經啟動。完全遷移的先驅者現在在 Pi 主網區塊鏈上運行，並根據封閉網絡的指導方針在 Pi 生態系統中使用 Pi（在此處閱讀白皮書章節），那些已經完成主網清單的人可以從手機應用程序遷移他們的可轉移餘額到他們在主網區塊鏈上的錢包地址。

為 Mainnet Migration 所做的工作分為三部分：

  1. 基於來自各種來源（如移動挖礦、KYC、主網清單、合規性等）的同步數據啟用和擴展大規模遷移。
  2. 在所有方面驗證遷移的正確性和準確性，包括（但不限於）餘額、合規性和政策違規。考慮到遷移的原始性，必要時將其還原；解決遷移問題，並從暫停中恢復和恢復。
  3. 通過確保每個基本組件連貫地拼湊在一起來簡化遷移的用戶體驗，例如清單、錢包、主網區塊鍊等。

下面列出了一些工作細節。

  • 在等待遷移期間對驗證檢查進行遷移後改進
  • 驗證 Pioneers 的 Pi 可轉讓余額是否正確
  • 一旦通過驗證，餘額就可以成功遷移到主網區塊鏈上
  • 進一步的驗證功能還可以為在完全轉移之前需要額外驗證的任何人回滾遷移
  • 錢包調整以促進主網遷移
  • 使 Pi 能夠在遷移等待期後被聲明
  • 允許先鋒訪問和管理主網遷移過程
  • 更新了整個錢包用戶界面
  • Mainnet Checklist 用戶體驗改進和錯誤修復，以確保盡可能順利地過渡到 Mainnet
  • 多個 Mainnet Checklist 用戶界面功能和錯誤修復，例如 Mainnet Balance 儀表板的改進、後端和顯示中的數據同步、Checklist 中的添加和修改步驟等。
  • 遷移後鎖定已開發，即將發布
  • 允許鎖定已遷移的Pi

了解你的客戶解決方案

作為每個先鋒遷移到主網的先決步驟，KYC 的工作主要集中在 1) 提高 KYC 應用程序的有效性、效率和安全性，2) 將 KYC 擴展到越來越多的先鋒，以及 3) 分析和為已提交申請但尚未收到 KYC 結果的真正先驅者制定解決方案。有關進展和後續步驟，請參閱此處關於 KYC 的最新詳細更新。

Pi平台和生態系統+社區

我們付出了巨大的努力來促進 Pi 生態系統的發展（封閉網絡的兩個主要目標之一），以便為 Pi 體驗帶來更多的實用程序和用例。雖然已經部署或正在開發多個新的基礎架構功能，但核心團隊正在通過參與黑客馬拉松、社區開發支持、外部業務合作夥伴關係和內部核心團隊創建的應用程序，在社區和新開發人員之間培養生態系統建設。

為此目的的具體技術和產品工作是：

  1. 使SDK能夠連接到Pi Mainnet，並為Pi SDK添加更多功能，讓開發者在主網上構建更完整和用戶友好的Pi應用程序。
  2. 鼓勵和吸引開發人員參與，並以更具指導性和教育性的方式改善開發人員使用 Pi 平台的體驗。
  3. 為開發人員提供基礎設施和諮詢支持。

下面列出了有關主要功能的一些詳細信息。

  • 在頂尖大學舉辦的主要黑客馬拉松，以及計劃於 2023 年 1 月初舉行的更多黑客馬拉松以及進一步的黑客馬拉鬆後計劃
  • 改造 Hackathon 開發工作以構建 Pi 生態系統的實用程序和用例
  • 計劃從 1 月 9 日開始舉辦更多正在進行的黑客馬拉松
  • 開發更多與開發者相關的項目以支持 Hackathon 開發者持續開發
  • App-to-Pioneers 的其他 API 正在開發中，將在今年年底或 1 月初推出
  • Mainnet Pi SDK 發布並改進
  • 允許 Pi 應用程序遷移到主網的主要更新
  • 更詳細和更清晰的文檔 -鏈接到 Github 文檔
  • 完整的開發人員入職指南（正在進行中）
  • 開發人員門戶更新
  • 發布新的“創建應用清單”以指導開發人員逐步完成應用創建過程
  • 開發人員可以更好地管理他們的 Pi 應用程序的新儀表板界面
  • 部署了演示應用程序，使開發人員能夠快速設置自己的 Pi 應用程序
  • 使開發人員能夠註冊主網應用程序
  • 部署了 Pi App Engine V2
  • 為使用 Pi App Engine 的開發人員提供更簡單的設置，以便在可擴展的服務器基礎架構上輕鬆運行他們的應用程序，同時專注於構建他們的產品和前端功能
  • 允許向開發人員提供更多關於部署的反饋
  • 全面改進生態系統中的 Pi 應用程序開發過程，特別是對於為生態系統做出貢獻並且資源較少且對可擴展技術基礎架構了解較少的選定團隊
  • MVP版開發者錢包即將上線
  • 開發人員可以很快創建特定於應用程序的錢包，這些錢包專為支持 Pi 應用程序操作而定制，而不是與個人 Pioneer 帳戶相關聯的個人錢包使用
  • 該功能計劃於 1 月對開發人員開放。
  • Pi Chats 基礎架構變更正在開發中
  • Pi Chats 後端基礎架構將發生重大變化，成為一個更加獨立的應用程序，從而可以更快地開發和部署未來的功能。
  • 將減少聊天的壓力和對其他服務的影響（過去導致使用中斷）
  • 將使未來更容易將 Pi Chats 功能集成到生態系統中

除了上述技術和產品方面的努力外，社區和商業方面也做了大量工作來培養生態系統建設文化，向開發人員介紹和教育 Pi 平台，激勵和吸引開發人員在 Pi 上進行構建，以及促進和支持開發人員部署 Pi 應用程序。這是構建可行的基於實用程序的生態系統的基礎工作——開放網絡的必要準備。

展望 2023 年

該網絡見證了 2022 年的重大事件和巨大進步，包括啟動大規模 KYC 和主網遷移以及生態系統建設的基礎工作。尤其是在今年更大的行業動盪背景下，Pi展現了其獨特的非共識策略的優勢——Pi的長遠眼光和注重循序漸進的物質和效用建設。Pi Network 不僅能夠在行業大環境的喧囂中屹立不倒，而且在其願景指引下一直在穩步前進和成長：“打造全球最具包容性的點對點生態系統和在線經驗，由世界上使用最廣泛的加密貨幣 Pi 推動。”

我們對 2023 年將要發生的事情感到非常興奮——又是朝著實現主網下一個階段的共同目標邁進的一年。去中心化的努力對於構建去中心化網絡至關重要，其中包括區塊鏈、社區和生態系統，具有更完整的基礎設施和廣泛的實用程序。這種努力不僅僅取決於任何一個小團體，而是取決於所有先驅者的共同努力。

到 2023 年，將提供更多的溝通和指導，說明在生態系統基礎設施的主要部分已經到位並為事情的發生和生態系統的發展做好調整的階段，先驅者可以在哪里以及如何联合起來進一步發展和壯大網絡。繁榮。我們知道您渴望，我們也是。

乾杯，我們祝你新年快樂，為 Pi 社區帶來新的工作、新的創新和新的機會。
