“维基解密”创始人阿桑奇获释

“维基解密”（WikiLeaks）6月25日在社交媒体 X 发布贴文表示，其创始人阿桑奇（Julian Paul Assange）已获释并离开英国。

“维基解密”称，阿桑奇已与美国司法部达成认罪协议，美国司法部将寻求法官判处阿桑奇62个月监禁。由于这一时间正好是阿桑奇在伦敦高设防监狱被羁押的时间，因此协议获法官批准后，阿桑奇可以立即获得保释，返回其在澳大利亚的家。阿桑奇已飞向北马里亚纳群岛，预计6月26日上庭要求法官同意认罪协议。

阿桑奇于2006年创办“维基解密”。2010年，“维基解密”获得并发布一批由前美国陆军情报分析员切尔西·曼宁（Chelsea Elizabeth Manning）所提供的美国机密文件，其中涵盖美军空袭巴格达平民的影片、在阿富汗战争的行径等内容。曼宁随后被捕并被判刑35年，随后于2017年获时任美国总统奥巴马特赦；阿桑奇则受到美国通缉，躲入厄瓜多尔驻英国大使馆长达七年，直至2019年4月被拘捕。美国一直寻求引渡阿桑奇至美国依据《间谍法》受审，因此阿桑奇一直被拘留在伦敦，并经历了多次引渡聆讯。

自阿桑奇被捕以来，以无国界记者（RSF）为首的国际组织与国际媒体发起了一场长期的“释放阿桑奇”活动。RSF 强调，针对阿桑奇的定罪“将为其他根据泄露的机密资讯发表报导的人提出类似的起诉铺平道路，为新闻业树立了一个危险的先例”。

“维基解密”称阿桑奇的获释是这场争取权利的“全球运动的结果”，“为与美国司法部进行长期谈判创造了空间”。

香港“47人案”求情开始

香港“民主派47人初选案”审讯历时118日，5月30日对16名不认罪被告作出裁决。有罪被告的求情程序由6月25日开始，预计将持续至8月8日。

第一批求情的是被指为组织者的 5 人，包括认罪的戴耀廷、区诺轩、赵家贤、钟锦麟，以及审讯后罪成的吴政亨。

“串谋颠覆国家政权”罪罪成后会分为三级罚则：首要分子或者罪行重大的，处无期徒刑或者10年以上有期徒刑；对积极参加的，处3年以上10年以下有期徒刑；对其他参加的，处3年以下有期徒刑、拘役或者管制。因此，法庭对不同被告参与程度的认定，可能会影响他们各自的量刑。

习近平会见波兰总统杜达

6月24日，中国国家主席习近平在北京会见了正在访问中国的波兰总统杜达（Andrzej Duda）。

根据中方通稿，习近平指双方要用好中波政府间合作委员会等机制平台，加强各领域合作的战略协调和统筹规划，高质量共建“一带一路”，保障和推进中欧班列等重大项目合作。

就乌克兰问题，习近平称“中方反对一些人借口中俄正常贸易转移矛盾，抹黑中国”。

根据波方通稿，杜达指波中关系“基于相互尊重的良好外交关系、良好政治关系、良好人际关系。”他又表示，“希望中国作为一个世界大国，作为联合国安理会成员，支持努力和平结束俄罗斯在乌克兰发动的战争”。

同时，习近平还宣布中国将对波兰公民实施15日单方面免签政策，同时又“欢迎更多优质波兰农食产品进入中国市场，支持扩大双向投资”。杜达则表示关于波兰食品向中国出口的双边协议可能是“史上最大的”。

两名日本公民在中国苏州遇袭

据日本公共广播 NHK 报导，一辆载有日本学校学生的巴士6月24日在中国江苏省苏州市遭到一名男子袭击。事件造成两名日本公民和一名中国公民受伤。

据报导，该名中国公民是在阻挡男子行凶时受伤，制止发稿时其仍在抢救。事发后日本学校已决定停课一天。

中国外交部新闻发言人毛宁在6月25日的例行记者会上证实此事，她表示“目前，2名日本公民均无生命危险，受伤中国公民仍在抢救中。”她又指“我们对发生这样的事情表示遗憾”。同时，毛宁强调“类似这样的偶发事件在世界上任何国家都有可能发生。中方将继续采取有效措施，像保护中国公民一样保护在华外国人安全”。

苏州高新公安局6月25日在其微信公众号发布通报。通报称，犯罪嫌疑人是“近期外地来苏无业人员”，“案件侦办工作正在进一步开展中，将依法依规对犯罪行为予以严惩。”

台湾持续极端高温

据中央社报导，台湾中央气象署6月25日发布高温资讯，各地天气高温炎热。

中央气象署指云林县为红色灯号，有连续出现38度极端高温机率；屏东县为橙色灯号，有38度极端高温出现机率。台北市、新北市、桃园市、新竹县、台中市、彰化县、南投县、嘉义市、嘉义县、台南市、高雄市、花莲县为橙色灯号，有连续出现36度高温机率。

中央社又援引卫福部统计报导，至24日全台因热伤害就诊人数达460人。

OpenAI停止对中国提供 API 服务

据中国《人民日报》主管的《证券时报》6月25日报导，OpenAI 停止对中国提供服务。

《证券时报》指6月25日，OpenAI 向中国用户发布邮件称，自7月9日起阻止来自非支持国家和地区的 API 流量。受影响组织若希望继续使用 OpenAI 的服务，必须在其支持的国家或地区内访问。目前，OpenAI 的 API 向161个国家和地区开放，由于中国未在其中，这意味着OpenAI 将终止对中国提供 API 服务。

OpenAI 的API 接口使不同服务中都能接入 OpenAI 提供的服务。《证券时报》指百度、阿里巴巴、智谱 AI 等多家公司已发布了相关“搬家”计划，让受此影响的用户可以切换至国产 AI 大模型。

6月22日，美国财政部曾发布一份规则草案，要求对美国在半导体和微电子、量子计算和人工智慧领域的某些投资进行监管，禁止或要求限制中国在 AI 和其他技术领域的投资。

中国“嫦娥六号”登月器成功返回

6月25日，中国国家航天局宣布，“嫦娥六号返回器准确着陆于内蒙古四子王旗预定区域”。

国家航天局指，嫦娥六号“工作正常，标志着探月工程嫦娥六号任务取得圆满成功”，而中国“实现世界首次月球背面采样返回。”

国家航天局指，“鹊桥二号”中继星在圆满完成“嫦娥六号”任务后，“将择机开展科学探测任务，其携带的极紫外相机、阵列中性原子成像仪和地月甚长基线干涉测量（VLBI）试验系统，将收集来自月球和深空的科学数据”。

美国和中国目前都有载人登月计划。中国仍在实验登月飞船和火箭，预计可能在2030年实现载人登月，而美国初步计划在2026年以载人飞船实现人类“重返月球”。
