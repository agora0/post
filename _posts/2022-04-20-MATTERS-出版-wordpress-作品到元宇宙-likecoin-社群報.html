🔎焦點故事：LikeCoin 連結 WordPress 與 Web3

過往要把文章以 NFT 形式出版，工序很䌓複：作者須先把內容及元數據儲存在如 IPFS 等分散式空間或區塊鏈上，再上傳到 NFT 市場定價出售，過程中還要入手幾款不同用途的密碼貨幣。然而現在 WordPress 的用戶有福了，因為寫完文章以後只需按一個鍵，便能完成出版 NFT 的前置工序，內容立即上鏈，過程只需一分鐘！

全新推出的 LikeCoin WordPress plugin 2.5.3 為古騰堡編輯器度身訂造，讓 WordPress 網站一鍵接通 Web3。功能包括：

  • 一鍵發佈文章到 LikeCoin 並儲存於 IPFS 及 Arweave
  • 以 LikeCoin 一筆過支付內容上鏈及分散式儲存費用

此 plugin 日後將支援進一步鑄造 NFT 並售賣給讀者，實現完整的分散式出版。

貢獻 WordPress 社群

目前 LikeCoin WordPress plugin 已有 1,500 個活躍安裝。根據 W3Techs 調查，全球高達 43% 網站使用 WordPress，於內容管理系統佔有市場主導地位。LikeCoin WordPress plugin 將為 WordPress 用戶帶來革命性的體驗，助 Web2 網站進入 Web3 新世界。

將支援 NFT

現時 LikeCoin 的 NFT 模組已經在開發中，預計於 5 月供測試，在 LikeCoin 上發行 NFT 指日可待。期望除了 WordPress 用戶可直接發佈 NFT 以外，depub.SPACE 及 Liker.Social 等 LikeCoin 上的 Dapps 亦能支援 NFT。

使用教學

📚DApp 應用

Capture App 整合 LikeCoin

Numbers Protocol 開發的 Capture App 最新版本已整合 ISCN 功能，在照片註冊完成後，點選照片再點右上角的 Settings 符號，再選「網絡動作」的 Metadata registration (ISCN) 註冊元資料，即可為照片儲存 LikeCoin 鏈上的永久憑證。立即 下載 Capture App 嘗試這個新功能吧！

Liker Social 也支援 DePub

Liker.Social 的去中心出版功能經已上線，用戶只需要嘟文後在 TOOT 頁面點擊 DePub 即可上載內容至 IPFS 及 Arweave 並取得 ISCN，為重要內容永留印記。想進一步了解更多可參考 使用指南。

dao.like.co 新設計

快來看看 dao.like.co 的 全新設計，可參看投票情況、驗證人投票取向，甚至可以在介面上直接進行提案！希望加入其他功能嗎？不妨到 #tech-subdao 頻道發表你的意見。
