完整標題：only 與 merely 的不同之橋接分析 -- Bridge Words Analysis about the differences between “only” and “merely”


1. only = o.n.l.y = 口.聯.二.一 = 口.聯.二.1 = 口.聯.二1 = 口.聯.二| = 口.聯.|| = 口.聯./ \ = 口聯/ \ = 只。其中 l 約略形聲台閩語「二」的 lí 式發音之起音。

2. only = only.y 省一個 y = 只.有 = 只有

3. only = only.y 省一個 y = only.yes = 只.是 = 只是

4. only = o.n.l.y = 口.鳥.連.ㄧ = 口.隹.連.ㄧ = 口隹.連.ㄧ = 唯.連.ㄧ= 唯連一 = 唯一

5. only = o.n.l.y = 口.鳥.連.有 = 口.隹.連.有 = 口隹.連.有 = 唯.連.有= 唯連有 = 唯有

6. merely = m.e.r.el.y = m.E.r.el.y = middle.|三.二.一|.人 = 中.|三.二.十.亻 = 中|三.二十.亻= 中|三.廿.亻= 中|三廿.亻=堇.亻= 堇亻= 僅

7. merely = merely.y 省一個 y = 僅.有 = 僅有

8. merely = merely.y 省一個 y = merely.yes = 僅.是 = 僅是


P.S.  only ㄧ字還可能橋接於一些類似的漢字語意，請參考〈only 與「單ㄧ容樣」或「ㄧ單容樣」、「ㄧ數容樣」及「只」、「只有」、「只是」或「唯ㄧ」、「唯有」或「僅」、「僅僅」、「僅是」、「僅有」、「僅只」等的轉換密碼〉-- https://vocus.cc/article/5fd2b9d0fd89780001c0f9cf


補記.1：mere 及 merely 與漢字的其它橋接轉換式 (其中 mere 多作形容詞，merely 多作附加形容詞或副詞)


mere = ㄇㄧ二 = 口II = 口 / \ = 只，其中 re 的實際發音近似華語「二」。

mere = ㄇㄧ二有 = 口II 有 = 口 / \ 有 = 只有，其中最後一個 e 約略形聲華語「有」的起音，但實際發音有點忽略了。

mere = nnㄧroundE = 人ㄇㄧ〇丨三 = 亻凵一口丨三 = 亻廿口丨三 = 亻堇 = 僅

mere = meree 省一個 e = mere.e = 僅.有 = 僅有

mere = 僅 + 只 = 僅只，此時即重疊轉換前述的「僅」和「只」。

mere = nnere = 鳥ㄇ一連一 = 隹口連一 = 唯一，其中「鳥」轉注「隹」。

mere = nnere = 鳥ㄇ一連有 = 隹口連有 = 唯有，其中最後一個 e 約略形聲華語「有」的起音，但實際發音有點忽略了。

mere = miniE二e = 小εee = 小幺一E = 糸一屮 = 糸一ㄩl = 糸一ㄩL = 糸一ㄩ乚 = 糸屯 = 純，類似「純粹的」之意，或也可能轉換漢字「純一」、「純之」。

merely = mere.ly = 只.肉丶丿= 只.月一丿 = 只.月𠂇 = 只.有 = 只有

merely = mere.ly = 只 or 只有.然 = 只然 or 只有然，其中「只然」即漢字語「然只」的同義倒裝詞。

merely = mereely 省ㄧ個 e = mere.ely = 只.E日一 = 只.屮旦 = 只.龰旦 = 只.是 = 只是。此外，同前式的模式，也可能轉換漢字「只是然」。

merely = mereely 省一個 e = mere.ely = 僅 + 是 = 僅是，其中 ely 同上式轉換漢字「是」。

merely = mere.ly = 僅 or 僅有.然 = 僅然 or 僅有然，其中「僅然」即漢字語「然僅」的同義倒裝詞。

merely = mere二 = 僅二 = 僅僅，其中 ly 約略形聲台閩語「二」的發音之一。


補記.2 : 此外，mere 一字有時又用來表達類似 mini (微；微之；小；小之；微小；微小之；滿小；滿小之；沒大之) 之性質的一些意思


mere = 沒ㄧ人㞢 = 没大之，意通「小的」。

mere = 3ㄧ連㞢 = 三i連之 = 三點連之 = ∴之 = 小之，意通「小的」。其中由三點構成的符號 ∴ 是約略表達漢字「小」的甲骨文字體。

mere = meeerre 省二個 e 省ㄧ個 r = manㄧE一ㄦ卜E = 人ㄧE一ㄦ卜彐 = 亻/ 山一ㄦ卜又 = 彳山一儿攴 = 彳山一儿攵 = 微

mere = mereee 省二個 e = 微Eㄧ = 微屮一 = 微㞢 = 微之

mere = mere + mere 省ㄧ個 mere = 微 + 小之 = 微小之

mere = 米粒E = 米粒㞢 = 米粒之，此時又類似漢語「米粒般大小的」，或意通「微不足道的」、「芝麻蒜皮的」、「不值一提的」。

mere = miniE連也 = 小ε連也 = 小3連也 = 小三連也 = 小氵連也 = 小池，類似「小塘」、「小池塘」，此時多作爲名詞使用。

merer：mere再；mere更。

merest：mere最；mere頂；mere極。

merely：mere然；mere哩；mere地；mere容樣；mere例樣；mere類樣；mere爾。其中的 mere 可以代入前述 mere 一字的各種含意。
