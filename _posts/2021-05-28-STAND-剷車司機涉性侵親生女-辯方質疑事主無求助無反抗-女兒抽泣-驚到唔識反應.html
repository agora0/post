年近五旬的男剷車司機，涉嫌自親生女兒 12歲起，5年間多次非禮及強姦她，並曾有一段時間強逼女兒 X 服用避孕丸，最終由 X 的社工揭發並報警。剷車司機否認強姦、亂倫等共7項控罪，今（27日）在高等法院續審，辯方多番質疑X沒有求助及反抗，X作供時抽泣並說「嗰時驚到唔識反應，個腦一片空白」。但辯方續指，若X在床上緊閉雙腳，被告根本無法與她性交，X不同意。

親生女兒 12 至 16 歲期間 1 次非禮、3次強姦

被告為 49 歲剷車司機，被控於 2015 年至 2019 年 8 月 1 日期間，在葵涌兩個單位及內地福建省漳州市雲霄縣某房屋，一次非禮 X、兩次對 X 作出嚴重猥褻作為、3 次強姦 X 及一次與 X 亂倫。X 為被告的親生女兒，案發期間是 12 至 16 歲。

控方早前陳辭指出，2015年，就讀中一的 X 與被告、妻子及哥哥，一家四口同住葵涌某單位。該年某一晚，被告趁妻子及兒子不在家時，叫原在另一房睡覺的 X 與他同眠，表示「我張牀軟啲、舒服啲」。X 不虞有詐，最終在被告的牀上被摸腰及胸，繼而被要求手淫、口交及強姦。

辯方質疑無求助    X：個腦一片空白

辯方以視像對話盤問 X，指當被告要求 X 進入其睡房時，X 有預感早前被非禮的事情再次發生，而她亦有足夠時間反鎖自己於房間內，或致電母親求助，但她沒有，稱「我淨係唔想入去但我冇做任何嘢」。辯方又指，在 X 被非禮後有否在家中戴上胸圍，以增被告對其再次摸胸的難度，X 指沒有。

其後 X 按被告要求睡在床上，一、兩分鐘後進而摸她、遭被告要求手淫、口交及強姦。辯方盤問被告，為何選擇近牆的位置上床，令自己陷入困境，X 開始抽泣其指「我個腦一片空白，所以我咩都諗唔到」；其後控方要求小休10分鐘獲准。

辯方指若 X 緊閉雙腳  被告無法強姦

辯方再指，X 之後替被告手淫，她當時身體背向被告，X 指「佢（被告）捉住我隻手向後伸」。在辯方盤問下，她同意當時沒有嘗試扔開被告的手。其後被告用手指觸碰X的陰部，又以陽具摩擦其陰部，辯方質疑若X當時緊閉雙腳，被告不可能做到相關動作，X不同意。

辯方質疑無反抗    X：我嗰時驚到唔知點反應

X 當時曾打算離開但被被告扯低，之後再無試圖起身反抗；辯方又質疑 X 當時為何沒有用有利的姿勢反抗如咬他、踢他下陰，X 則激動地指「我嗰時驚到唔知點反應，我點會知咩係有利姿勢？」其後被告陽具進入被告陰部，過程中X曾叫「唔好」，但沒有反抗；事後被告起身如廁但 X 沒有離開，並睡至翌日。

辯方又質疑 X 為何沒有向母親、學校老師求助，X 指「唔敢求助，我驚我講唔到出口。

中二於廁所遭強姦 辯方質疑沒有關上門如廁

X 就讀中二時，曾遭被告從後攬腰，再次強姦。辯方指，X當晚起床如廁時，沒有關上門，X 解釋因為知道全家都在睡覺，當她聽到腳步聲時以為時母親，故沒有為意並關上門。之後 X 彎下身時，感覺到有人抱著她的腰，「我嚇一嚇，望吓係邊個」。X 又同意當時門打開，若X大叫正睡覺的哥哥和母親會醒來，但因父親叫她不要叫，所以她沒有。

控方案情續指，之後被告強逼X服用避孕丸，但X兩、三個月後未有跟從，只是假裝服用。2019年2月5日，X一家人回福建家鄉過年，被告趁機強姦了 X 。同年暑假，被告與妻子鬧離婚，搬到附近另一單位居住。

X 為取得每月3000至4000元零用，跟隨被告居住，被告繼續對她施淫，與 X 性交時使用性玩具「震蛋」，事件最終因 X 情緒爆發，輾轉經社工轉介至警方而遭揭發。案件周一（31日）續審。
