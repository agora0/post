這是最好的時代，這是最壞的時代。這是政確的時代，這是 error 的時代。

當這時代這城市的對與錯被政權所壟斷，一切與政權的立場不相符的意見言行皆被視為錯誤之時，堅持民主自由法治的香港人的存在，彷彿就是最大的 error。遊行示威集會是錯誤，喊口號舉標語是錯誤，就連拍節目及教導真實發生的歷史故事，也是錯誤，只有緊跟政權搖旗吶喊，擔當忠誠廢物，才是政確的事。當政權畫下的紅線愈粗愈密時，香港人所能選擇的，可能就只有移民離開，或者噤聲留下來乖乖聽話保持政確。以往香港人引以為傲的思辨精神，此刻已經容不下了。

當過往發聲的渠道被剝奪時，文藝娛樂，彷彿是僅餘可讓我們喘一口氣盡訴心中情的樹洞，只是當博物館被迫進行政治審查，大台對娛樂圈施加重重壓力，藝人紛紛北上賺人民幣之時，能讓我們心甘情願投入而不用陷入政權製造娛樂至死陰謀的，彷似沒有多少選擇。幸好，我們仍有一隊 Mirror 及 Error，讓我們重拾我們曾經擁抱過的追星文化。

如果說 Mirror 是我們映射美好形象的鏡子，那麼 Error 便是我們不少人本身的樣子。與一般偶像團體不同，Error 沒有那些光鮮亮麗的形象，由《全民造星》成軍時便飽受不少批評，樣衰、MK、肥、廢，這些負面標籤，曾經一直伴隨 Error 四子，未能成為偶像的他們，只能擁抱他們的缺憾成團，毫不遮掩本身的性格缺點，拍攝被道貌岸然者認為低俗的節目，進行名符其實嘔心的遊戲，就如不完美的我們往往會為五斗米進行我們不願意做的事。可是，Error 四子的結合卻有特殊的化學作用，把一眾嘔心的任務變成熱血的挑戰，在無制限團隊的精心設計下，低俗騎呢的節目，竟產生了超乎尋常的激勵作用，甚至能引發一眾觀眾深思，讓一眾飽經挫折的觀眾如同 Error 四子般得到救贖。

褪掉虛偽的粉飾，從《花姐 Error 遊》到《Error 自肥企劃》，以及 Error 與無制限團隊一眾節目，把台前幕後一眾演職人員真實的一面反映給觀眾看，就連那些為節目效果而虛構的劇本與各種內部矛盾及拍攝失敗也不介意坦言向觀眾道出，只為讓觀眾能夠產生共鳴，發現鏡頭後眾人的實際面貌。過去的我們，總被教導只能把最美好的一面展現人前，把缺憾弱點隱藏於心間，電視節目的世界往往也是美滿幸福彷似童話，卻無形加深了我們對完美的抑鬱及壓力，當發現我們嘗試如電影般竭力爭取時卻得不到電影般美好的結局時，更使得我們陷入後社運抑鬱。幸好，Error 與無制限團隊把自己的失敗也呈現幕前，讓經歷失敗的我們找到鏡頭前的同路人，一同盡情爆粗痛哭發洩情緒後，再告訴我們縱使失敗過無數前，仍要挺起胸膛繼續嘗試，不用懼怕在上位者的目光，堅持自己的初心追夢，無論成功與失敗，也會學到重要的一課。

感謝 Error 與無制限團隊，甘冒被守舊者投訴及廣管局譴責的風險，仍堅持為我們製作低俗卻高質的一系列節目，讓我們在放聲大笑後，聽着你們的志氣 bite 及蘊含的種種信息，令我們重新振作起來。這個時代，需要 Mirror 的正能量鼓舞，更需要 Error 的 underdog 奮起打氣，最需要無制限團隊那份堅持到底至死方休的拼搏精神。只有堅持，才可看見希望，就算我們長於 error 的時代 error 的地方，也要堅持我們心中的正義，才能有機會糾正真正的 error，使我們的家回復本來的光輝。
