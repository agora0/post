我們可以在首頁看到審核名字和3天的倒計時時間。
點擊審核名字！！！

用英文寫名字的規則

First Name + Middle Name /Maiden Name+ Last Name

名+中間名+姓

First Name (名字)->例如：Lucille，Charlize，Cameron，Irene，Jeanette

Middle Name /Maiden Name  (中間名)->例如：Marilyn, Betsy

Last Name（您的姓氏）->例如：Adams‎, Ball‎, Theron‎, Diaz

如何輸入 名字，姓氏，中間名

+共有3個字段：名字，姓氏，中間名

例如，如果您的名字是：  Rachel Meghan Markle

那麼最準確的書寫和輸入方式如下：

名： Rachel

中間名： Meghan

姓：  Markle

+只有2個欄位：名字和姓氏

在這種情況下，您不會看到 中間名，這並不意味著您不會輸入中間名。如果省略，則中間名將與您的Yoti帳戶名不匹配。您只需要像這樣完全輸入：

  • 名字（您的名字）： Rachel Meghan
  • 姓氏（您的姓氏）： Markle

或如下：

  • 名字： Rachel
  • 姓氏（您的姓氏）：Meghan Markle

一次改名的機會

正如之前多次溝通過的，我們將為先驅者提供一次在主網之前更正其帳戶名稱的機會。由於Pi網絡每人一個帳戶的政策以及網絡人性的高度重要性，因此，只有通過KYC的Pioneers才能在Mainnet上聲明其Pi，以及Pi帳戶的名稱和他們的名稱KYC必須匹配。因此，在您的Pi帳戶上擁有正確和準確的名稱至關重要。

在過去的一年半中，我們收到了許多真正的先驅者提出的更正其名字的要求。在註冊過程中，某些先鋒沒有輸入拼寫正確的名稱。可以理解的是，許多人在初次認識Pi時並不想分享其真實正確的正式名字。除了新用戶的心態外，還存在通常的差異，例如中間名，娘家姓或日常生活中使用的名稱與身份證明文件中的名稱。另外，某些地區的某些KYC流程僅包含本機語言名稱，例如使用國民身份證，而Pi帳戶中的名稱可能僅是英文。因此，為了確保與KYC的名稱完全匹配，許多先鋒需要更新名稱。為了進一步確保匹配，核心團隊將名稱更新過程設計為包括輸入您的中間名或姓氏，為了簡便起見，在註冊過程中將其省略。如果適用，名稱更新過程還提供了以相應語言輸入名稱的選項。

這是所有現有先驅者更正其姓名的唯一機會，而且此選項的時限很短。該選項今天僅在有限的時間內可用。每個現有的先鋒在下次訪問該應用後僅3天就會提供此選項。提交後，該選項將不再可用。如此短的時間窗口的原因是為了防止不良行為者或偽造賬戶能夠被交換或出售給其他人。時間到期後，所有現有帳戶將被鎖定為它們具有的任何名稱，並且如果這些帳戶的將來的KYC無法與這些帳戶的名稱匹配，則這些帳戶中開采的Pi將不會遷移到主網。我們了解此功能的時間壓力，但是請理解如此短的時間限制的必要性，以確保網絡的健康並防止通過虛假帳戶非法開采的Pi進入主網。我們也鼓勵所有了解此功能及其時限的先鋒人士，敦促您的賺錢團隊成員和安全圈成員確認其姓名正確或進行任何必要的更改。

對於以後創建的任何新帳戶，他們將在註冊後有兩個星期的時間進行一次名稱更正，因此不會再為他們提供泛網絡更新其名稱的機會。任何新創建的帳戶的單獨名稱更正窗口不會招致出售假帳戶的風險，因為這些新創建的帳戶幾乎不會開採任何值得出售的Pi。

我們想在此進一步強調，Pi網絡的服務條款明確禁止銷售其他法定貨幣或Pi貨幣的Pi或Pi帳戶。這種行為違反了Pi的政策，並損害了生態系統的健康。如果報告或發現任何此類違規行為，將導致該帳戶被凍結，並丟失所有此類帳戶中開采的Pi。我們敦促每位先鋒不要出售或購買任何Pi帳戶。

通過名稱更正和更新過程收集的所有名稱數據將用於驗證先鋒的身份並與KYC數據匹配，以確定先鋒是否是真實的人並確保網絡的人性化。

在名稱更新中，我們還為先鋒提供了輸入他們的電子郵件地址的選項。這旨在解決多年來由於密碼丟失，電話號碼更改或Facebook禁用帳戶而導致的任何帳戶恢復問題。添加電子郵件地址為先鋒提供了另一種方式來證明其對Pi帳戶的所有權或在其他登錄方法丟失時恢復其帳戶的方式。它還為我們提供了一種通知重大網絡事件的方法，例如啟動Mainnet或在Pi區塊鏈上創建地址，以防萬一您在應用程序上不再活躍。您現在輸入的電子郵件地址將來需要進行驗證，例如給您發送驗證電子郵件和鏈接，但是現在您只需要輸入驗證電子郵件和鏈接即可將其與您的Pi帳戶相關聯。

以下是有關允許使用的名稱更改功能的一些常見問題：

問題：如果某人沒有護照怎麼辦？（沒有英文名字）

答：以當地語言非常準確地輸入一個名稱，然後輸入您對英文名字的最佳猜測。例如，對於中國人，即使他們沒有護照，也要輸入他們名字的拼音。

問題：如果一個人在英文名稱或暱稱中加上外來字符會怎樣？

答：正確的語言環境名稱非常重要。英文名稱或暱稱有點不正確應該沒問題。

問：如果某人在KYC之後更改其法定姓名會怎樣？

答：只要此人有關於舊名字的文件，就可以了。

問題：小寫字母或大寫字母是否重要？

回答：不，字母大小寫無關緊要。使用大寫和小寫的護照都可以。
