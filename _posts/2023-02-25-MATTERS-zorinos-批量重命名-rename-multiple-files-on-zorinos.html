緣起，
下載了一套寫真集，但是每一張相片的名稱都帶有某個網站的網址
而且帶有的是類似“[cao]”這樣的國內粗口字眼，對於想要收藏的我來講，是不能忍受的
所以，我想要去掉幾百張相片名稱中的特定字符

網上搜尋一番，因ZorinOS 是基於 Ubuntu 的，所以立即就得到兩種簡單的方法（Linux 通用吧

方法一，GUI 式
全選（ctrl+a）
右擊重命名（f2）
查找和替換（Find and replace text）
end
例如，要去掉所有包含的[test]的字符，只需在 Existing Test 內輸入這個字符，Replace With 留空，再點擊 Rename按鈕即可完成

方法二，命令列式
使用命令 rename ，ZorinOS 沒有預置，需要 apt install rename
命令示例，
rename -n ‘s/test/new/’ .jpg （預覽一下把 test.jpg 改成 new.jpg 的效果
rename -v ‘s/test/new/’ .jpg （沒問題的話，直接修改

速度上，
命令列式，瞬間完成，全程沒有任何卡頓。只是學會了一個小命令，就突然好喜歡 Linux 
GUI 式，全選打開重命名窗口的時候，稍微有一點卡，只是相對命令列來講。但是一鍵替換的時候也是瞬間搞定

複雜性上，
GUI 一目瞭然，無需學習任何命令
命令列的話，因爲通配符或特殊字符的關係，有時未能按預期的顯示。例如使用 rename 把 [ixxx] 替換爲 love ，簡單地按照上面的命令，也許會變成了 [lovexxx]
這時候就需要學習 rename 的用法，rename -h 或是 man rename

總之，ZorinOS（or Ubuntu/Linux) 很優秀，期待着 ZorinOS 17 的來到
