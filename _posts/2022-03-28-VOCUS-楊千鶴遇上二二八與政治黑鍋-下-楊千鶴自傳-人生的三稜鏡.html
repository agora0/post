上次介紹台灣第一位女記者——楊千鶴，與 N 先生的曖昧一直沒有突破點。（旁人看得好急）​
綜觀他們身處的年代，感情還參雜了戰爭、族群、傳統價值觀等各種因素。小曖昧其實是大時代的縮影啊。（當事人應該覺得更苦）​
而且，真正的考驗不是遇到渣男，往後有更大的挑戰呢。​

羊羹的啟示​

在楊千鶴生日宴會之後，兩人持續保持若有似無的曖昧關係。就在某天，N 先生突然說，他被派去馬尼拉出差，礙於戰爭機密的關係，沒辦法告訴楊千鶴太多資訊。​
有趣的是，N 先生託楊千鶴買羊羹寄去東京家裡。​
可能因為 N 先生覺得台灣的羊羹適合送本島人做伴手禮？
有各種想不通的地方，但楊千鶴還是照做，寄去了東京。​
於是有好一陣子，楊千鶴失去 N 先生的消息。​
即使 N 先生回來台灣了，依舊遲遲沒跟楊千鶴聯絡。直到 N 先生再次聯絡楊千鶴，是告訴她自己即將被調去高雄。​
（楊千鶴 OS：到底在演哪齣）​
N 先生態度轉變得太快，也不打算解釋清楚。​
最後事情的真相，是楊千鶴在 N 先生家中，不小心看到一封信。​
那封信是 N 先生提過的「前」未婚妻寄來的。​
對方稱呼 N 先生的方式很親暱，並且在信中道謝說，謝謝你寄來的羊羹，非常好吃。​
羊羹代表的是愛情的傻子，還包含兩人出身的極大差異，不可能有個完美的結果。​
​

比渣男可怕的是......

不過楊千鶴其實不必太傷心，因為另一位男主角始終在等待她。​
那位男主角後來成為楊千鶴的丈夫，他叫林嘉雄。​
林嘉雄早在楊千鶴讀靜修女學校時，就寫過情書給她，幫楊千鶴度過亡母的消沉時期。​
不過跟 N 先生相比起來，林嘉雄個性軟弱，不是公子哥、才子，還是超級無底傻的愛情傻子，推掉許多相親，始終忘不掉初戀情人楊千鶴。（明明就還沒交往）​
還在他人生困頓的時刻，大吵大鬧說不想活了，想去報名當台籍日本兵。​
這種軟弱性格反而激發楊千鶴的俠女基因，立刻跑去阻止林嘉雄。最後，她給予林嘉雄勇氣的方式是答應嫁給他。​
​（？？？？）​​
也許楊千鶴答應結婚的當下，想的不是什麼愛情，也不覺得非得犧牲奉獻自己的一生，走入家庭，但那一刻她總算決定卸下新時代女性的外表，成為一般的家庭主婦。​
真正的考驗這時候才來臨。​
她遇到了惡婆婆，又有個不敢幫她說話的丈夫，許多苦悶事只能往心裡吞。她時常懷疑，自己當初怎麼會願意嫁給性格如此懦弱的丈夫。（會懷疑真的滿正常的）​
那時候還是戰爭時期，就在婆媳關係惡劣、三不五時空襲的情況下，楊千鶴挺著大肚子，幸運地活到了戰爭結束之後。​
夫妻倆順利搬出來，不需要跟惡婆婆纏鬥，而林嘉雄得到了日產接收委員會的工作，準備搬去台東展開新生活。​
事情似乎準備有新轉機，但就在 1947 年，他們聽說了台北的緝煙事件。​
​

二二八與政治黑鍋​

就在夫妻倆聽聞二月二十八日的台北緝煙事件後，政府的人半夜找上門，問林嘉雄要不要加入二二八事件處理委員會？​
​找林嘉雄加入處理委員會，乍聽是非常合理的選擇，因為他是知識分子，會講台日語，又在政府部門做事。​
​況且，一群台籍青年聚集，用民主方式向政府共商解決辦法，另一方面幫政府安定民心。這在日治的新文化運動時期就有脈絡可循。​
一切看起來都很正常，但楊千鶴就是隱約覺得不安，阻止林嘉雄加入。​
多虧楊千鶴驚人的第六感，撿回林嘉雄的一條命。​
二二八事件處理委員會最初是行政長官陳儀同意設置的，集結各地許多優秀青年加入，但就在台北的委員會提出 32 條處理大綱，陳儀拒絕接受。​
三月十日，全台宣布戒嚴，開始武力鎮壓。​
處理委員會的成員多半凶多吉少，許多人下場是失蹤或遭槍殺。​
幸運也好，觀察力敏銳也好，在這邊不得不佩服楊千鶴的精明跟勇敢。​
二二八事件沒有打退楊千鶴的信心。​
她參選 1950 年的地方議員，而且不負眾望地選上了，感覺楊千鶴想拯救的不只是失意想當兵的少年，還想保護地方上所有的朋友。​
不過，丈夫林嘉雄卻意外捲進私賣米的政治黑鍋。​
林嘉雄那時是台東糧食事務所的所長，某天突然得知長官要來突襲檢查。原來他們的倉庫有包稻穀，不知道被誰替換成粗糠。​
林嘉雄因此被判刑入獄。​
這件事對楊千鶴來說很不能接受，怎麼能不調查清楚，直接判定是林嘉雄的錯，分明是一場政治的黑鍋事件。​
她的丈夫林嘉雄成為他人利用的棋子。​

失望與希望​

二二八事件與丈夫入獄的打擊，讓楊千鶴對國民黨把持的政治感到失望。​
晚年的楊千鶴隨子女移居到美國生活，人生後半場是在國外度過。​
她用日文寫成的回憶錄《人生的三稜鏡》，原初只打算在日本出版，所幸有張良澤老師翻譯，我們才得以在台灣看到這本書。​
雖然是老奶奶的回憶錄，但能看到女孩成長過程的矛盾感，還有如何學會堅強。​
也很佩服楊千鶴經歷了如此多重大的年代，殖民、二戰、二二八、白色恐怖......，還有她身為女性、台灣人，傳統社會價值、殖民統治者的眼光，試圖成為她身上的枷鎖。​
但楊千鶴仍用她的方式，在困難中挺立生存下來。​
《人生的三稜鏡》真的是一本看了會充滿勇氣的自傳，希望大家會喜歡。
