（獨媒報導）財政司司長陳茂波今早宣讀新一份《財政預算案》，今年封面為「晨曦色」，寓意期盼香港未來一年的經濟逐步改善。多名政府官員及立法會議員一如以往夾「Dress code 」，身穿杏色或橙色衣著現身，陳茂波亦打橙色呔示人。

議員與官員夾「Dress code 」，成為近年立法會常態。去年發表《財政預算案》時，多名政府官員及立法會議員均以橙色服裝示人；而行政長官李家超發表《施政報告》時，除了官員一致以綠色領呔和領巾顯示「隊形」，不少立法會議員亦身穿綠衣，以示支持。


財政司司長陳茂波

公務員事務局局長楊何蓓茵身穿深橙色外套率先步入議會，而律政司司長林定國打橙色呔；房屋局局長何永賢穿著淺杏色西裝外套及西褲，行會召集人新民黨主席葉劉淑儀亦穿著杏色西裝外套及上衣。
