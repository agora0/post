Web3Press 讓你在區塊鏈出版文章，如常寫作但以不一樣的方式出版。

Web3Press 前身名為 LikeCoin WordPress plugin，為了更清𥇦地集中發展 WordPress 市場而重塑品牌。Web3Press 能成為佔全球網站 43.1% 的 WordPress 網站跟 Web3 世界之間的橋樑。

Web3 是互聯網的新標準。互聯網在過去幾十年間逐漸朝更分散的方向演變。於 Web1 時代，資訊是單向發佈的；於 Web2 時代，用戶參與產生內容。來到 Web3 時代，資訊加入了被擁有的概念。回應 WordPress 「出版民主化」的信念，Web3Press 再向前推進一步：讓作者獲得擁有的自由。沒錯，開放內容，為自由故。

開放內容，為自由故

Web3Press 提供一個創新的，尤其適合開放內容的營商模式，從此不用倚賴付費牆和廣告。

Web3Press 建基於 LikeCoin 這條專注為創作者經濟提供技術建設及社群生態的區塊鏈。

你可以 Web3Press：

出售你的文章

你的創意，是為你粉絲而設的最佳商品。讀者因為喜愛你的文字而購買你的文章。Web3Press 讓你如常寫作但在區塊鏈以嶄新的 NFT 形式出版文章，讓讀者在閱讀時藉著購買你的作品支持你。

以你的藏品為傲

「你讀故你在」。閱讀塑造你的內涵，藏品展示你的品味，還可收集珍貴和稀有的文章。

建立讀者社群

你可於公開的鏈上記錄獲知誰曾購買你的 NFT，所以也可按此記錄回饋你的支持者，例如輕鬆地送他們附上溫馨留言的，珍貴限量的 NFT 禮品；當然也可攄此資訊提供其他獨特的禮遇。


保存你的內容

在 LikeCoin 鏈上註冊內容元數據 (ISCN)，將文章儲存在 IPFS 和 Arweave，備份到 Internet Archive，一次過完成。


如何使用

前提：你需要安裝了 Keplr 錢包及於其中有少量 LikeCoin，才能以 Web3Press 出版文章到區塊鏈。

  • 安裝 Keplr：Keplr 是基於 Cosmos 生態系的密碼貨幣錢包。你需要安裝 Keplr 的 Chrome 瀏覽器插件以使用 Web3Press。請觀看此影片（廣東話）以詳細了解如何免費安裝 Keplr。（國語版教學影片在此）
  • 領取免費 LikeCoin：你需要少量 LikeCoin 以支付網絡費用。你可參考這裡的方法免費從 faucet 獲取一些 LikeCoin。你亦可以在去中心化交易所 Osmosis 購買更多 LikeCoin，並在此參考如何從 Osmosis 購買 LikeCoin。

當你安裝了 Keplr 並在錢包中預備了一些 LikeCoin 之後，餘下操作就很簡單了：

  • 將 Web3Press 插件安裝到你的 WordPress。
  • 在任何文章的側邊欄，點擊右上角的「拍手」按鈕。 點擊發布後，將出現一個彈出視窗。請依照屏幕上的說明進行操作。
  • 在此過程中，Keplr 錢包會觸發幾個彈出視窗。不用擔心，點擊批准 ( Approve ) 並讓系統通過。批准是授予 Keplr 使用 LikeCoin 鑄造 NFTs 的操作。
