柏拉圖先生在對話錄《理想國》(即《論正義》) 中提到幾個寓言，其中一個是黑暗洞穴寓言，改編下這個寓言，可以變成這樣。

很久很久以前

很久很久以前，有一個黑暗的洞穴，這個黑暗的洞穴有著迷宮一樣的結構，洞穴不是扁平的，而是有著複雜的立體結構，人們已經在這個黑暗的洞穴中生活了許多許多年，一代又一代都是在裏面繁衍生息。

幸好陽光雖然不能直接照射進洞穴，但洞穴不同層面還是能接受到一些陽光的折射再折射，當然，整個洞穴本質上是一個迷宮，越是深層的迷宮，陽光越難折射到裏面。

所以實質上，大部份在洞穴生活的人們看到的「事物」都只是一些模模糊糊的投影，他們從未見過陽光下的世界到底是怎樣。


飛機學大全

在洞穴中，人們的日常是他們以為他們見過「飛機」，但其實上他們概念中的「飛機」只是山寨版的「紙飛機」，除此之外他們對「真飛機」有多複雜一無所知，他們語言中的「飛機學」幾千年前都是圍著「紙飛機」開展的，他們對紙飛機研究了成千上萬年，已對紙飛機非常熟悉，並認為他們已經解決了飛行問題，並把名為「飛機學」實為「紙飛機學」按他們的理解寫成了「飛機學大全」，反正在他們語言中，飛機和紙飛機都是同一回事，都叫「飛機」，都能「飛」。

某程度上，也不能說人們錯了，因為紙飛機的確能飛起來，這給人們無限的信心，認為他們已經在這個領域走到了非常高的高度。


迷宮的 ABCD 區域

整個迷宮有多大，目前是沒有人知道的，但經過人類一代又一代的探索，到了比較上層的人們，把迷宮分成四個區域，並且把四個區域由深層到淺層標為了 A、B、C、D，而深層到淺層的主要區分就是陽光在該層的「含量」。

因為先天的限制，語言的限制，迷宮的限制，層與層之間的人們非常難溝通。

在人數上，A 層的人最多，D 層的人最少。

同時，D 層的人意識到 A、B、C 層的存在，是對迷宮最了解的人。

但 A 層的人則完全意識不到自己所在區域外還有 B、C、D 層，是對迷宮最無知的人。

在中間的 B 層人們知道 A 層的存在，並且擅長於利用 A 層的人為他們收集資源。

C 層的人知道 A、B 層的存在，同時隱隱約約知道迷宮裏還有一個 D 層。

D 層的人們在尋找 E 層，他們很肯定 E 層的存在。


我們很有信心

有趣的是 D 層的人們因為有更多陽光折射下，看的東西更清晰，所以 D 層人們承認「紙飛機」的局限性，會承認自己其實對「飛機」不夠了解，會承認目前的「飛機學大全」是不完備的。

反而是 A 層的人們卻日常信心滿滿，對自己對飛機的認識有著無比的信心。

折飛機戰爭

B 層的人，因為比 A 層的人對「紙飛機學」有更高一層的理解，所以 B 層的人們，利用他們的信息優勢，煽動 A 層的人們打了一場「折飛機戰爭」。

整個戰爭的源起，就是關於飛機怎折，主戰方有兩派人，第一派人認為「從右折到左」是正統，第二派人認為「從左折到右」才是真正的折法。

B 層的人知道其實無論「右折左」還是「左折右」出來的飛機是一樣的，但 A 層人們完全意識不到這點，並且被煽動起來打了幾千年的「折飛機戰爭」。

B 層的人利用這些沖突，去讓 A 層的人為自己獲取利益。而這「利益」其實源自於 B 層的人一個上千年留下來的信念 「帶血的紙飛機能飛得更高」。


喜劇與悲劇

而 C 層的人，看到這些則覺得很有趣，他們對 A 層的人為了「左折右」「右折左」而血流成河無法理解。

同時，C 層的人而對 B 層的人認為「帶血的紙飛機能飛得更高」也是無法理解，所以 C 層的人不時看到這些奇妙的行為與信念，就像小傻子一樣咯咯地笑，同時 C 層的人們又無比的悲哀，想人類為甚麼能如此的愚昩，我們 C 層是不是也是更大愚昩的一部份，在此喜劇和悲劇相通了，在瘋狂面前，悲和喜是可以交融的。


誰來當飛機學之王

而 D 層的人，看到 A、B、C 層的表現，感到無比的傷感，但他們人數最少，而且迷宮的限制令到他們無法好好和 A、B、C 層溝通，而在效率上講，D 層的人和 C 層的人享有更大的共同語言，反而是對 A 層人們一些莫名奇妙的行為想了半天也理解不能。

D 層人們作為目前的「飛機學之王」，有一天就商量「不如我們去管管他們吧」，然後 D 層的人們就開了一個大會，然後討論來討論去都沒有一個結果。

因為 D 層的人們，根本就不太想管 A、B 層的熊孩子，對他們來說，「不如我們去管管他們吧」的意思，就等同於「我要跟著一群熊孩子後面幫他們執手尾執屎執尿」，一想到這些他們就覺得超級可怕。

所以 D 層的人們雖然是「飛機學之王」，但他們推來推去，就沒有人真的想做「孩子王」。

有一個折衷的方法，就是 D 層的人們誘導 C 層的人去當這個「孩子王」，去管管 A、B 層的倒霉孩子。

問題是 C 層的人們也不是小傻瓜，在 C 層的人們眼中，A、B 層的熊孩子為一些莫名奇妙的東西都能打起來，誰愛管就誰去管，所以 C 層的人寫了一個「飛機王計劃」後就放棄了。


飛機王的誕生

又很多很多年之後，某一天，有一位 B 層的人，在路上不知怎撿到 C 層討論「飛機學之王」的設想，這位充滿天份並有著優厚家庭背景的人，對這個「飛機學之王」的構思深深著迷。

這位 B 層的人於是根據 C 層人們的「飛機學之王」山寨版計劃，利用各種不乏血腥的手段，征服了整個 A 層，並且讓 B 層的其他人都聽令於他。

如此，我們崇拜這位偉大的王，並在洞穴上刻下關於「飛機學之王」的壁畫。
