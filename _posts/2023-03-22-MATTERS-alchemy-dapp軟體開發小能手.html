公司簡介

Alchemy 提供一個平台給開發者製作去中心化的應用程式，簡單來說，就如同蘋果提供App Store 給用戶下載需要的應用程式，而Alchemy 提供給軟體開發者數項實用的工具，幫助開發者專注於產品的研發。除此之外，Alchemy 近期也對風險投資公司以及web3 的線上學習課程傾注心力，希望能夠擴大區塊鏈的生態圈。

提供的服務項目

Supernode

此服務的主要目的在於幫助公司進行大規模的開發計畫，並且不需要花大量心力在基礎設施的維護。首先，先簡單說明何謂node，當我們在以太坊區塊鏈上的交易的時候需要將交易紀錄提交到區塊鏈網路上，而區塊鏈網路又是由稱為節點的node 組成，這些鏈上的節點主要負責確認交易的效力並記錄於區塊鏈上。

而一般的node 在處理大量的資料時需要花非常大的時間成本，因此Alchemy 提供開發者Supernode 的功能，他具有強效的處理能力並且優化基礎設施給開發者，讓開發者可以塑造一個更加安全快速的交易環境，減低用戶對於資料安全的擔憂。

Alchemy’s Buid

提供給開發人員進行軟體內部調整及測試的服務，開發人員可以在裡頭製作應用程式的prototype 並且進行迭代測試乃至發佈產品。其中包含了以下四種工具：

  1. Explorer：類似於一款資源管理工具，透過這款工具能讓你在數百萬個歷史搜尋紀錄中找到你想要的特定資料，例如特定種類的錯誤等。
  2. Mempool Vistualizer：可以查看Mempool 中最即時的實況資訊，用以檢測失敗、阻塞或延遲的交易。
  3. Composer：讓開發者可以直接從儀表板送出JSON-RPC calls，透過回應的資訊來快速檢測問題。
  4. Debug Toolkit：透過快速偵測最近期的搜尋以及錯誤資訊並且視覺化，幫助開發者更快速進行迭代測試。

Alchemy’s Monitor tool

針對軟體開發商all-in-on 的工作介面，介面中簡單明瞭的查看開發者在Alchemy上開發、上市的所有App，並且提供更深入的數據分析，包含用戶的使用習慣、每秒搜尋速度以及App 的健康程度等等。透過每日的App 健康檢測報告可以避免並且掌握錯誤的發生，方便開發者進行優化即觀察。

Alchemy’s Notify tool

就如同我們現在使用各種應用程式相同，當我們在某個軟體上有異動時會收到來自城市的推播訊息，而這樣的技術也被放進Alchemy 的服務範疇中了，目前主打能自動推播鏈上的買賣交易明細並掌握燃料價格的變動、挖礦相關資訊甚至是交易失敗時的推播通知，讓用戶更快速掌握交易的流向以及細節，省去等待及查詢的時間。

對於軟體開發者而言，要自行維護軟體的基礎設施是非常耗費精神而且缺乏效率，而Notify 這項工具對於開發者來說把最煩人卻至關重要的程序取代，幫助開發者專注於產品的開發。

Enhanced APIs

首先，先簡單說明API 在網際網路或者是區塊鏈中的角色為何？以目前較為常見的網路訂購機票來說，當我們在網頁中選取各項條件，包括啟程時間、是否加購行李托運及餐點之外，就必須交由叫做API 的服務幫我們將需求送往資料庫中撈出符合需求的資料。在區塊鏈中的交易也如同樣道理，我們需要透過API 幫助我們快速找到鏈上需要的特定資料，不過由於區塊鏈中的資訊龐雜，如果倚靠一般的API 服務可能會耗時較多的時間，因此Alchemy 提供Enhanced APIs 的服務項目，目的就是降低搜尋的時間成本，而enhanced 顧名思義即是一種加強、擁有更強大功能的意思，因此以下將簡要分享幾項此產品的特點：

  1. 提供完整的交易紀錄：可以從整個區塊鏈中詳細列出在各個特定區塊中的交易明細，同時提供更詳細的資料洞察。
  2. 找到特定的帳號或地址：在廣大的區塊鏈交易中可以透過這款enhanced APIs 可以不必橫越整個區塊鏈，而是在單次呼叫之後獲得對應的資料。
  3. Debug更容易：提供像是一般網際網路的執行模式，更方便使用者操作。
  4. Token Identifier：透過呼叫API，可以得到token metadata 的重要資料，像是計算用戶的餘額與token 資訊。
  5. 支援Token allowance：支援透過API 來確認由買方所設定允許的支付金額。
  6. Alchemy Web3 ：Web3 j.s 最無痛的取代品，無論是介面配置及操作等皆可以方便與Alchemy 的其他服務連動串接。

Alchemy’s Amplify

即使現在於加密空間中以已經有許多與過去不同的觀點及優勢，但是在加密市場中創作者仍舊在發表作品時諸多的不便跟阻礙， 因此Alchemy 提供了一套Alchemy’s Amplify 的系統，其性質類似於我們目前主流市場中的App Store 或者是Google Play，我們可以在平台中找到各種我們需要的軟體或者是遊戲，而Alchemy’s Amplify 就如同這樣的商店平台替軟體開發者提供展示的平台。

在這項服務中主要以兩大主要管道替創作者提升曝光量：Supercharged newsletter 以及Social hype。其中Supercharged newslette r會將資訊、產品發送給對應的觀眾甚至是可能感興趣的用戶，這對於新的草創公司來說可說是一項極大的宣傳助力；另外Social hype則可以幫助創作者跟一些老用戶、高級用戶等建立良好的關係。除此之外，Alchemy 也提供創作者加入certified infrastructure alliance 聯盟中，在裡頭可以得到更多的保障。

這項工具出現之後不僅提供產品露出了機會也提升了曝光率，希望可以幫助創作者專注於軟體的開發，並且減少煩惱如何行銷作品的管道跟方式。另外針對新創公司，Alchemy 提供了Alchemy's accelerator program 讓團隊能夠在裡頭獲得相關的諮詢資訊以及web3 支援等，並且評估公司的成長空間

其他特殊計畫 - 線上教育課程

除了上述Alchemy 為程式開發者即用戶提供的各項服務之外，Alchemy 也致力於拓展、擴大web3 技術的人才。不僅設置了Alchemy Ventures 創投公司，也創辦Web3 University 並提供各種相關資源幫助像是新創公司等對web3 感興趣的群眾共同開拓區塊鏈生態圈。而課程內容包括：writing chain-agnostic smart contracts, mint NFTs, and build fully-functional end-to-end dApps capable of scaling 

目前此計畫也有許多合作夥伴例如Andreessen Horowitz, Pantera Capital, Arbitrum, Polygon, Flow, OpenSea, buildspace, Chainshot 等皆參與其中。

獲利模式

Alchemy 提供給DApp 的軟體開發者一個展示及展售平台和各項DApp 開發工具套組及基礎設施維護的服務作為主要獲利模式，除此之外，也推出各項付費版的高階功能幫助開發者有更穩定、有效的研發環境，將開發者維持監控基礎設施的時間省下來專注於DApp 的開發上，希望替區塊鏈帶來前所未有的體驗。

近況更新

近期Alchemy 持續精進、開發最新的技術支援開發者製作DApp，其中包含今年初才發布的工具Transaction Simulation，吸取2022年加密騙局的經驗，推出一款讓用戶及開發者都能更精準掌握及模擬的交易工具，除此之外，原先就有的功能Notify 也推出了V2 的升級版，增強並提升了可靠性、實用性以及安全性。另外，近年來也與不少公司有合作的關係，包含在數位運動、文化產業中活躍的公司DraftKings 攜手朝Web3 的時代前進。

馬克碎念

Alehcmy 對區塊鏈產業來說，意義非常重大。區塊鏈作為一個新興技術，需要有越來越多的實際應用場景，才能夠發揮其真正的價值。而開發去中心化應用程式，可以讓使用者更直接地體驗區塊鏈的優勢，例如：安全性、透明性、不可篡改性等等。因此，Alchemy 幫助開發者更容易地建立這樣的應用程式，可以加速區塊鏈產業的發展，同時也能夠讓更多人接觸到這個新興的技術，促進區塊鏈技術的普及和發展。

此外，Alchemy 提供的工具和資源也有助於提高區塊鏈開發的效率和品質，讓開發者更容易地建立高品質的應用程式，進一步促進區塊鏈產業的發展。馬克認為任何一項產業在快速發展之前，除了大量的需求出現之外，另一項重要的因素是基礎建設的發展，某個程度上來說，Alchemy 就是扮演著這樣的角色。

補充資訊：Web3的進展歷程

Web1至Web3的進程

第一階段（Web1）：紙本數位化

1989年英國電腦科學家柏內茲李（Tim Berners-Lee）發明了全球資訊網（WWW），也就是所謂的Web1 時代。在這個階段人們傳遞資訊的方式以「單向」進行，舉例來說，部落客以在網路上發表文章的方式分享個人的旅遊經驗，最主要的突破是讓大家可以於網路上搜尋到數位化的資訊，而不是只單靠紙本資料獲取最新消息，以更新速度來說已經是一大突破。

第二階段（Web2）：社群網路大勢崛起

雖然我們在Web1 階段得到了速度更快、獲取方式更便捷的優勢，不過讓人詬病是讀者無法給予雙向的回饋，簡單來說就是觀眾無法對資訊做評論，因此當我們進到Web2 時代以後各項社群軟體的興起Facebook、Twitter、Google到Instagram都讓資訊不再是單向傳輸，而是具有強大的互動性。目前我們所處的時代就屬於Web 2發展的成熟期，我們享有各種與創作者互動的渠道，無論是影片貼文的留言回覆，或者問答類等參與式的功能。

不過，無論是Web1 還是Web2 階段都存在著一個巨大的問題：創作者在分享資訊後僅能從平台上得到極少的利益。這問題的原因在於創作者在平台發表作品時需要透過一個中心化的系統去操控，也就是說作品需要依據系統的遊戲規則走，在社群網路日漸蓬勃的今日，我們常看見某則影片因涉及色情內容而遭到平台黃標的警告，這將導致創作者無法得到本部作品的流量收益，除此之外，系統的規則也不只是限縮瓜分走創作者的權利，也同樣可能造成觀眾言論自由的損害，舉例來說在各大平台上我們因為某些言論受到系統偵測不當、「違反平台規則遭到移除」的警示也層出不窮，甚至被平台移除帳戶等。因此，為了解決被某個中心化平台限制操控的情況出現，Web3 便成為未來最佳的解決良方。

第三階段（Web3）：去中心化且高度自治

目前此階段的發展還在草創至發展期，因此目前人們將「以區塊鏈為底層技術」的所有應用皆含納入Web3 概念中，如同前述中心化系統對使用者的種種限制，在未來Web3 的時代裡人人將不受審查並且透過區塊鏈的技術做資料的驗證和公平的分潤。也就是說，創作者不必再受限於因平台的抽成而降低收益，而是直接透過區塊鏈內的觀看紀錄直接公平的分潤給創作者；另一方面，觀眾同樣也能掌控個人資料的主控權，包含觀看紀錄、興趣內容及喜好作者等，這些數據都可以由用戶主動提供再請平台反向提供推薦清單 。

目前對於Web3 沒有一個精確的定義，只能廣義評估以區塊鏈做為底層技術的產品皆屬於Web3 的範疇。目前Web3 有幾項特點分別是：改變人類過去習慣儲存、紀錄的方式、個資所有權以及流通性以及講求共識信任的制度。其中資料所有權及流通性的特性好比今天我們能在自己經常使用的購物平台中提供自己歷年的購買紀錄給特定的賣家或廠商，讓他們參考這些你主動提供的資料之後推薦販賣可能合適你的產品，而這些資料的提供是可以由用戶自行決定而不是被平台或第三方操控。

總而言之，Web3究竟對我們未來的生活有多少影響力？我們生活的方式又會如何被改變呢？ 

現代人習慣也喜愛在各大串流平台上收看各國的影視作品，你是否有思考過這些從平台購買版權而來的作品是否真的有獲得相對應的回饋呢？簡單來說，暫居排行版首位的作品與相對小眾的作品現實中就是會有迥異的點擊觀看率，無論點擊率高或低這些都在平台收購的當下無法更動了。而未來若進入到Web3 成熟的世界，我們將不再受制於訂閱某家串流平台才能觀看電影，而是在我們點開某個作品後，觀看紀錄將會紀錄在區塊鏈上並且根據這項紀錄支付相對應的版稅給作品創作者，這樣一來收益將可以達到更公平並不被平台抽成的狀態。

不過這樣的烏托邦目前也僅能假想並且期待，目前Web3 還尚未成熟，雖然針對加密貨幣的管理各國皆已漸漸有相關的法律監管政策監察，不過想要達到蓬勃發展的境界仍還有一大段路程要走。
