作者與讀者的關係很微妙。文字和藝術創作的特性，是通過媒介如文字、聲音、畫面等表達作者豐富的情感。讀者若理解作品，產生共鳴，便猶如深入作者的內心世界，甚至比頻䌓相見的友伴更親密。生活中總有些人幾乎每天相見，思想上卻從沒深度交流；偶爾在數位世界以文會友，卻反而喜獲知音 – 習慣在數位世界生活的各位可能也有過相似的感受吧？

然而隔著創作媒介，作者與讀者之間卻保持著一定的距離感，沒有面對面的壓力，也不存在時間的拉扯。因此以文會友是既深刻又輕省，既近且遠的人際交往方式。

出版成 Writing NFT 的文章，也附帶讓作者和讀者通過特別的媒介互動留言的功能，卻不像討論區或電郵般有回覆的催迫感。這種溝通方式有如物理世界中，作者在書的封底寫下「謹把此書送給我親愛的太太」、贈書時附上一張寫下祝福語句的書籤等，只是「親筆」換成了在 Web3 世界代表身份的錢包地址。

在物理世界中沒有方便的方法讓讀者留言給作者。在書店買了一本書後，作者不會知道是誰買了，讀者也無法留下隻言片語。然而，現在 Writing NFT 也推出了讓讀者在購買一刻留言的功能，再添一個支持作者的渠道。鼓勵各位讀者在閱讀，收藏作品，寫下感想或任何想向作者表達的說話，留言將成為作品生命的一部份，永存在鏈上。

如何在購買 Writing NFT 時留言給作者？

步驟一：在「留言給創作者」欄位輸入給作者的說話

步驟二：選擇「用 LIKE 購買」或以信用卡付款

收藏者留言將顯示於作者留言之上。

群發 NFT 紀念品給支持者

給讀者最好的紀念品，就是「親筆簽名」的作品。但作者如何能獲知所有支持過自己的讀者的錢包地址呢？而且，逐筆發出作品的 NFT 豈不十分費力？

最近 Liker Land 推出了「導出收藏者數據」的小工具，讓作者可導出曾收藏過自己任何作品的所有錢包地址，然後作者可利用另一小工具，一鍵群發 NFT 紀念品給讀者，回饋支持。

Liker Land 將陸續推出更多協助作者經營社群的小工具，培養作者與讀者間的關係，敬請期待。大家不妨考慮向支持者發送「賀文」呢，因為新年快到了！

如何群發 NFT？

步驟一：在 LikeCoin NFT Dashboard 搜尋（Search）自己錢包的收藏者 ( Get Collector )。

步驟二：導出所有數據 ( Export all data )。

步驟三：將數據的 csv 導入試算表工具進行分類與排序，並將支持者錢包地址整理成清單。

步驟四：在 LikeCoin NFT Marketplace 的介面工具 ( Tools ) 中選 Send NFTs 並登入 Keplr。

步驟五：在用以群發給支持者的 Writing NFT 找出 NFT Class ID。

步驟六：將 NFT Class ID 輸入 Send NFTs 工具，並輸入支持者的錢包地址清單 ( Recepient Address list ) 及留給他們的話 ( Transfer message ) ，再按「Send」及在 Keplr 簽署，即可群發 NFT。
