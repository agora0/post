1. A 类型

A 类型是最常见的类型。他们的分布没有明显特征，或者准确地说，到处都是。相对应的，他们也是相对来说对人威胁最小的类型——并不是因为他们态度较为友好，只是他们的感知力过于差了，一般无法在没有其他类群引导或者有人类主动暴露身份时主动感知到异类的存在。尽管如此，仍然不能对他们掉以轻心，A 类群的数量效应是非常恐怖的；而且，有不少案例证明其他判断力与反应力较高的类群会利用 A 类群发动攻击或达成其他目的。

A 类型的表现特征很容易识别。他们只会复读一些简单的句子，这些句子您在日常生活中或者是在教科书中经常能看到。如果出现了其他在引导他们的类群，他们的用语会向那个类群靠拢。但是，谨记——一定不要出于同情心或者私人关系等种种理由试图自己去应对或者“话疗”A 类群，除非您有专业的相关经验，因为他们的接受能力和思考能力同样地差，这样做很难得到结果，反而容易暴露自己：即使他们表现得再人畜无害，在思维逆行清除病毒的过程完成之前，感染者始终是危险的。

请时刻记住：您没有义务，也不一定有能力帮助感染者完成思维逆行。时刻优先保证自己的安全。

2. B 类型

尽管数量少，B 类型曾经在 10-20 年前占主导地位；正如我们之前所说的，A 类型容易被其他智力和知觉更强的类型引导，因此尽管数量大，也总是其他类型占主导地位。不过近几年来，由于温斯顿计划导致的大量 C 类型的异军突起，他们的地位被完全替代和压制了，关于该计划和两类型之间的关系，下文讲到 C 类型的时候还会提到。

B 类型的特征在于传统叙事。注意，这里说的传统上至两千年前，下至近代的二十世纪初；在地域上可以是本土传统，也可以是异域传统。总之他们的发言会围绕着传统价值观出发，比 A 类型的重复性极高的呓语更容易引起人的亲切感，也更具危险性；同时，也由于 B 类型感染者的年龄普遍较大，绝大部分的 B 类型年龄都在 35-40 岁以上，他们有时会以您长辈或是上司等高社会地位人物的身份出现，这时一定要记住：不要受私人感情影响，无论您是出于不平还是想帮助他们的心理。尽力隐藏自己的身份，暴露的后果是不可估量的。

今年是温斯顿计划实施的第十个年头，受计划影响，大部分 B 类型要么被无害化、要么大大降低了传染性、要么已经转移根据地到海外。然而，仍然不能太轻视他们可能造成的影响以及潜在危害。

3. C 类型

C 类型的历史并不漫长：他们仅仅起源于不到一百年前，并在 40-50 年前由于【已删除】事件趋于衰微，被敌对的 B 类型压制，直到 10 年前温斯顿计划的启动。时至今日，C 类型的数量仍然在不断增多。

C 类型和 B 类型之间是不可调和的完全敌对关系。这种对抗的根源在于 C 类型本质上是被【信息已加密】主要为了应对 B 类型创造出来的独立毒株，也正因此，C 类型的主要表现型体现在对其创始人本人的无条件崇拜，即使是【信息已加密】死后亦然；但在他死后，多数 C 类型感染者开始逐渐失去控制，温斯顿计划的实施也未能重新掌握他们的控制权。

C 类型相比其他两种类型更喜欢结群行动，让他们的危险性和破坏力成为三种感染者当中最高的。如果你偶然遇到 C 类型组成的集群，不要与他们过多接触，若是他们主动纠缠你，尽快想办法摆脱，尤其注意不要在他们面前提及几个禁忌词，或【信息已加密】的名字。如他们主动提及，可以跟着应和，但万万不可提及【信息已加密】的任何蔑视性称呼，或对他使用任何否定性的措辞。记住，在任何时候，你都无法准确地预测感染者的行动，所以无论你对【信息已加密】的真实态度如何——为安全起见，不要在他们面前表露出来。

4. S 类型（特别型）

S 类型可以说是起源于温斯顿计划，也是四种类型中最特别、危险性总体上来说也最大的一种。有说法认为 S 型的毒株是基于 C 型特征中对【信息已加密】的崇拜制造出来的，旨在制造一批新的、能受控制的感染者。关于计划本身的成功与否，本文在此不作评价，但 S 类型最终脱离了计划的控制是个无可否认的结果。试图通过对【信息已加密】的崇拜来完成类似【信息已加密】之于 C 类型的作用已被证明不能长久生效：计划的最初几年，S 类型表现得似乎相当顺从，然而当他们被要求完成更多、更复杂的行动——尤其是“保持镇定”时，S 型中的绝大部分却出现了包括而不限于不安、躁狂、轻中度癔症在内的情况。

鉴于【信息已加密】身份的特殊性，其派出代言人试图停止 S 型类群的异动，然而未达到平息效果的同时，【信息已加密】的代言人们甚至受到了不同程度的攻击。因此，试图控制 S 型感染者的计划以失败告终，S 型类群们被遗弃和切断联系。

分辨 S 类型与 C 类型没有特别明确的办法，但某些时候，由于 S 类型独特的危险性——在年龄、数量、威胁度上比 C 型显著更高——你可能必须明确地区分二者。当然，我们不鼓励和感染者产生任何不必要的接触；但如果你由于某些需要，不得不和疑似 S/C 型的目标进行接触，务必确定目标更接近哪一型。一个大致的区分方式是其更多提及【信息已加密】还是【信息已加密】，或者从其“言辞”上区分，但不能保证准确性。建议如无十分必要的情况，不要接近疑似 S 型或者 C 型的任何一个感染者。请各位读者记住，在目前的局势下，保护自身安全最重要，不要因为感染者曾是/现在是你的亲人、朋友而试图对其产生多余的接触。

5. 结束语

用一句话来总结四种类型：迟钝的是 A 类型，守旧的是 B 类型，固执的是 C 类型，和 C 类型相似但更危险的是 S 类型。由于缺乏详尽的统计，我们无法估计近年来感染者的数目以什么样的趋势变化，甚至是增是减也不能明确得知，唯一能确认的是，即使是最迟钝的 A 型，活跃度也肉眼可见地一直在急剧上升，更不用说其他类型了。请读者们保护好自己和身边的人，尽可能地避免与任何类型感染者的非必要接触，尤其不要受感性影响，由于一时赌气或感染者与自己的私人关系贸然暴露身份，也不要擅自尝试对任何一种感染者私下实行思维逆行解除病毒，即使其看起来再“在控制之内”，这不是属于你的责任；世界并不会因为多一个和少一个感染者而变得更好或更差，但“你”很重要。

意识形态病毒是人类自作孽的产物，只有“我们”——全体剩余的健康人类，共同努力，和使用恰当的方式，才能真正化解危机，保护自己和重视的人。请各位谨记。
