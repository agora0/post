Firefox version 109

about:config

entension.unifiedExtensions.enabled

set its value to “false”

restart Firefox

Firefox version 111

about:config

toolkit.legacyUserProfileCustomizations.stylesheets

set its value to “true”

New Folder — “chrome” (Profile Directory)

New File — “userChrome.css”

Paste :

#unified-extensions-button, #unified-extensions-button > .toolbarbutton-icon{

width: 0px !important;

padding: 0px !important;

}

restart Firefox
