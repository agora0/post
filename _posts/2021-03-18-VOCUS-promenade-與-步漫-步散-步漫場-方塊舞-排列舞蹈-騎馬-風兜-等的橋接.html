完整標題：promenade 與「步漫」、「步漫為動移」或「陪(伴)人于漫為動移」或「步散為動移」或「步散」或「步漫場」或「方舞蹈」(方口舞、方塊舞) 或「排列舞蹈」或「排列于多人為動移」或「步鄰于海岸(沿)的道」或「濱鄰于海之人的道」或「跑馬以人為動移」或「騎馬」或「風兜動移」或「風兜」等的轉換密碼

【附圖：「步、共、散」字形演化】

1. promenade = pro.men.a.de = foot連偶.曼.合.點三 = 步.曼.合.氵= 步.曼合氵= 步.漫 = 步漫，即漢字「漫步」ㄧ詞的同義倒裝詞。其中 p 通過 ph 與 f 同音的關係代表 foot 而意通漢字早期類似「屮」形或「止」形而代表足部或腳印的符號，所以此處 pro 所轉換的「foot連偶」即代表一個「二足相連」的符號亦即符號「屮屮」上下組合或符號「止止」上下組合所構成的漢字「步」之初文。

2. promenade = pro.men.ade = 步.漫.為動移 = 步漫為動移，也是類似「漫步」之意。其中 pro 同前述轉換漢字「步」，men 則約略形聲漢語「漫」。此外，拼音文字的字尾 ate 常常有「為作」等表示作為、動作或動態的動詞語意，字尾 ade 則可能有「為動移」或「為做之」等表示為於移動或做某件事之類的動詞語意。

3. promenade = p.r.o.men.ade = 陪 or 伴.人.于.漫.為動移 = 陪人于漫為動移 or 伴人于漫為動移，類似「陪人漫步」或「伴人漫步」之意。

4. promenade = p.r.o.m.e.n.ade = p.r.o.moon.e.n.ade = 步.卜.又.月.ㄧㄇ.為動移 = 步.卜又.月.ㄧㄩ.為動移 = 步.攴.月.廿.為動移 = 步.攵.月.龷.為動移 = 步.攵月龷.為動移 = 步.散.為動移 = 步散為動移，類似「散步」之意。漢字符號「共」本爲代表物件的「口」形符號或雙十狀的「十十」形或「廿」形符號與類似「屮屮」形的雙手符號之組合，後來篆文演化爲「廿」與「屮屮」形上下組合，然後又演變爲「龷」形符號，所以此式中 en 轉換的「一ㄇ」可能組合表示「共」字早期字體中的「口」形符號或形似「ㄧㄩ」組合的「廿」形符號，也意通後世「散」字中的「龷」形符號。(P.S. 參考【附圖：「步、共、散」字形演化】)

5. promenade = p.r.o.men.a.de (此處 men 約略形聲 moon) = p.r.o.moon.a.de = 步.卜.又.月.合.竹 = 步.卜又.月合竹 = 步.攴.月合竹 = 步.攴月合竹 = 步.攵月竹 = 步.散 = 步散，即漢字「散步」的同義倒裝詞。此處 promenade 中的 romenade 是在描述漢字「散」的金文字體之一，其中字尾 de 約略形聲台閩語「竹」。漢字「散」的甲骨文和篆文較接近，都帶有符號「林」，金文則以「竹」或「艸」或「木屮」等代替符號「林」的部份。(P.S. 參考【附圖：「步、共、散」字形演化】)

6. promenade = pro.men.n.na.a.de  省二個 n 省一個 a = 步.漫.勹.聯1.A.旦 = 步漫.勹.11.Λㄧ.旦 = 步漫.勹.//.土.旦 = 步漫.勹//.土.旦 = 步漫.勿.土.旦 = 步漫.勿土旦 = 步漫.場 = 步漫場，類似「漫步場」、「散步場所」之意。其中符號「Λ一」的上下組合即約略象徵漢文「土」的甲骨文字體之一。

7. promenade = p.r.o.m.e.na.de = 方.人.和.卌.一.聯夊.蹈 = 方.𠂉.和.卌.一.夊夊.蹈 = 方.𠂉.和.卌.一.舛.蹈 = 方.𠂉和卌一舛.蹈 = 方.舞.蹈 = 方舞蹈，類似「方塊舞」之意，又簡稱 prom (方連口舞，即「方口舞」，意通「方塊舞」)。

8. promenade = p.r.romenade 省一個 r = 排.列.舞蹈 = 排列舞蹈，其中 romenade 同上述「轉換式7」所言橋接於漢字「舞蹈」，但是否覺得也帶有一點「浪漫」或漢字古語「八佾」的語意呢？頗爲耐人尋味！(P.S. 參考〈ballet (芭蕾舞) 與「八列蹈」、「八列舞」或「八佾舞」及「八佾」的轉換密碼〉 -- https://lsdesignworld.blogspot.com/2017/05/ballet.html?m=1)

9. promenade = p.r.o.m.e.n.ade = 排.列.于.多.與.人.為動移 = 排列于多人為動移，意指「多人列隊而前進移動」。其中 m 轉約 45 度象徵漢字「多」，n 約略象徵漢字「人」的篆文字體。

10. promenade = p.r.o.me.en.a.de 省一個 e = 步.鄰.于.每E.岸(沿).的.道 = 步.鄰.于.每三.岸(沿).的.道 = 步.鄰.于.每氵.岸(沿).的.道 = 步.鄰.于.海.岸(沿).的.道 = 步鄰于海岸(沿)的道，類似「濱海步道」或「沿海步道」之意。

11. promenade = p.r.o.me.e.n.a.de 省一個 e = 濱.鄰.于.海.之.人.的.道 = 濱鄰于海之人行道，類似「濱海人行道」之意。其中 a 通過 Aa 約略象徵漢字「的」。

12. promenade = pro.m.e.n.ade = 跑.馬.以.人.為動移 = 跑馬以人為動移，類似「騎馬」之意。但也可能直接轉換漢字「騎馬」，即：promenade = 平round偶馬一人直 = 一圓馬馬大丨= 一〇馬馬大亅= 一囗馬馬大亅= 騎馬。此義可能來自騎馬的時候，馬背上的人也會有一種類似波浪起伏的浪漫之感覺。

13. promenade = p.r.om.mena.de = p.r.om.nnena.de = 凡.入.worm.nn與n白.動移 = 凡入虫.ﾧℶ與人白.動移 = 風.ﾧℶ與ㄦ白.動移 = 風.兜.動移 = 風兜動移，類似「兜風」之意。但是也可能直接轉換漢字「兜風」的同義倒裝詞「風兜」，即：promenade = 凡人和ㄇ一聯1兜 = 凡λ和口聯丨兜 = 凡λ和中兜 = 凡虫兜 = 風兜。其中 p 通過「p = lo = Λone = Л一 = 几ㄧ = 几丶= 凡」轉換漢文「凡」，但字母 p 在轉換漢文「凡」時也可能直接通過 ph 之類似 f 的發音約略形聲漢語「凡」的起音。
