【獨媒報導】COLLAR 成員 So Ching（蘇芷晴）今日在社交網站發文，稱男友阿MO受傷後，留下無法被療癒的創傷，宣布退出 COLLAR。餘下成員下午在 ViuTV 錄影廠召開記者會交代詳情，引來粉絲在門外等候。粉絲中五生 Nick 放學後趕來九展，希望向成員表達鼓勵。他坦言，COLLAR 今年年初齊人回歸，以為她們能夠永遠齊齊整整，惟 So Ching 突然宣布退出，難免感到難過，但會尊重及支持她的決定。他又祝福 So Ching 未來生活順利，繼續當 COLLAR 成員是好友。

So Ching 今日在 Instagram 發文，指因阿MO受傷的創傷一直無法被療癒，在心裏留下烙印，成為不能磨滅的痕跡，至現時仍未能完全適應幕前工作，決定退出 COLLAR。So Ching 所屬公司 MakerVille 今日下午3時舉行記者會交代詳情，COLLAR 經理人阿 Wing 確認 So Ching 退出，公司及成員均尊重其決定，並祝福對方在日後找到自己方向，並鞠躬感謝支持。

記者會完結後，有 COLLAR 粉絲來到九龍灣國際展貿中心 ViuTV 錄影廠外等候 COLLAR 成員離開，希望向她們表達鼓勵。

中五男粉絲望So Ching 不要忘記COLLAR 成員

在大圍上學的17歲中五生 Nick，今日得悉 So Ching 退團後，便來到九展ViuTV 錄影廠外等候 COLLAR，希望跟她們說聲「加油，唔好放棄」。Nick 又對於成員陳泳伽（Winka）稱「8-1=0」即歸零重新出發表示認同。他指，雖然現時失去一塊拼圖對整個團影響很大，但認為 COLLAR 得來不易，不應解散和不應放棄。

雖然 Nick 的「本命」是蘇雅琳（Ivy So），但 So Ching 是他曾經喜歡過的成員之一。他希望 So Ching 未來可以繼續在幕前工作，不論是拍電視劇或廣告，還是拍跳舞片，「希望可以喺電視再見返佢。」他又想跟 So Ching 說：「永遠支持同尊重你嘅決定，無論前面嘅路點樣，都希望你順利，希望你多啲同 COLLAR 聚舊、傾吓心事，當佢哋係一個好好嘅朋友。」


粉絲Nick

以為可以齊齊整整、重新嚟過 對So Ching退出感到難過

COLLAR 去年1月12日出道，但出道約半年，去年7月 MIRROR 演唱會發生墮屏事故，舞者李啟言（阿MO）嚴重受傷，現時仍在留醫。作為阿MO的女友，So Ching 停工照顧對方，至今年年初才復出參與演出，COLLAR 到今年回復8人工作。

COLLAR 今年軍一周年，Nick 坦言 COLLAR 今年全面復工後，很期待她們的歸來，感覺到「個團火重新再嚟過」、「終於可齊齊整整」，惟復工不足半年，So Ching 突然退出，亦難免感到難過及可惜。不過，Nick 認為自己不是 So Ching ，難以明白她的傷痛，既然她最後決定離開，亦會尊重她的選擇。


資料圖片

對於今日由 COLLAR 其餘7名成員主持記者會，Nick 認為她們有交代的需要，但認為 MakerVille 高層應同樣現身交代事件，及就 MIRROR 演唱會事件道歉，「始終所有事都係由演唱會事件而起。」他又稱，未見公司在事故後向 So Ching 作出任何援助，認為公司需要解釋。

COLLAR一周年才看到8人有遺憾 「嗰次係第一次亦係最後一次」

從未喜歡過其他國家偶像團體的 Nick，稱香港一直缺乏本地女團，對於香港終出產女團感到十分光榮。雖然友台亦有出產女團，但 COLLAR 成員較成熟，每位都有個人特色及長處。他又大數每位成員的優點，如隊長沈貞巧（Gao）和李芯駖（芯駖）表現得體、成熟穩重、陳泳伽（Winka）有很好的歌喉、邱彥筒（Marf） 和 So Ching 不單跳舞出色亦能唱歌。

Nick 對 COLLAR 十分着迷，他大方向記者展示收藏夾，裏面放滿 COLLAR 成員的應援海報，又在書包拿出 Ivy So 襟章。他向記者分享追星的成果，展示他與 Ivy So 的合照，及她的親筆簽名。Nick 又提到在 COLLAR 成軍一周年，到西九海濱草地觀賞他們的新歌表演，他指哪天是第一次一次過看到8名成員，但他慨嘆「嗰次係第一次亦係最後一次」。

粉絲稱感受到So Ching復出後仍顧慮

不願上鏡的40歲 COLLAR 粉絲 Jack，今日亦同樣來到錄影外等候 COLLAR，希望跟她們說加油。Jack 稱自己喜歡阿Gao和心駖，但當初最先留意是 So Ching，因為她跳舞最好看。不過，Jack 指自從 MIRROR 演唱會事故發生後，就算 So Ching 復出繼續工作，都感受到她「因為阿MO沉咗」、「有顧慮同陰影，佢繼續做都未必做返以前咁」，認為 So Ching 退出對她自己及成員都有幫助，自己亦明白她的決定。

他又對於 MakerVille 行政總裁魯庭暉未有在今天的記者會交代 So Ching 退團及演唱會事件感到不滿，認為公司「側側膊」避開公眾。
