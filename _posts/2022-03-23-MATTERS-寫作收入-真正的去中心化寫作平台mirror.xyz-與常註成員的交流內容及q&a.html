Mirror.xyz，集募資、寫作、NFT發佈、發幣功能於一身。

前天聆聽編輯的意見，直指模式非常 Web 3.0 的 Mirror.xyz 將會成為未來主流，於是開始試用 Mirror.xyz 來寫作，順便到他們的 Discord 逛了一圈，留下新上傳的文章，獲得頗有趣的體驗：

沒有索引、沒有編輯推介、沒有熱門推薦，為何這樣的平台將會是 Web3.0 的未來呢？

一個常在 Mirror Discord 出現的常駐成員 Tian 在這時與本熊搭話，因為他發現本熊的作品會同時上載到「方格子」和「Matters」，但他似乎不認識這兩個平台，於是好奇平台的人口結構，我們便開始交流。

與常駐成員 Tian 的交流內容

以下的對話將經過一些修改，以加強閱讀流暢感，另外在這個框架下的對話是本熊的吐糟。

熊太：
我覺得「方格子」和「Matters」大部分都是台灣使用者，香港用戶相對上其實比較少，特別是前者。另一方面，眼見也有不少中國用戶會在 Matters 上發文，但我不確定人數。

Tian：
嗯嗯，Mirror 的港台作者好像比較少，熊太清楚是什麼原因嗎？大家沒聽說過 Mirror 還是習慣了另外兩個平臺？

熊太：
我覺得兩個平台都不是大型寫作平台⋯⋯就是比較中型、小型的。
台灣人少用 Mirror 的原因，可能是因為界面是英語而非中文的問題。而香港人之所以會少用，應該是因為比較著名又能賺錢的香港創作者，都會比較愛用 Patreon 和 Penana 這種曝光和推廣比較高、搜索方便、文章分類方便、可以直接社交、又賺錢簡單的平台⋯⋯
當然如果你只想討論 Web 3.0 的寫作工具和作者，那你的想法大概也正確，雖然方格子還是十分 Web 2.0。

順帶一提，有些人會認為 Matters 的開啟圍爐（付費訂閱）門檻低，但其實Penana 那邊連審核也不用，就可以開始收費和打賞功能，而方格子的付費訂閱功能開啟審核幾乎跟 Matters 一樣，不一定要成為 Premium 會員或作者。

Tian：
所以 Mirror 既沒有捕獲高流量的作者，對小、中型作者的吸引力也不大。其實熊太可以在方格子和 Matters 上寫篇介紹 Mirror 的文章，或者比較的文章，可能大家並不清楚有這麼一個平臺。

熊太：
本來小、中型作者就會比較想要流量，讓更多人認識。他們沒什麼資本，可以讓自己慢慢地在沒什麼人發現的地方中創作。

說起來，比起以前默默耕耘的創作者，現在的網上創作者似乎更在意收入和收益。那寫作的初心呢？如果只是為了利益而寫作，到底可以走到多遠，文章的深度又能夠去到哪個位置呢？

Tian：
但其實現在 Mirror 中文圈比較適合小、中型作者。如果內容優質，我們還可以幫忙傳播的，所以不用擔心這裡是沒什麼人發現的地方。

熊太：
我也有這樣的感覺。不過，如果沒有發現這個 Discord，我都不知道要怎樣找大家寫在 Mirror 的文章。

Tian：

哈哈，其實找不到才是保護小中作者。記得之前讀過一本書，提到「隨著網路的流行，大家花點錢就可以在互聯網聽一場盛大音樂會，越來越少的人願意去聽一個本地樂團的演出了。」
有索引，大流量作者 Catch All，小流量作者就真成了時代的一粒沙了。

熊太：
我覺得現在太容易找到消息，演算法又被大手持有，早就讓世界變成大手遊樂園，而 Web 3.0 正在糾正這個狀態，不過要習慣和說服到大眾還有很長的路要走。

Tian：
而且可能又會走回去，因為雖然大家覺得自己的喜好被演算法指引，但是這也讓大家獲取資訊更容易。現在 Mirror 不提供內容導航，結果很多人都問怎麼看到 Mirror 的文章……
所以，要去糾正人們的習慣非常非常困難，很可能反而向大眾的習慣妥協。比如 Mirror 現在的每週好文精選，選中的作者會獲得 $WRITE 獎勵，勢必會成就一部分作者而損害另外一部分作者。
Mirror 應該一直尋找一個平衡點，既能喚起大家的主動意識，又能盡可能的便利大家。

燃燒一枚 $WRITE 可以獲得 Mirror 子域名（ENS）和 DAO 成員資格。簡而言之，Mirror 提供作者工具，為作者支付 AR 存儲費用、鑄造 NFT 技術，並會對入選 Spotlight 精選的優秀內容獎勵 $WRITE。

熊太：
其實這已經很理想。凡事不用一步到位，因為時代一直在變，現在適合的狀態在未來也不一定適合，所以一直試著找一知平衡點，找到後又繼續改進才是理想狀態。
在 Web 2.0 的世界中，其實大家都被慣了。以前會因為一些需要或好奇而會主動去探索，例如想聽音樂，會到 CD 店試聽，從而發現更多的可能，但現在基本上都是待別人送東西進自己口裡眼裡，要找資源時靠演算法就能集中找到想要的，眼光都盯著首十個搜尋選項，結果連去搜索更多可能性的行動都消失了，眼光也理所當然地變短淺。
嚴重一點的人，甚至是所有事都依靠懶人包，連閱讀和自己嘗試也懶了怕了，結果水文愈來愈多，Facebook、Twitter 等因而極速冒起，人類就這樣跌入了一個個緊扣在一起的陷阱裡。

Tian：
We're a human. Not a machine!
說到這裡，我覺得挺無語的，大家遇到什麼第一個想著就是去問，而不是去探索，去實驗，去嘗試。General help（Mirror Discord中的其中一個頻道）裡很多老外問的問題，其實只需要自己寫一篇文章就可以驗證，結果他們第一步是問。

對於凡事先問，而沒有先做好資料搜集或嘗試的人，一般會被稱為「伸手黨」。

熊太：
就像很多人會到 General 聊天頻道問：這個 NFT 企劃怎拿白單，然後要別人 tag 那個頻道給他們才會自己走過去看一樣。但其實，那個頻道就在 General 的上面不遠處！想起某個香港歌手曾說過：不明白為何現在的人可以無知得這麼光明正大⋯⋯
第一次使用 Mirror.xyz 發文，我對寫作編輯器產生好感，使用方法有點像 Notion，我覺得這是個重要優點之一，至少比 Matters 的編輯器要好用太多了。

Tian：
嗯，編輯體驗的確好一些。不過對於想用原生 Markdown 的創作者來說就不能會感到不適應，因為 Mirror 的編輯器直接給預編譯實際效果，結果大家看不到 md 的原始程式碼。

Q&A

本熊搜集了一些現時常會被問到的部分，並把群內成員的解答放在這裡，以方便大家認識 Mirror.xyz。

問題一：我在 Mirror 上寫的文章，怎麼分享給其他人看呢？同時，試著發起了一個眾籌，要怎麼能讓大家看到？

答：想要更多人看見，就需要自己把連結分享到例如 Twitter、Facebook 的平台，Mirror 只是提供一個內容發佈平臺。一般大家把原創內容發到 Mirror，也有人多個平臺同時發文（Mirror 和其他）。 眾籌也是一樣道理，不過有不少願意課金的人會關注 bress.xyz/ 來認識 Mirror 上的項目。

本熊十分同意「在其他的網站中，沒有被推薦或推送至首頁的文章跟透明沒分別」這句話，所以在 Mirror 平台中，所有創作者將會因為其機制而變得地位相對平等，反之就是任何人都能夠擁有一個「機會」。對於這個話題有興趣，建議亦一同閱讀鴻雁寫的《方格子的互動冷清，其實不是缺點，是優點──以Matters為比較對象》。

問題二：是否所有內容都必須要做成NFT，其他人才能看到？作品上鏈後是否無無法刪除？

答：不需要 NFT，除非你想眾籌或者 NFT 打賞。你的文章將會直接存到 AR，而你不需要花費一分一毫，因為 Mirror 已經為你支付了費用。上鏈了內容就永久在AR，你可以在 Mirror 修改或刪除文章，但是刪除的文章內容，仍然會保留在 AR。

問題三：如何獲取 $WRITE？

答：一、文章在 Mirror Discord 中被精選；二、做生態專案被認可；三、二級市場購買，如 Uniswap，不過沒啥流動性，而且很貴。

問題四：Mirror Discord 中的「本周好文聚焦（精選）」活動是每周都會進行嗎？下周如果想投稿還可以參加嗎？

答：是每周進行。可以參加。

問題五：怎麼能成為 Mirror DAO 的成員呢？

答：燃燒一枚 $WRITE 可以獲得 Mirror 子功能變數名稱和 DAO 成員資格，就可以在 Discord 中申請這個成員組。

問題六：Mirror Discord 中的中文頻道似乎都是簡體中文使用者，繁體中文使用者能夠加入嗎？

答：可以。Mirror 是去中心化寫作平台，沒有審核，所有人都可以加入，並進行創作──不管是否暱名。怎麼用Mirror都可以，沒有什麼限制。Mirror並不對內容審核，這個是創作者自己的事情

問題七：要用什麼鏈及錢包連接 Mirror？

答：必須使用以太鏈。另外你可以使用最普遍的狐狸錢包連接 Mirror。 問題八：ENS是什麼？

答：ENS即以太坊名稱服務(Ethereum Name Service) ，通過為您提供與您的錢包相關聯的易於記憶的地址（有點像域名，亦與Unstoppable domains性質類似）來解決這個問題，您可以使用它在 Web 3.0 平台上進行交易。 一旦您註冊並將其連接到您的 mirror.xyz 帳戶，這將成為您鏈接到您的內容的共享地址，以及您的觀眾可以提示您的貢獻的一種方式。 在註冊期間，您可以選擇在續訂之前要支付的年限，年費非常低，可是 Gas Fee 很高，差不多 USD 150美元，如果不想每年都付那筆 Gas Fee，並確定會經常使用，可以先訂閱長一點的年限。 （本答案引用了《中文翻譯：Mirror.xyz 入門教學》一文。）
