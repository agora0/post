教練，其實只練不教。不會教導案主，只會啟導、激勵、陪伴、發問。發問是高深學問， 需要精準到位，否則生吞活剝、力度過猛，或者兜兜轉轉、錯失良機。譬如，May 說「昨天 老闆又發脾氣，罵我做事拖延，我很傷心和內疚。」教練有 18 種有效發問的方式，可以參考： 

1、當下：如果老闆發脾氣，只是導火線，老闆真正觸動妳的甚麼情緒？ ( 暫時放下老闆這根導火線，將焦點放在自己的內在情緒和需要。) 

2、靜心：我可否邀請妳慢慢深呼吸，感受此時此刻，內心真正的需要？ ( 讓案主放下頭腦，透過呼吸，回歸當下，覺察內在情緒、期望和信念。) 

3、區分：發脾氣是老闆的責任，那麼妳自己需要承擔的責任，又是甚麼？ ( 區分責任：將老闆的責任還給老闆，同時自己誠實承擔自己的責任。) 

4、後果：如果妳繼續拖延，老闆又再發脾氣，最惡劣的結果將是怎樣？ ( 讓案主客觀評核自身處境，理解自己將會承受怎樣的結果和風險。) 

5、臣服：如果這是上天為妳度身訂做的禮物，上天想妳學懂甚麼功課？ ( 讓案主學習逆境感恩、反敗為勝，將挫敗視為上天贈予的珍貴禮物。) 

6、面對：如果妳為自己的拖延行為，完全負責任，妳會怎向老闆表達？ ( 教練鼓勵案主為自己的生命負責任，落實具體行動，創造有效溝通。) 

7、投射：妳如此抗拒老闆對妳發脾氣，妳自己也會對別人發脾氣嗎？ ( 讓案主看到自己是否雙重標準，批判對方，從而逃避自己的責任。) 

8、內觀：妳對老闆的批判，是否因為自己也有對自己潛藏的批判？ ( 讓案主從批判老闆轉為內觀自己的問題，並為自己的問題完全負責任。) 

9、模式：真正令妳傷心和內疚的是甚麼？這些情緒背後的根源是甚麼？ ( 讓案主探索自身的情緒模式，放下對方，覺察自身情緒的根源。) 

10、期望：妳對老闆，有甚麼未放下的期望？如果放輕期望，將會怎樣？ ( 情緒的源頭，不是外在事物而是自身期望。放輕期望，境隨心轉。) 

11、願景：如果妳選擇盡情創造，妳渴望與老闆創造怎樣的雙贏關係？ ( 讓案主的焦點，從抱怨對方改為真正負責任，創造雙贏的關係。) 

12、尋寶：請問過去有甚麼成功經驗，讓妳有效轉化了傷心和內疚？ ( 重拾內在資源，體驗自己有充足內在資源，化解人生所有問題。) 

13、信念：請問是怎麼樣的信念，讓妳持續拖延，然後又會內疚？ ( 讓案主認真看清實相，認知信念如何局限自身的行為和情緒模式。) 

14、自助：下次老闆發脾氣，我們可以講句甚麼話，讓自己即時平靜？ ( 讓案主學習自我啟導，找出面對逆境時，自我即時轉化的關鍵法門。) 

15、行動：如果我們不再拖延，真正勇敢面對，妳的第一步行動是甚麼？ ( 讓案主坐言起行，不再停留於情緒和感受，主動選擇、創造具體行動。) 

16、雙贏：如果要創造雙贏，妳會用甚麼心態，與老闆有效溝通？ ( 讓案主看到改變必需從自己開始，提出雙贏建議給案主參考。) 

17、同理：請妳感受老闆的壓力，體驗老闆對妳最不滿的是甚麼？ ( 讓案主易地而處，放下我執心，學習理解、感受對方的處境和需要。) 

18、量化：如果十分滿分，妳覺得自己有多少分，還差甚麼才能滿分？ ( 讓案主客觀自我評核，看自己有甚麼優點，以及甚麼需要進步之處。)
