【文：筆漏道九】

先道歉 ERROR 的粉絲，在 2021 年 5 月前，我是完全零認識 ERROR。

一輯《ERROR 自肥企画》後，我用一星期時間，全數追回過去幾年的《ERROR 自救 TV》《花姐 ERROR 遊 1》《花姐 ERROR 遊 2》（及四子九萬幾個訪問、MV 及片段），四子在節目裡要幾瘋狂有幾瘋狂，要幾癲喪有幾癲喪，他們和節目組的挑戰已經不是一般綜藝遊戲，是人類極限，隨口都講出十個八個癲狂嘗試，夜遊鬼屋與自殺勝地、赤腳踩炭、跳橋、笨豬跳等等等等等還未夠；還要食隔夜死魚頭，昆蟲邊爐、食辣食到「入廠」、蜜峰上身，甚至連伴隨尿味的屎（！）都試過，真心五體投地地拜服。你期待節目有幾狂，笑可以笑到盡眼淚流，但也喊到盡感人，綜藝節目令觀眾看到嗚咽和抽泣實在是世界奇聞。

為了理想　可以去到幾盡

ERROR 四子，第一眼肯定是「旺尖旺隨手執到九百幾十打的 MK 仔」吧。成軍三年，他們沒有 MIRROR 般的巨大資源支持和支援，他們行的是「騎呢搞笑男團」路線，當MIRROR 有資金一個 MV（《IGNTED》）都可以拍出兩個版本時，另一邊廂 ERROR 拍一個 MV（《我們很帥》）反而要靠賣T裇眾籌。有 ERROR 成員在節目講過成日「俾人鬧」，或許三年工作經歷，人生凌角未見「高光」已磨蝕了一半，鏡頭後總是經歷低潮、疑問、氣餒，但鏡頭前他們仍然瘋狂賣力，做出口碑叫好的節目。其實四子的目標並不複雜，肥仔想發展唱跳、POKI 玩音樂、阿 Dee 想演戲，193 想做潮流唱歌兼備的藝人。他們異常的投入，就是做好工作，那怕總有一天有人賞識，出人頭地，成就自己的理想。

返回現實世界，回看身邊不少廢青廢中朋友，人生從來沒有真正努力過，就長期滿口埋怨。連準時開工、投入工作、俾心機準備文件都不達標，加班滿口投訴，遲收工 15 分鐘已有如世界末日，總認為手持大學學位就可以得天下，帶著沒有靈魂的軀殼坐在辦公室，機會不用爭取就可以不勞而獲，成功起跑線在線等諸如此類。又眼見不少中年朋友長期意難平，自覺懷才不遇，永遠自怨自艾，不如反問：你是認真做實事，定「日日做柒事」，人生沒有永遠的高潮，在低潮時願不願意放下身段，讓自己從頭再來？有朋友因為放不下身段，三年呆在家中等待，自以為有一天可重回生的高峰，我大抵懷疑這種坐以待斃的態度，真的可以不勞而獲，或最終一無所獲。

觀看 ERROR 的幾個節目，我都一直在回顧自己，在年青的日子，廿零歲的人生起步，有沒有努力過，有沒有為人生奮鬥過？我肯定答案是：「有的」，但實在對不起，我自問沒有用盡最大的努力，即是 ERROR 四子付出努力的那個「高度」，去成就我的理想，所以我的理想最終沒有達成，是遺憾吧。看 ERROR 為了理想，原來是真的可以去到好盡．好盡。

第二個對不起，是看到保錡兩次跳高失敗哭崩、193 奈良餵鹿「嚇到喊」、阿 DEE 食辣、肥仔玩空手道分別「入廠」，我沒有嚎嚎大笑卻反而兩眼流淚，尤其是保錡哭崩（花姐 ERROR 遊 1/2 跳高）那兩幕 –他無法戰勝心魔，自責不能完成拍攝，鏡頭前不停道歉，又再學懂人生一課。回想現今世代的「道歉」，其實早已變成是大牌的奢侈品，身邊有同事有朋友做錯事，不但不會道歉，更不會反省自己有錯。那怕說一句「SORRY」不用半秒時間，但踏出講SORRY 的勇氣和態度才是千斤重。曾幾何時是理所當然的事，卻原來慢慢已點滴流逝。「有錯就認，打就企定」，看綜藝節目我找回做人的「初心」。

「無制限 OT 編集団」負責的這三個節目（《ERROR 自肥企画》《花姐 ERROR 遊 1》《花姐 ERROR 遊 2》），瘋狂背後，是成長，是挑戰自己的勇氣，是突破自己的心魔。不用說三年，那怕是十三年、和三十年，不少人的腦袋和人生，從來都沒有真正成長過，幾多人還躲在象牙塔空談空想，甚至是井底之蛙式的批評。ERROR 四子鏡頭前嘗試不同「能人所不能」的挑戰，他們也會瘋狂破口大罵，與其說是「好喇，我鏡頭前做咗佢算」，隊友們和幕後團隊在旁的鼓勵和支持，是「做好份工」裡互相支持和一起突破的進步，你可以認為四子仍然 MK，但至少他們有努力去成長，去面對工作困難的勇氣。

努力不一定成功，但九成九成功的人，都一定徹徹底底努力過。ERROR 終於踏出吐氣揚眉的第一步，但實在不想過份褒獎四子和無制限 OT 編集団，因為人生永遠有進步的空間，太多的讚美會導致迷失；繼續期待他們的製作，看他們付出的努力，去達成真正的出人頭地。

喂！最後一句：認識 ERROR 四子未？有梁業（肥仔）、何啟華（阿Dee）、郭嘉駿（193）同吳保錡（保錡）；最後最後：Mike 導，可摸耳唔好再迫保錡跳橋，我唔想再一齊哭崩呀！
