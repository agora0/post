2019 年 7 月 7 日九龍區反修例遊行後，有女子疑被誤當便衣警被包圍，2 男 1 女被控一項非法禁錮及非法集結罪，包括一名女教師及一名男社工，其中一名男子另被控非禮。區域法院法官練錦鴻今天裁決各人所有罪名全部成立，案件押後至 5 月 4 日判刑，以索取背景報告及事主創傷評估報告，期間三人須還押；屆時辯方將作出求情。

練錦鴻在裁決時指，退一步說受害人即使真是女警，亦並非眾人強行凌辱她的理由，眾人對 X 的行為純屬宣洩對執法人員的不滿。法官更指男社工自辯時文過飾非、強詞奪理，女教師則以溫文的包裝要求 X 留下，實為助紂為虐，認為兩人所言與錄影片段不符，最終裁定三人所有罪名成立。

練錦鴻：首被告乘機佔便宜或故意侮辱　裁定非禮罪成

法官裁決時指，30 歲的女事主 X 身形嬌小，較一般女子瘦削，她於案發當日獨自前往旺角。晚上 11 時左右， X 在山東街及西洋菜南街交界以手機拍攝現場情況，其後一名黑衣女子大聲指 X 在偷拍並嘗試搶走她的手機，其他人隨即箝制她，阻止她離開。裁判官續稱， X 出庭作供時情緒激動，被問到許多細節時均表示不清楚，雖其供詞細節欠佳，但接納她曾大聲求救、提出反駁及掙扎，反映她不願被拘留於現場。

法官又指，除 X 的證供外，各網媒的現場錄影片段也是主要證供，片段比 X 的口頭憶述更詳細，但強調無論科技怎樣發達，亦無法將事件全方位、毫無遺漏地記錄下來。法官在內庭多次反覆觀看錄影片段，稱畫面顯示首被告黃子隆身形高大，身穿白色 T 恤，背著黑色背囊。黃首先拉 X 的右手，兩人雙雙倒下，當時亦有其他人箝制著 X ，圍觀群眾不少於 20 人。黃再以左手伸進 X 的左側腋下，並左手拇指在 X 的乳房外側，X 掙扎大叫期間黃雙手扯著 X的手肘，更以下體抵著 X的臀部。

警方事後於黃的居所搜出他當日所穿的衣物，經化驗後亦證實黃的 T 恤及背囊上均有 X 的 DNA。法官強調，在錄影片段中可見黃觸及 X 的乳房，X 當時大叫「非禮」，並在作供時表示感到被侵犯；當在場有其他人制服 X 時，體形較大的黃無須作出上述動作。因此，法官認為黃是乘機佔 X 便宜，或故意藉此侮辱 X，裁定黃一項猥褻侵犯罪罪名成立。

表面上欲令事件降溫　練錦鴻斥女教師助紂為虐

法官續言，片段顯示當 X 被多人拉扯時，女被告蘇瑋善上前跟 X 稱「乖啦，乖啦」，「冷靜、冷靜」，並為 X 整理上衣，擋著嘗試拉扯 X 衣領的手。蘇的語氣溫和，但一直要求 X 交出手機，向她追問「你係咪真係影我哋？」、「你攞部電話出嚟 show 畀我哋睇」、「我諗大家擔心嘅都係你影咗啲我哋唔想出街嘅相」等說話。她亦有捉著 X 的手，不讓 X 離開。

蘇自辯供稱當時是希望上前調解，在目睹 X 打首被告後要求她留下解釋回應，多番要求 X 出示手則是誤以為 X 偷了他人的手機。惟法官認為，蘇的說法與畫面不符，從她使用「我哋」二字可見她與在場人士有相同的心態，證實她與眾人一伙，不可能不知道眾人圍著 X 的目的以及誤當 X 為女警。

法官又稱，蘇表面上希望將事件降溫，但其所指的調解則是要求 X 遵從眾人對她不合法、不合理的要求，用手拉著 X 與他人合力箝制她，而非勸喻眾人將 X 放開，再要求 X 作解釋，明顯是助紂為虐，並不是蘇口中的調解。法官坦言，蘇的正義感不足以使她制止他人對 X 作出拉扯、辱罵等的集體欺凌，不接納蘇真心打算作出調解。

官指男社工強詞奪理　文過飾非

在片段中最遲出現的被告吳睿哲曾要求 X 出示委任證，並大叫「圍住佢」，示意眾人築成人鏈阻止 X 離開。吳則在庭上辯稱，誤以為 X 是干犯了襲擊市民罪的女警，故舉起雙手分隔 X 與他人，希望保護在場市民，又向 X 表示「你啲伙記好撚鍾意除人口罩架嘛？」試圖引起 X 的注意。吳供稱，自己亦有搭著 X 的手臂防止她離開，但未從目睹 X 打人。法官則指，其說法根本與畫面記錄根本不同，即使接納吳真誠相信 X 為女警， X 的行為都不屬違法，認為吳強詞奪理。

法官指出，吳沒有親眼目睹 X 向他人施襲，無法行使其公民權將 X 拘捕，無權要求 X 澄清，直斥吳在庭上的解釋為文過飾非，希望藉此脫離罪責。法官續稱，吳在眾人箝制 X 之時，出言侮辱她，可見他參與挑釁行為，有意激發紛爭及破壞社會安寧，故不接納其證供。

法官另指，事主 X 在公眾地方拍照，雖對被拍攝者帶來影響，但亦非犯法行為或眾人強行留下、辱罵 X 的理由。他認為，退一步說 X 若真是女警，在拍下示威者的容貌，亦並非眾人對她凌辱的理由，故眾人對落單弱女的行為純屬宣洩對執法人員的不滿。就 X 打首被告臉額及扯下其口罩，法官指該行為是正常反抗，惟眾人指控 X 「打人」、「偷野」即是用作掩飾的藉口。

女子曾打被告臉額及扯口罩　官：正常反抗

法官裁定三人在場有共同目的，與他人一同將 X 非法禁錮，令 X 受到傷害，包括四肢多處瘀傷，右眼眉有輕微腫脹觸痛；三人行為亦會令在場第三者有合理擔心社會安寧受到破壞，非法禁錮及非法集結罪名成立。

三名被告分別為黃子隆（廚師，32 歲）、吳睿哲（社工，24 歲）及蘇瑋善（中學教師，26歲）。三人各被控非法禁錮罪及非法集結罪，他們被指於 2019 年 7 月 7 日，在旺角山東街與西洋菜南街交界非法囚禁女事主 X，違反其意願而將她禁錮；及於同日同地連同其他不知名人士非法集結在一起，作出擾亂秩序的行為或作出帶有威嚇性、侮辱性或挑撥性的行為，導致其他人合理地害怕集結人士會破壞社會安寧；黃另被控一項猥褻侵犯罪，指他於同日同地猥褻侵犯女事主 X。

案件編號：DCCC 893/2019
