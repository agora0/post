合併影片，

1.新增一個 star418.txt 的檔案，內容爲：

file ‘star00418hhb1.wmv’

file ‘star00418hhb2.wmv’

file ‘star00418hhb3.wmv’

file ‘star00418hhb4.wmv’

2. 開啓命令列，使用 ffmpeg：

ffmpeg -f concat -i star418.txt -codec copy star418.wmv

壓縮影片，

使用 ffmpeg 預設的最簡單的壓縮優化（-i 參數）：

ffmpeg -i star418.wmv star-418.wmv
