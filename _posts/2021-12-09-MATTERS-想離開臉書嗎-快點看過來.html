本文同步自：Web3.0的BBS論壇開張！

前幾天在幣安的網站上看到『Binance Labs領投BBS Network 150萬美元種子輪融資』這則新聞。

內文提到，幣安旗下的風險投資與孵化機構Binance Labs投資了150萬美金到Bulletin Board System (BBS) Network這間去中心化的BBS。

一直都對這類去中心化的社群平台特別有興趣，於是就跑去Bulletin Board System (BBS) Network的官網研究了一下。

在Bulletin Board System (BBS) Network的官網上講到他們的特色有：

  1. 抗審查
  2. 去中心化
  3. 利用區塊鏈儲存資料
  4. 每一篇文章都是一個NFT
  5. 使用者可自由開設自己的討論區
  6. 在BBS論壇上活動可得到加密貨幣
  7. 使用者可自由購買他人文章內的廣告空間
  8. 使用者可自由販售自己文章內的廣告空間

基本上光是看到抗審查、去中心化我就支持了！

相信不少人已經受夠臉書帝國的獨裁政策，但又苦無一個更好的社群平台可以去。

勉強去到推特發展，卻也感受到推特其實沒那麼言論自由。

各位朋友，別再猶豫了，趕快把你的重心轉移到Bulletin Board System (BBS) Network來吧！

來到Bulletin Board System (BBS) Network這個建構在區塊鏈上抗審查、去中心化且百分之百言論自由的BBS論壇發展！

也歡迎大家來加入我在Bulletin Board System (BBS) Network上創的討論版：

  1. Tarot（聊聊塔羅牌）
  2. 政治時事輕鬆聊
  3. 大吃四方美食分享（民以食為天，吃飯皇帝大。）
