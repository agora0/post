完整標題：invest 與「資投」或「資以金」或「資以時」或「意於獲益之投入」或「意於獲益之行動」或「益賦予」或「益授予」或「益包覆至」或「益包封至」或「益包服裝」或「進行圍封鎖定」等的轉換密碼



1. invest =冫⺈人eye三t = 次目ttt = 次目two點投 = 次目二點投 = 次目㇀丶投 = 次貝投 = 資投，此時即漢字「投資」的同義倒裝詞。其中v 轉 180 度通過 Λ (lambda) 的小寫 λ 之左右鏡射約略象徵漢字「人」，e 代表 eye 而轉換漢文「目」。

2. invest = inve + e + vesst 省ㄧ個 v 省二個 e 省一個 s = 「橋接式1」中的 inve 但 e 直接代表 shell 而轉換漢文「貝」+ 以 + 𠆢E雙雙t = 資 + 以 + 𠆢EE雙點 = 資 + 以 + 𠆢王丷 = 資 + 以 + 金 = 資以金，即「資以金錢」或「用金錢投資」之意。

3. invest = inve + e + st 省ㄧ個 e =「橋接式2」中的 inve + 以 + 時time = 資 + 以 + 時(時間) = 資 + 以 + 時 = 資以時，即「資以時間」或「用時間投資」之意。

4. invest = 意於獲益之投入。其中 s 約略象徵漢字「之」，t 約略形聲漢語「投」的起音且同時通過大寫 T 轉約 45 度再左右鏡射而約略象徵漢字「入」。

5. invest = 意於獲益之行動，意思與「橋接式2」所轉換的「意於獲益之投入」類似。其中 s 約略形聲且象徵漢字「之」並同時約略形聲漢語「行」。

6. invest = i.n.ve.s.t = 益.聯.ve.s.t = 益.聯.賦.ㄙ.T = 益.聯.賦.龴.一丨= 益.聯.賦.龴.乛亅= 益.聯.賦.龴乛亅= 益.聯.賦.予 = 益聯賦予 = 益賦予，類似「增賦予」或「加賦予」之意。

7. invest = i.n.ve.s.st 省ㄧ個 s = 益.聯.v一.受.予 = 益.聯.䒑.受.予 = 益.聯.k.受.予 = 益.聯.扌.受.予 = 益.聯.扌受.予 = 益.聯.授予 = 益聯授予 = 益授予，類似「增授予」或「加授予」之意。

9. invest = i.nve.v.est 省ㄧ個 ve = 意.勹與巳.封.一厶土 = 益.包.封.至 = 益包封至

11. invest = invvest 省ㄧ個 v = 進行圍封鎖定，類似「包圍鎖定的目標」之意，此時或爲軍事用語。其中 in 同時轉換漢字「進」和「行」而可能合爲「進行」之意，即「in = .l 鳥 = dotL隹 =丶L隹 = 辶隹 = 進」和「in = 點人 = ㇀亻= 彳= 行」。


P.S.


investment：資投名詞；資以金名詞；資以時名詞；資之投的物件；意於獲益之投入名詞；意於獲益之的行動名詞；益賦予名堂；益授予名位；益包覆至名詞；益包封至的名詞；益包服裝的名詞；進行圍封鎖定的名詞。其中第一個 t 或同時約略形聲漢「的」。

investor：資投者；資以金者；資以時者；意於獲益之投入員人；意於獲益之行動者；益賦予者；益授予者；益包覆至者；益包封至者；益包服裝者。

investitive：資投的；資以金的；資以時的；意於獲益之投入的語意；意於獲益之行動挺有感覺；益賦予的；益授予的；益封席至地位，類似「封爵位」；益封席至地位之，類似「封爵位的」。此字多作形容詞使用。其中 itive 或通過「點亻點與E」轉換「丶勹丶與日」轉換「勺白」的組合而轉換漢字「的」，若轉換「的語意」或「挺有感覺」時則 itive 的第一個 i 爲輔助發音。

investiture：資投的為作；資以金的為作；資以時的為作；意於獲益之投入的為作；意於獲益之行動的為作；益賦予的為作；益授予位職；益封席至地位，類似「封爵位」；益封席至的式儀，意通「封爵位儀式」；應於務事的式儀，類似「任職儀式」。

investigate：於內閱視調與查為作、於內查調為作，類似「內部調查」；於內追究為作、入內探究為作、進行查調為作，類似「進入內部以追究調查」；察調為作；調查為作。此字與前述的 invest ㄧ字完全無關。其名詞爲 investigation (於內閱視調與查為作之名；於內查調為作之名；察調為作之名；於內追究為作之名；入內探究為作之名；進行查調為作之名；察調為作之名；調查為作之名)。


補記：invest 與其它漢字的橋接轉換


invest = ㇀丨⺆複一殳兜 = ㇀\ ⺆二殳兜 = 入月殳兜 = 入股，其中「複」和「兜」僅作構字用語，s 約略形聲漢語「殳」的 shu 式發音之起音。其中的「股」字是代表「股份」或「股票」的「股」。

invest = ㇀丨几與E手兜 = ㇀\ 殳彐兜 = ㇀\ 殳扌兜 = 入投，此時爲「投入」的同義倒裝詞，但此「投入」應是指時間、金錢、財力、心力這方面的投入。其中「與」和「兜」僅作構字用語，e 通過大寫 E 約略象徵漢文「彐」再轉注漢文「扌」。
