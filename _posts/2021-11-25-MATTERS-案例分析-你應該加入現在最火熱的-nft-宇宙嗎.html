NFT 是 2021 年最紅的流行語之一，它和時下流行的酷炫元素都有點關係，如區塊鏈、以太幣、DeFi 去中心化金融、元宇宙等等。

我第一次聽到 NFT 這個詞是從《科技島讀》今年 3 月的一場 Podcast 節目，在這集中來賓寶博士提到 NBA 發行的 Top Shot Moment (進球時刻) NFT 和其他相關商品。當主持人周欽華試著舉例說明 NFT 到底是什麼，我馬上就聯想到，這不就像哈利波特裡面那些會動的畫嗎？當時覺得這個概念好炫：因為這個新問世的數位資產可以以任何形式存在，不論是照片、音樂、影片都可以變成 NFT。

幾個月過去了，當人們仍因疫情的爆發不得不關在家裡，NFT 宇宙正以驚人的速度成長。為了搭上這股浪潮，我先在 OpenSea 平台做了第一個自己的 NFT，而在這篇文章中我想做個簡單的案例分析，探討還沒這麼做的人究竟應不應該踏入 NFT 的世界。

NFT 市場有什麼最新消息？

亞洲音樂市場當紅的新聞，應該就是馬來西亞歌手黃明志一夜間在 OpenSea 平台以自己歌曲的 NFT 賺進新台幣 2400 萬元的報導。

突然間，我開始在不同的群組聽到對這個發明的諸多討論；舉凡投資人、作家、設計師、律師，甚至是行銷人員，每個人都對 NFT 議題似乎都可以給出一兩條評價。

前面提到，NFT 的形式是沒有限制的；自然地，從供給和需求面來看它都自帶強勁的動能。

有鑑於此，你可能已經可以想像到NFT 世界的競爭異常激烈。你的競爭者來自世界各地不同年齡的創作者，甚至有年僅 12 歲的男孩。在這樣的環境下，可想而知，要打造一個熱賣的 NFT 產品絕非易事。因此，你必須找到讓自己作品獨樹一格的特色，還需要絕妙的行銷策略；這點待我們稍後再談。

NFT 有什麼特別？

NFT 是 non-fungible token (非同質化代幣) 的縮寫。從本質來看，因為底層技術架構在區塊鏈網絡上，每個 NFT 都是獨特的。因此，每次 NFT 轉手時系統就會在紀錄中加上一段代碼，不論下一個所有者是誰，他或她都會自動繼承歷年所有的紀錄。

如果你是個古董收藏家，你可能會想購入某個歷史人物擁有過的收藏；一方面可以彰顯這份收藏的價值，另一方面可以展現你的絕佳品味。而這一套邏輯當然也適用在 NFT 市場。在 NFT 平台上，使用者可以查看某個 NFT 是什麼時候鑄造、誰曾經出價、又出了多少錢；而這絕無僅有的紀錄就是 NFT 特別又風靡全球的原因之一。

賣家當然也可以來點加碼。譬如在 OpenSea 平台上有一個 "Unlockable Content" (不可鎖住的內容) 項目，創作者可以想一些點子讓線上的活動轉化為線下的互動。還有一個例子是關於即將上映的科幻電影《駭客任務：復活》，華納兄弟公司已經宣佈在電影 12/22 首映前會發行 10 萬個兩階段可變身的人物肖像 NFT。

NFT 還有另一個賣點就是現在最夯的元宇宙運用。然而，我的觀點有點不同，有鑒於目前大部分 NFT 都還存在於 2D 的世界，而 3D 的元宇宙又還是個半成品，若真要看到大量 NFT 在元宇宙裡使用恐怕還得花上好長一段時間。

那麼獲利能力又如何呢？我知道你可能很羨慕藝術家 Beeple 在今年稍早以新台幣 19 億元的天價賣出一組 "Everydays: the First 5000 Days" (每天：最初的5000天) NFT；但我們還是實際一點吧，別忘了你必須考慮鑄幣成本跟轉移 NFT 的礦工費。記得要慎選你的販售平台跟拿來鑄幣的區塊鏈，Polygon 鏈不用鑄幣費我想是個很不錯的選擇。

如何打出知名度？

回到我們稍早的行銷策略討論，既然每天都有無數的熱情創作者投入 NFT 市場，你要如何在人群中脫穎而出呢？

我不是專家，但或許可以分享一些其他人用過的技巧；但究竟這是好是壞，就得由你自行判斷了。

首先，你需要準備一個好故事；就像那位用自己的程式創造出像素鯨魚的 12 歲男孩 Benyamin Ahmed。也可以讓知名角色助你一臂之力，像最近的 "The One Token That Rules Them All" (一枚 NFT 統治天下) JRR Token 代幣。或著尋求 AI 人工智慧的幫忙，例如圖形處理器的晶片設計公司 Nvidia 剛宣布開發出可以將文字指令轉換成精美圖片的 GauGAN AI。

再來，如果你已經有一群粉絲支持，也可以試著走名人路線；Linkin Park 聯合公園成員 Mike Shinoda 就在今年加入了 NFT 宇宙 。我在這個月的 2021 Singapore FinTech Festival (新加坡金融科技節) 聽過一次他的分享，他說自從接觸這個新科技後，再也不用擔心如何將創作變現，聽起來很真心。你仍可以在 ZORA 平台上對他的作品進行出價。

更甚者，透過合夥關係增加媒體曝光也是可行的。然而，你必須知道如何做得正確，並用聰明的方法進行。說到這裡，你可能會想先看一下獨立部落客 Amy Castor 所寫的一篇關於 Beeple 創紀錄拍賣背後爭議的文章。

寫在最後

從上面的段落，我們已經從產業動態、產品特性、行銷策略的角度分別探討了 NFT 的潛力。現在，我想我們應該都同意這個市場大到不能忽視，更何況摩根大通剛宣布它的市值突破了驚人的 70 億美元。

但在進入 NFT 宇宙之前，有些事情你必須知道，就是它的相關風險。不管從買方或賣方角度，NFT 的著作財產權都不可以等閒置之。即使是大導演昆汀·塔倫蒂諾 (Quentin Tarantino) 都因想出售自己 1994 的經典作品 《黑色追緝令》NFT 吃上官司。

同時，不要把 NFT 當作是個單純的投機投資工具。如果你真的想投資，找一個你有熱情的標的吧！如此一來，即便哪一天這把火慢慢熄滅了，你還可以帶著驕傲跟感情保有這份收藏。想想你小時候那些得來不易的收藏，你當年是為了賺一筆而買下它們的？還是出自對他們的真心喜愛？   

你或許要問，下一步是什麼？你不一定要成為某種形式的創作者，但我建議至少自己要做點功課去瞭解 NFT 背後的科技運用。

畢竟讓 NFT 宇宙變為可能的基礎建設 Web 3.0 是個革命性的新發明，很可能在下一個 10 年大大改變我們的生活。欲了解更多可以參見我的另一篇文章《 如何用 Web 3.0 設計金融服務? 》。

當然我們要達到電影《 一級玩家》 ("Ready Player One") 裡勾勒的虛擬世界還要很多年，但在這個時間點，不論你身處哪個產業，你都應該努力累積更多 NFT 和其相關運用的知識。為了自己做準備，然後有一天利用你的虛擬身份在萬眾期盼的元宇宙大放異彩。
