研究樓價與股市不同，沒有Google Finance或Yahoo Finance之類的平台，不可能簡單快捷的比較不同國家股市的價格指數，即使找到每國的樓價指數，但由於歸一時間點不同，還是需要把數據做Indexing (姚，2021) 加以處理才可繪製國際樓價指數比較圖，本文介紹一種pivot table方法，學懂後只需幾分鐘便可畫出多國樓價和租金(歸一時間點的)指數比較圖。另外，我製作了短片(https://youtu.be/5vJI6hA8Z_E)，可一步一步跟著用pivot table製圖，較易上手。

國際樓價與租金指數資料

首先，最重要當然是數據資料，否則若須自行估算樓價與租金指數，可能須要幾天時間。幸好國際經合組織 (OECD) 免費提供成員國的數據，其中包括樓價與租金指數，而且可選擇國家，年度或季度或月度等設定，更可下載數據資料，可惜下載的數據是以單行序列，對畫圖很不方便，因此本文簡介用 Excel Pivot Table 功能製圖。

OECD data 的缺點是沒有包括非成員國的資料，另外數據多以國家為單位，沒有城市為本位的數據，版面亦只有英文或法文版。圖1是OECD data 的版面 (https://data.oecd.org/price/housing-prices.htm#indicator-chart)，圖左邊是選擇數據的選擇檔 (Indicators)，尾二的便是樓價資料 (Housing prices)；圖的底部是選擇圖形表達檔，包括 (A) Perspective - 可選擇樓價 (Nominal house prices)，實質樓價 (Real house prices)或租金(Rent prices)等等；(B) Countries - 可選擇不同國家數據，本文以美、加、英、德、法、日、韓、澳、紐九國的樓價與租金指數作為示範，(C) Time - 可選擇年、季、月度數據，並可選擇數據的時間範圍，本文以2015年第一季至2021年第三季為例，因為數據的歸一時間點是2015年=100。圖中的十幅圖表顯示所選國家的數據，右上角藍色鍵 download 可下載數據。

圖1 OECD Data Economy 版面- https://data.oecd.org/price/housing-prices.htm。

下載數據形態

下載的數據是以單行垂直疊起 (stacked) 形態，如圖2 所示，九國的數據序列是垂直排列，不便於製圖。因此需使用Excel 的 Pivot Table 功能把數據轉為矩陣 (matrix) 形式，更可簡易製作指數比較圖，一舉兩得。若不用 Pivot Table 亦可手動製作，但所需時間較長，亦容易出錯。

圖2 下載數據的形態。

如何用Excel Pivot Table 功能

利用Excel的Insert - Pivot Table 鍵，如圖3 右邊所示，選擇 LOCATION (國家名稱) 作 COLUMN (垂直欄)，選擇 TIME (時間 - 季度) 作 ROW (橫行)，選擇 VALUE (數值 - 指數) 作 VALUE便可製成圖3的PIVOT TABLE - 矩陣形指數，可直接畫圖。

圖3 PIVOT TABLE 製作的矩陣國際指數。

有些較詳細的功能，如定小數點等，不在這裡介紹，詳情可看短片。

在 PIVOT TABLE 畫圖

在 PIVOT TABLE 畫圖有不少好處，圖示等資料會自行選定，更可選擇畫出不同國家和不同時間區域的圖表，如圖4所示是選擇了這九個國家從2015至2021年的指數圖，完全自動化。其中紐西蘭、加拿大、德國和美國的樓價在疫後急速颷升，與2015年比較已升超過五成以上，而且九國的樓價全部上升，只是幅度不同而已。

圖4 九國的樓價指數，2015-2021年。AUS澳洲，CAN加拿大，DEU德國，FRA法國，GBR英國，JPN日本，KOR韓國，NZL紐西蘭，USA美國

相反，租金的升幅較少，以紐西蘭和美國為例，紐西蘭的樓價升幅達65％，而租金升幅只有17％左右，美國的樓價升幅達60％，而租金指數升幅只有23％，日本的租金更出現輕微下跌。[當然，數據是否可靠準確則不在本文討論範圍之內。]

圖5 九國的租金指數。

兩圖中的灰色鍵是選擇器，TIME 可選擇時間區間，LOCATION 可選擇國家。更有趣的是PIVOT TABLE 可以協助繪製月度變化率圖表，但方法比較複雜，有興趣的讀者可自行根據網站指引https://exceljet.net/pivot-table/pivot-table-year-over-year嘗試，成功後可與讀者分享。YOUTUBE 短片(https://youtu.be/5vJI6hA8Z_E) 亦有示範。
