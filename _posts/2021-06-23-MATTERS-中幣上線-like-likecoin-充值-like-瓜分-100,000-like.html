中幣（ZB）於香港時間 2021年6月23日上線 LIKE（LikeCoin）。具體安排如下：

  1. 2021年6月23日 14:00 開放 LIKE 充值
  2. 2021年6月25日 16:00 開放 LIKE/USDT 交易
  3. 2021年6月26日 14:00 開放 LIKE 提幣

同時，中幣（ZB）將於香港時間 2021年6月23日16:00開啟「充值 LIKE 瓜分 100,000 LIKE」活動。

活動詳情

活動時間： 2021年6月23日 14:00 至 2021年6月30日 14:00（GMT+8）

活動規則：

活動期間，從外部地址向平台成功充值 LIKE 的用戶，將按照其 LIKE 淨充值（充值 - 提現）數量，瓜分總計 100,000 枚 LIKE 代幣獎勵。

獎勵計算公式：

  • 淨充值佔比 ＝ 用戶淨充值 LIKE 數量/用戶 LIKE 淨充值數量之和 x 100%
  • 獲得 LIKE 數量 = 用戶 LIKE 淨充值佔比 x100,000
  • LIKE 累計淨充值數量低於 1000 枚的用戶不參與獎勵。
  • 獎勵將在活動結束後 7 個工作日內發放至用戶賬戶。

有關中幣（ZB）

中幣（ZB） 集團是全球領先的區塊鏈和數字資產金融服務商，旗下業務涵蓋數字資產交易平台集群、電子錢包、投資基金、研究機構等區塊鏈產業上下游，中幣（ZB）集團在新加坡、美國、韓國、中國香港、瑞士、馬耳他、馬來西亞均有業務佈局。

中幣（ZB）集團旗下包括全球一線老牌交易平台 ZB.com、全球創新加密貨幣交易平台 ZBG.com，全球首個基於礦池的交易平台 BW.com，並先後成立開展 ZB 實驗室（ZB Labs）、ZB 資本（ZB Capital）、ZB 研究院（ZBRC）、ZB 創新智庫（ZB Nexus）等綜合業務佈局，從資本、技術、人才等多方面扶持產業鏈上下游的獨立區塊鏈項目，為區塊鏈發展提供領導力和更穩定的環境。

ZB 創新智庫（ZB Nexus）致力於打造一體化區塊鏈研究體系和頂級產業智庫，實現區塊鏈行業全景透視。依託行業研報、數字資產評級、指數研究、通證經濟實驗室四大產品板塊，ZB 創新智庫（ZB Nexus）將通過研究創造價值，推動區塊鏈與實體產業的結合、應用和落地。
