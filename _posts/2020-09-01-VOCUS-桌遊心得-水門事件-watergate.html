「在憲法第一修正案中，開國元勳保障了新聞自由，使其擔任在民主政治裡的重要角色。新聞出版是為被統治者服務，而非統治者。」
－美國最高法院大法官休戈·布萊克，在紐約時報訴合眾國案。
水門事件是一款節奏緊湊的兩人對弈遊戲，主題建立在 70 年代美國總統理查·尼克森任內發生的水門案醜聞，一位玩家擔任揭發尼克森政府與水門案關聯的記者方，另一位則是試圖掩蓋證據的尼克森政府。水門案是美國史上影響甚大的政治醜聞，除了造成尼克森的辭職（在他得知眾議院的彈劾表決已經沒救之後），也讓對手民主黨成為接下來兩屆國會的多數派。
遊戲圖版的左半部是證據板，看起來就是每部犯罪影集中，警察拿來釘嫌疑犯與證據照片的那種板子。正中央是尼克森的照片，而周圍則是提供情報的線人。在這款非對稱的兩人遊戲中，記者方的目的是拿到足夠多的「證據 token」，把兩個線人和尼克森連起來就得到勝利，表現當時媒體懷疑尼克森政府涉案，但還沒有真確的證據，需要線人的幫助，像是 John Dean 這個人一開始幫忙掩蓋水門案，更得到尼克森的讚賞，後來卻與檢察官合作供出了對尼克森不利的關鍵證詞。
而尼克森方當然就是要阻止記者連線囉，尼克森拿到的「證據 token」會翻到黑色面（國防布的意思）來斷記者的線，並且試圖獲得五個紅色的「勢力 token」來獲得勝利，如果回合數夠多的話也會判尼克森獲勝，因此尼克森是防守方，記者是攻擊方。
遊戲的進行是透過「卡片驅動」的機制，雙方都會有一套自己專屬的牌組。有玩過《冷戰熱鬥》的朋友應該會覺得很熟悉，打出卡片的時候，可以使用卡片的點數來移動 token，或是使用卡片上的文字來影響戰局。遊戲圖板的右邊有一條像拔河的計數條，每回合一開始我們會將三種 token 放在中間，玩家就是使用上述的機制去移動 token，回合結束時，雙方會拿走落在各自區域的 token 並做對應的動作：像是拿到證據 token 就把它釘上證據板、勢力 token 能夠讓尼克森離勝利更近一步、先攻 token 則會決定下一回合的起始玩家。
爭奪這三種 token 是遊戲的重點，尼克森就有許多幕僚卡能夠做出很有效的爭奪，象徵他的行政優勢。記者則要靠證據板來給尼克森壓力。
我很喜歡「線人」這個機制，在遊戲一開始，證據板上是沒有任何線人的，記者方如果打出線人卡，就可以把該線人的照片釘在證據板上，但是尼克森也有相反的卡片，能夠切斷記者與線人的聯繫（怕）。一方面線人的點數很高，拿來當點數的話可以做很多事，但如果尼克森早一步打出線人，就會被斷線而對後續狀況非常不利，這個也是記者玩家會面對到的取捨問題。
而其他每張卡片也是水門案裡的關鍵角色或：踢爆水門案的華盛頓郵報記者 Bob Woodward、因為此案鋃鐺入獄的司法部長 John Mitchell、號稱不小心洗掉錄音帶（國防部？）的 Rose Mary。
對這一部份的歷史稍微了解的話，能夠非常進入這款遊戲的情境，相當有帶入感。我玩尼克森的時候每回合都在擔心證據曝光，玩記者的時候就對尼克森強大的幕僚效果恨得牙癢癢的，常常已經快要連線成功了，卻在關鍵時刻被斷了線，用「連線」這個機制來表現記者查案，真是太巧妙了！
就算拿掉情境，他也是一款優秀的兩人遊戲，在策略中仍然保持一些隨機元素，讓遊戲變得更加刺激，是今年我新學的遊戲中最愛的一款兩人遊戲。
