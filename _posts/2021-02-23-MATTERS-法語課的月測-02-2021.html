每個月底我們都會有一個測驗，不為別的，只是讓老師能追蹤我們的學習程度，如有需要可以在下個月換到程度較深或較淺的班別。

這次是交一份寫作 production écrite，其實就是英文的essay。題目如下，三選一。:

  1. Les réseaux sociaux nous rendent-ils malheureux? 社交網站使我們不快樂/不幸福嗎？
  2. La télévision nourrit-elle la violence? 電視會助長暴力嗎？
  3. Devrait-on encourager l'apprentissage via les outils technologiques (applications sur smartphone, tablettes...) ou l'apprentissage doit-il se faire avec des méthodes plus classiques (livres)? 是否應該鼓勵使用科技產品 (手機、平板的應用程式...)的學習方式 抑或 學習就應該以傳統方式(書)進行？
隨便列出超過10個社交網站 : Facebook, Youtube, Twitter, Instagram, Likedin, WhatsApp, Reddit, TikTok, Skype, Matters...
中國版 : weibo、小紅書、wechat、QQ、bilibili、愛奇藝、豆瓣...

然後我選擇了1️⃣號朋友。大概這個話題已經很多人討論過，亦有不少人紛紛已經從社交網站退出或者不再出post，我遲來地也參一腳、聊一聊。

社交網站=使人們彼此連繫、互相交流、影響的平台

一切煩惱都來自人際關係《被討厭的勇氣》，如非獨自隱居深山荒野，生活於人群，本來就有煩惱。

互聯網和智能手機先後出現，更使人與人的連繫難以中斷。人們在社交網站上經營"生活"，同時忍不住去窺探別人所經營的"生活"，其實都在刻意經營，都不可能反映生活的全部面貌。

一旦開始比較，就有可能成為不幸的開始 : 某些希望自己的post有更多的like，有更多follower，時刻關注，不僅關注自己亦關注別人 ; 某些不獲關注的只能羡慕妒忌，甚至也無法從中抽離，更添一層負面的心理壓力 ; 某些從不出post的，也會耗費很多時間精力去品嚐別人的成品。

But 真的是社交網站使他們不幸嗎？

社交網站亦有各種資訊和知識的傳播，思想的交流。人們分享經驗、發佈自己的創作，獲得回應，不受時空、語言、資金的限制，可以跟youtube學結他、學語言，加入fb社群交換心得與困難，更有利於結交到同路人。心情糾結苦惱，點開ig，看見蔡康永的哲言 或 nasa拍的宇宙 或 某些攝影師賞心悦目的作品，也許就能夠轉換心態。

所以說吧，社交網站是工具是媒介，我們不在上面生活，怎麼經營都好，還是各取所需後，回歸現實。煩惱還是會有，太多的話去跟阿德勒聊聊天，學習下課題分離，把罪名歸到社交網站不是辦法。

最後，剛來馬特市滿一星期，再來新人打卡刷下存在惑😆
