domino = d偶盟一ㄇ偶 = dd盟Iㄇㄇ = 歺低盟Iㄩㄇ = 骨卑盟片 = 骨牌，其中「歺」轉注「骨」，「低」轉注「卑」。其複數形式爲 dominoes (骨牌複數；骨牌一雙；骨牌一些；骨牌多；骨牌眾)。
domino = 多囗move辶聯車 = 多牌動連，意通「多牌連動」，類似「多牌連鎖而動」之類。此時其中第一個 o 約略象徵漢文「囗」再約略象徵「牌」的意思；字尾 o 約略象徵車輪狀而代表漢文「車」；m 代表 move 而轉換漢字「動」，即 move = nnove = 力人o複earth = 力亻oo土 = 力亻偶口土 = 力亻二口土= 力重 = 動，此時 m 分解爲 nn，第一個 n 約略象徵漢字符號「力」，第二個 n 約略象徵漢字「人」的篆文字體再旁通漢字符號「亻」，「偶」轉注「二」表示二個橫畫。
domino = 多球move辶聯車 = 多球動連，意通「多球連動」，類似「多球連鎖而動」之類，其實在玩撞球時就也會用到這種技巧。此時其中第一個 o 約略象徵「球」的意思；字尾 o 約略象徵車輪狀而代表漢文「車」。
domino = 多object.move辶聯車 = 多物(件)動連 = 多物動連，意通「多物連動」，類似「連鎖而動」之類。前述的第一個 o 不論是「牌」還是「球」或是什麼「單元」，簡單講也就是一種「物件」；字尾 o 約略象徵車輪狀而代表漢文「車」。
domino = 倒one31人over = 倒一三丨𠆢over = 倒一全口復一二 = 倒一全口復一II = 倒一全覆，類似「倒了第一個，就全軍覆沒」之類的意思。此意又衍生出「骨牌效應」(domino effect) 一詞，不過漢文也有另一種音義混搭的講法「多米諾效應」或「多米諾骨牌效應」。
在政治體制上，「獨裁制」似乎最容易發生上述這種類似「一旦倒了龍頭就變得群龍無首而陸續倒下以致全倒」的 domino effect (倒一全覆效應；骨牌效應)。巧合的是，domino 也能橋接於漢字「獨裁制」，即 domino = d偶min偶 = dd.minmin = 獨decade.minmin = 獨十巾衣牛雙i刀 = 獨十巾衣牛ii刂 = 獨十巾衣牛弋點刂= 獨十巾衣牛弋㇀刂= 獨十巾衣牛戈刂= 獨裁制。
奇怪的是，搞政治的人士可能明明就知道「獨裁制」有這樣難以長治久安的險厄弱點，爲什麼歷史上還是常常會簇擁出獨裁式的政權呢？……除了圖謀一時的「獨佔權力」、「霸佔利益」甚至是「撈一票就走」之類的連動效應之動機，或是由於「一人得道，雞犬升天」的情勢所致，實在找不到其它的原因了。也因此，當「一人得道，雞犬升天」的情勢逆轉之時，也就很容易變成「一人失道，雞飛狗跳」。
就經濟發展這方面來看，其實也存在著「骨牌效應」(domino effect) 的潛在問題，例如「獨佔事業」或「資源壟斷」之類都對於整體經濟有舉足輕重的影響。然而，這類問題的思考，早在先秦時期的《荀子》思想中就已經談過了，也可能是昔日電力科技草創時期的 「Tesla 系」和「Edison 系」對於供電系統架構的爭論主題之一，不是什麼很新鮮的議題。
簡單說，《荀子》的看法也許就是：少數的大資源中心雖然有其管理上的效率，但是也不能漠視多數而分散的小資源中心存在之必要。…… 大雞蛋放在大籃子裡當然是沒錯，但是也要準備一些小籃子放其它的小雞蛋；如果把大大小小的雞蛋都放在同一個籃子裡，一旦籃子意外被壓毀，所有的雞蛋就可能同時一起泡蛋花湯了！
或者是說，一張大餅雖然壯觀，但是將它分成若干塊小餅，讓幾個人分別持有，也是有「未雨綢繆」(take precautions；save for a rainy day；save for the hard days；make strategy for the possibly difficult situation；think ahead；prepare for the worried things that might be coming true；rains haven’t fallen yet but pre-discussing for the flood) 的好處。
如果只有一張大餅，當大餅被風吹走了，大家就沒得吃了；但是幾個人分別持有一塊小餅，就算有一、二個小餅被風吹走，剩下來的小餅大家還是可以分著吃啊！

Domino’s Pizza：漢文多譯「達美樂披薩」、「達美樂比薩」等，是一家在公元 1960 年創辦於 America 之 Michigan 的老牌烤餅店的品牌名稱。其中 pizza (餅；餅食；餅加雜食) 在現代中文漢字裡多音譯「披薩」、「匹薩」或「比薩」等。
