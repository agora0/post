（獨媒報導）2019年11月13日網民號召全港「大三罷」，上水大會堂外有兩批人士疑因路障問題爭執，期間一名七旬清潔工遭磚頭擊中頭部，翌日終告不治。兩名16及17歲青年否認謀殺、蓄意傷人及暴動共3項罪名。案件今（16日）於高院續審，另一名遇襲受傷的六旬老翁以匿名身份出庭作供，他於當日早上發現馬路滿佈磚頭和雜物，於是到場清理，期間遭一名穿著黑衣、紮辮子的男子阻止，後來在警署的認人手續中認出該男子為第一被告。

男子 X 稱清理磚頭時遭拍攝

第一名控方證人、男子 X 出庭作供，法庭早前頒下匿名令，任何人包括傳媒不得公開 X 的身份和個人資料。男子 X 供稱，當日早上6時20分出門上班，步行至上水港鐵站與上水中心之間的行人天橋時，目睹新運路行車道上滿佈雜物，包括磚塊，路牌和大型垃圾，於是他行落路面搬走磚塊和雜物，「希望可以清，等條路可以通返」。

X 其後發覺有2名穿著黑色衣褲的男子，一人身材較健碩、頭上紮一條辮子，另一人稍為矮和瘦削，戴鴨舌帽和揹著一個背囊。其中一人正在用手機拍攝清理雜物的市民，X 大聲要求2人「唔好影」，對方隨即轉身離開現場。

X找不到巴士上班 其後跟蹤兩名黑衣男子

X 稱他於大約7時完成清理，然後前往上水廣場巴士總站，希望看有否巴士出市區，可是發現沒有任何巴士能讓他前往上班。此時 X 發現剛才2名男子正步行上天橋，「覺得佢哋有份將雜物拋落馬路」，於是上前跟蹤二人，途經上水港鐵站和彩園商場，直至到達太平邨天橋便終止。

X 其後到彩園商場食早餐及回家，約11時半再度外出，打算到上水匯的百佳買咖啡粉，途經行人天橋時卻發現馬路重新被人放滿磚頭。因此 X 再次到馬路清理磚頭，「不能什麼也不做」。

X再度清理磚頭但遭首被告阻止

X 指有一名蒙了面的男子上前，「叫我唔好執，話我唔應該幫政府，我唔應該幫警察」，X 沒有理會，繼續清理磚頭，但該蒙面男子隨即把磚頭重新擲回馬路上。X 表示清楚記得蒙面男子向他說：「阿叔，你小心你個背脊，小心你個腰骨！」X 回應：「我個腰骨冇事。」然後蒙面男子離開。

X 繼續清理磚頭，其他市民開始加入。控方續播放數段網上公開影片，顯示 X 正在清理馬路上的磚頭，有另一批人士批評清理磚頭的人：「依家政府好就唔會有人破壞啦！警察家陣殺人又強姦女仔，睇清楚啲事實啦！」、「大把路俾車行！」X 則反駁指：「你哋唔破壞就唔會有建設。」

X 供稱，其間第一被告劉子龍嘗試阻止他，其後愈來愈多蒙面的人走過來阻止，有人大叫「開遮！」X 稱：「大家都知道，開遮即是佢哋想對我進行一啲更暴力嘅嘢。」

影片顯示兩批人就清理路障發生爭執 其後互毆

控方今早繼續開案陳詞，並在庭上播放兩名被告之間事前的 WhatsApp 語音訊息，劉子龍綽號為「牛屎」，其中一人問：「你手上有冇鎚呀？」另一人稱他家裡有一把，但亦可在「追思牆」附近路邊花槽尋找。

控方播放的公開影片顯示，其後一批手持雨傘的示威者與清理路障的人互相毆鬥及拋擲硬物，有人大叫：「打佢老母臭X！」最終拿傘的示威者撤退離開，清理路障的人歡呼。另一段影片顯示死者羅長清躺臥在路邊，數名市民在旁協助，拍片者問：「Call 咗白車未呀？」、「泵到頭呀？死呀，點算呀？」

控方指遭割傷男子X 認人手續中認出首被告

警方從死者羅長清手機檢取的影片顯示，羅倒地前正在拍攝，倒地後，手機仍然繼續拍攝，但只拍到天空和樹，以及有硬物在半空中互相拋擲，其後有一名市民上前協助，將羅移動至馬路旁。

控方播出事發前後兩名被告所居住大廈的閉路電視錄影片段，並指陳彥廷在警誡錄影會面時，在影片截圖中辨認出自己，並註明「我」。控方又指，男子 X 在警署認人手續中認出第一被告劉子龍。

審訊下午續。

被告劉子龍（案發時17歲、現年19歲）和陳彥廷（案發時16歲、現年18歲），同被控一項謀殺、一項有意圖傷人及一項暴動罪。控罪指，兩人於2019年11月13日在上水北區大會堂外連同其他不明人士謀殺男子羅長清；同日同地與不明人士意圖使男子X身體受嚴重傷害而非法及惡意傷害他；同日同地與不明人士參與暴動。兩人還柙至今逾2年，分別由大律師 David Boyton 和蒲立新（Ian Polson）代表。

案件編號：HCCC322/2020
