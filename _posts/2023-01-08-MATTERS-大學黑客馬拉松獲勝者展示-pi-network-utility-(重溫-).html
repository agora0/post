大學黑客馬拉松獲勝者展示 Pi Network Utility

作為招募新開發人員的外展計劃的一部分，Pi Network 自豪地贊助了2022年 10 月 14 日至 16 日週末在哈佛大學、康奈爾大學和加州大學伯克利分校舉辦的三場美國編程馬拉松。在那裡，來自全國和世界各地的熱情的學生開發者在 Pi 的平台上開展獨特的項目，並將 Pi Cryptocurrency 整合到他們的應用程序中。

作為上一篇黑客馬拉松博文的後續，今天我們將介紹我們的獲勝團隊和他們的 Pi 應用程序項目。參加 HackHarvard、BigRed//Hacks 和 CalHacks 的優秀學生每個人都有機會使用 Pi 的技術並直接向 Pi Core 團隊學習。這些學生開發者利用 Pi 易於使用的平台和開發者工具，展示了只需一個週末即可開發的創意原型。這些多樣化的項目標誌著可以在 Pi 網絡上構建的更多產品和用例的潛力，並且可以從區塊鏈集成中受益。Pi 的使命之一是擴大 Web3 應用程序的範圍，並拓寬基於區塊鏈的服務可以為世界上的日常生活提供什麼的視野。

今天，我們將重點介紹我們參觀過的三場編程馬拉鬆比賽的獲勝者。團隊的創新想法和 Pi 在他們的應用程序中的使用使他們脫穎而出；這些項目僅在一個週末的時間內建成，現階段還只是原型。我們很高興許多黑客馬拉松參與者，包括下面提到的一些團隊，都表示有興趣並打算繼續構建他們的 Pi 應用程序。核心團隊繼續為這些年輕的構建者提供技術和設計建議，目標是支持他們開發成成熟的 Pi 應用程序。

獲勝隊伍

在每次黑客馬拉松中，Pi Network 都會選出一支優秀的獲勝團隊，我們很高興向您介紹他們的項目！

HackHarvard 獲勝者：Sus 無人機

在 HackHarvard，獲勝團隊將他們的精力集中在構建一個促進可持續發展的無人機系統上。這個名為Sus Drones的項目由 Akshay Kumar、Garv Jasuja、Vijay Bhaskar Chiluveru 和 Muntaser Syed 領導，包括一個支持傳感器的無人機，可以準確測量其下方土地的 CO2、CH4 和反照率（太陽反射率），從而更好地圖表環境因素。

Sus Drones 團隊成員在哈佛大學獲得“最佳使用 Pi SDK”獎。

使用此信息，Sus Drones 可以計算農場或任何其他土地的碳排放量，並在 Pi Network 上出售相應的碳信用額度。農民、公司和個人將能夠看到他們土地上的確切碳排放量，並使用 Pi Network 平台和 Pi 硬幣購買或出售他們的碳信用額度。

為什麼選擇 Pi 網絡？Vijay Bhaskar Chiluveru 表示，Pi 是“一個令人興奮的合作夥伴，因為它在 Web3 中具有影響力的規模和地位。” 團隊成員 Akshay Kumar 補充說，他們希望利用“已經存在的明確受眾和用戶——加上易用性非常好，因為我們對區塊鏈知之甚少。”

Pi Network 很自豪能夠通過我們易於編碼的環境簡化複雜的項目。區塊鏈不應成為創新的阻礙，我們很高興 Sus Drones 的項目能夠繼續發展。

Cornell BigRed/Hacks 獲勝者：NFTree

另一個關注可持續性的項目是 BigRed//Hacks 的獲勝團隊NFTree，由 Atiksh Bhardwaj、Collin Li、Owen Oertell 和 Žiga Kovačič 創建。

NFTree 團隊從康奈爾大學的核心團隊成員 Swaroop Poudel（中）那裡領獎。

NFTree 是一個建立在 Pi Network 上的 NTF 項目，旨在資助和支持全球植樹。NFTree 的願景是每個 NFTree 代表一棵真實的、獨特的樹，用戶可以使用 Pi crypto 買賣這些代幣。NFTree 的專有算法考慮了樹木的類型、位置、年齡以及環境和氣候因素，為用戶提供了對真實樹木的健康和價值的估計。Pi Network 和 NFTree 有一些明顯的協同作用；正如團隊成員 Owen Oertell 指出的那樣，“我們的項目專注於可持續性，而 Pi 的平台是生態友好的，因此我們希望構建一個符合我們目標的平台。”

這個項目為社區和個人提供了以虛擬方式採取實際行動的能力，我們很高興看到這個項目能長多高！

CalHacks 獲勝者：Sparkle

在 CalHacks，在約 2,000 名黑客的競爭領域中，Sparkle作為一款娛樂性遊戲應用程序脫穎而出，其遊戲內 NFT 組件與 Pi 加密貨幣無縫集成。Sparkle 團隊成員 Aidana Mustafina、Arububim Roman、Merxon Yuan 和龍芯 Ajodo 創造了一種方式來吸引和娛樂我們超過 3500 萬活躍用戶的用戶群，其核心產品可以隨著時間的推移而擴展。

Sparkle 團隊成員向 Pi Network 創始人 Chengdiao Fan（中）和 Nicolas Kokkalis（右）展示他們的應用程序

在 Sparkle 中，用戶玩放置遊戲的目的是增加他們的“Sparkle”，並在達到某些成就時解鎖新的關卡。然後，玩家可以使用 Pi 硬幣購買遊戲內的 NFT，每個 NFT 都可以為他們的遊戲玩法添加新的獎勵或功能。而且，通過將 Sparkle 放在 Pi 生態系統上，玩家隨後可以將這些改變遊戲規則的 NFT 道具出售給其他玩家。

Team Sparkle 的 Godson Ajodo 向我們介紹了他們在 Pi 上的開發過程：“起初，我們認為創建 NFT 和 Pi 應用程序太複雜了。但在黑客馬拉松結束時，我們能夠推出一個功能齊全的原型，並且驚訝於我們在一天半的時間內如此輕鬆地完成了這項工作！”

在 Pi，我們同樣對團隊接下來的工作印象深刻和興奮！

美國黑客馬拉松突出了 Pi 加密貨幣的實用性和易用性

Pi Network 擁有一個極其易於訪問的平台——不僅適用於我們的成員，也適用於世界各地想要在此基礎上構建的開發人員。Sparkle、Sus Drones 和 NFTree 背後的開發人員利用了 Pi 的可訪問技術；開發人員不需要任何先前的加密或區塊鏈經驗——構建 Pi 應用程序的簡單步驟是 (1) 構建 Web 應用程序，(2) 添加 Pi JavaScript 庫，以及 (3) 進行 API 調用。

這些黑客能夠在一個週末的時間內完成一些令人難以置信的項目的第一階段，提供了一個經過驗證的成功案例研究，我們希望其他開發人員可以從中汲取靈感。

除了這些獲獎者之外，其他令人印象深刻的項目在黑客馬拉松過程中集成了 Pi 的技術，值得表彰。其中包括 Instafit（醫療保健和健身應用程序）、Pi Bakery（市場）和 Craft Pi（教育技術應用程序）。

週末創建瞭如此多的相關和多樣的項目，黑客們強調了 Pi 社區的用例範圍以及可以從 Pi Network 上開發中受益的應用程序。

正如這些黑客馬拉松所表明的那樣，在 Pi 平台上進行構建既優雅又簡單，並且是立即訪問擁有超過 3500 萬會員的活躍用戶群的好方法。Pi Network 上的開發人員可以利用簡單的支付和用戶身份驗證集成、低交易費用、環保協議和無縫用戶體驗，為數百萬經過驗證的全球社區成員創建一流的應用程序和實用程序。

最好的部分？借助 Pi 簡單且兼容的平台，無需專業語言或區塊鏈特定經驗即可在 Pi Network 上進行開發。

先驅者，如果您想加入樂趣並在 Pi 上構建應用程序，請在 Pi 開發人員門戶網站上成為 Pi 應用程序開發人員，並使用您最喜歡的 Javascript 兼容語言進行編程。如果您不知道如何自己編程但熱衷於為 Pi 生態系統做出貢獻，請與您的開發人員朋友合作或介紹 Pi 平台以構建基於實用程序的 Pi 應用程序。（請注意，此鏈接只能在 Pi 瀏覽器上訪問，Pi 瀏覽器是 Pi 移動生態系統的所在地。有關如何開發的更多詳細信息，請參閱此處的 Pi Developer github 。）

我們很高興看到你接下來會做什麼！讓創新之流流動起來。
