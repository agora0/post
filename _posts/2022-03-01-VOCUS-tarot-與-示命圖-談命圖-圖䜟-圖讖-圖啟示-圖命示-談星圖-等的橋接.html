tarot = Tarot = ㄒΑ人口圖 = 示亼卩口圖 = 示命圖，其中第一個 t 通過大寫 T 約略象徵漢字「示」的甲骨文字體中最簡單的一個字形，所以 tarot 一字也常常寫爲字首大寫的 Tarot，這樣就更能看出字首 T 與漢字「示」的橋接關係 。現代中文漢字多音譯為不知所云的「塔羅」或「塔羅牌」。

tarot = 談Α人口圖 = 談亼卩口圖 = 談命圖。此時字首 t 也和 tell 或 talk 表達「談話」、「談說」、「談論」、「言論」、「吐露」、「透露」、「告訴」、「告知」之類含意的單字相關。其中漢文「人」轉注漢文「卩」。

tarot = tarooottt 省二個 o 省二個 t = 圖言戈one.one.one.three.three.ten = 圖言戈一丨丨三三十 = 圖言戈韭十 = 圖言韯 = 圖䜟

tarot = tarrrooott 省二個 r 省二個 o 省一個 t = 圖言戈人人one.one.one.three.three = 圖言戈从一丨丨三三 = 圖言㦰韭 = 圖言韱 = 圖讖，意通前式橋接的「圖䜟」。漢字「讖」與「䜟」同義，而「讖」字也是「䜟」字的俗體字，亦即「讖」可能是以符號「从」來表達其發音特徵的「䜟」字之同義字。

tarot = 圖а連口T = 圖戶連口ㄒ = 圖启示 = 圖啟示，意通「圖䜟」或「圖讖」。其中 t 通過大寫 T 約略象徵漢字「示」的甲骨文字體之一；a 通過左右鏡射約略象徵漢文「戶」；「启」轉注「啟」。

tarot = 圖A卩口T = 圖亼卩口ㄒ = 圖命示，此時爲前述「示命圖」的同義倒裝詞。

清代段玉裁《說文解字注》記載「讖，驗也。驗本馬名。葢卽譣之假借。讖驗曡韵。有徵驗之書。河雒所岀書曰讖。十二字依李善鵩鳥、魏都二賦注補。釋名。讖、纖也。其義纖微也。从言。韱聲。楚蔭切。七部。」。也就是說，「讖」有「驗」的意思，也有「意義纖細微妙」之意；不過，「驗」字本來只是一種馬的名稱，後來纔被假借為「譣」的意思。

例如，古漢字文化的「讖書」就是一種細談事物內涵或揭露事情微言妙義的書籍；而最早的「圖讖」則是指談論天文星相圖之奧秘的言論或文本，但是此類學說在先秦時期就已經出現一些爭議，後世也沒有變成比較嚴謹而正式的學術。巧合的是，tarot 一字也能橋接於漢字「談星圖」(意通「談論天文星相圖」)，即 tarot = 談一日ox圖 = 談一日牛圖 = 談星圖，或意通「談星圖讖」。

換句話說，這類 tarot 、Tarot、「圖䜟」、「圖讖」或「談星圖讖」的學問，通常與講解者的詮釋技巧有很大的關聯，如果要變成比較正式而且具有較高可信度的學術，關鍵還是在於《說文解字》記載的「讖，驗也。」，也就是其談論的各種預測和建議，結果是不是真的應驗和有效。

漢字文化有一本由唐代流傳至今、仍然很有名氣的時運書《推背圖》，書中內容就包含了圖䜟式的配圖，此外還有卦象符號和詩䜟的記載，是一本頗爲燒腦的預言書。但整體來說，現在常見的 tarot，卻大多又是從西方文化東渡而來。(P.S. 參考〈《推背圖》第三十九象的試譯〉-- https://vocus.cc/SubtleSpaceOfText/61db7390fd89780001ea3855)

廣義來說，其實每個人都可能有過經驗的「作夢」(dream) 和「解夢」(oneiromancy)，也可能算是一種風格上比較私人化的「私人專屬圖䜟」，只不過夢境的內容通常是一種比 tarot 更加身歷其境、也更加逼真又立體的影像或圖像環境 -- 超四度空間虛擬實境 / 4D Tarot。(P.S. 參考  〈夢的原理-The Elements of Dream〉-- https://vocus.cc/Elements_of_Dream/5c6d7172fd89780001eb98cb)

P.S.2.

German、Deutsch：Taroch -- 圖示啟；談命之畫；談命彩繪；圖讖；圖䜟。但據說公元 15 世紀末到 16 世紀初，Taroch 一語也有「呆笨」、「蛋笨」、「愚蠢」甚至「白癡」、「白痴」之類的嘲笑式語意。Trionfi 則是類似「示命畫藝」或「談命畫藝」之意。

Italian：tarocch -- 圖示啟；圖裡有示啟；示命彩繪(畫)；談命彩繪(畫)；圖讖，即 tarocch = taroc.ch = 啚合繞圈形.讖 = 啚合〇.讖 = 啚合囗.讖 = 圖.讖 = 圖讖；圖䜟。此字可能源自公元 16 世紀初的 tarocho (圖示啟；圖裡有示启；示命彩畫；談命彩繪；圖讖) 一字，後來又出現一種流行的紙牌 trionfi (圖示啓；示命畫藝；談命繪藝；圖讖；圖䜟)。

English：triophe -- 圖示啓；示命畫藝；談命繪藝；圖讖；圖䜟。此字是 Italian 之 trionfi 一字的 English 式寫法。

Italian：tarocco -- 橙紅果，此字爲一種果肉顏色較紅的橙類水果之名稱，和前述 tarocch、tarocho、tarot 都無關。
