完整標題：mud 與「泥」或「泥土」或「溼土」或「溼泥」或「抹土」或「抹泥」或「汙沾」或「毋值」或「沒有值」或「非道」或「抹汙道」、「衊污道」等的轉換密碼


1. mud = 3.匕.尸 = 三.匕尸 = 氵.尼 = 泥，其中 m 轉 90 度約略象徵三個點狀符號構成的水偏旁符號「氵」，u 逆時針轉 90 度再左右鏡射約略象徵漢字符號「匕」且 d 轉 180 度約略象徵漢字符號「尸」而以 ud 轉換「匕尸」再組合轉換漢字符號「尼」。漢文「尸」本來也是一個側視的人形符號，常用來表達「身體」的意思，而漢文「尼」原來是寫爲兩個人形以背部靠近上下相貼狀，後來纔演化爲漢文「尸匕」的上下組合，或引申爲「黏貼」、「貼近」、「靠近」等類似含意，所以具有黏著性的建築材料「水泥」(cement) 也是用「泥」這個字。

【附圖：「土、泥」字形演化】

2. mud = mud.d 省一個 d = 泥.土 = 泥土，字尾 d 約略形聲漢語「土」的起音，但從 t 音轉爲 d 音。

3. mud = m.u.d = moist.u.d = 溼.與.土 = 溼與土 = 溼土，意思類似「泥土」。

4. mud = m.mud 省一個 m = moist.mud = 溼.泥 = 溼泥

5. mud = mu.d = 抹.土 = 抹土，也指「抹泥土」、「抹溼土」。

6. mud = m.mud 省一個 m = 抹.泥 = 抹泥

7. mud = m.u.d = 3.于.沾 = 三.于.沾 = 氵于.沾 = 汙.沾 = 汙沾，類似「弄髒」之意。其中 d 約略形聲漢語「沾」的起音之一，可能類似漢語「玷」的起音。

8. mud = mu.d = 毋 or 沒有.值 = 毋值 or 沒有值，此時意指「沒有價值的東西」或「無價之物」，其中 d 約略形聲台閩語「值」的 dâ 式發音之起音。

9. mud = m.u.d = 3.聯.道 = ε聯.道 = E聯.道 = ΕΕ.道 = ⴺⴹ.道 = 三丨丨三.道 = 非.道 = 非道，類似中文漢字「誹謗」之意。其中的「道」字不是「道路」的「道」，而是「莫道人之短」或「道聽塗說」的「道」。作此義解時，mud 又可能有「抹汙道」或「衊污道」之語意，即類似現代中文漢字俚語所謂「抹黑」的說法。其中「衊污道」的「衊污」是漢語「污衊」的同義倒裝詞，但是《說文解字》記載「衊，污血也。从血蔑聲。」，有可能古漢語「污衊」使用的「衊」字也是一種古俚語式的假借用字，而且此時「衊」字的發音不一定與「蔑」字同音，也有可能是類似「謾罵」的「謾」音。

muddle：糢糊；糊塗；糊塗亂作；糢糊容樣；糊塗容樣；糢糊的；糊塗的；泥搗容樣；攪混搗爛；攪混搗亂；攪混搗亂樣；攪混搗爛樣；糊塗心；糢糊頭；混亂腦。

muddler：泥搗棒、攪動棒、混搗亂支、混搗爛支、混兜融支，意通「攪拌棒」。

muddled：糢糊的；糊塗的；糢糊樣的；糊塗心的；心有沉重容樣的，類似「昏沉的」、「頭腦沉重的」，此時字首 m 與 mind (心) 相關。

muddling：糢糊的樣子；糊塗的樣；心有沉重容樣；糢糊之名動；糊塗之名動；抹糊之名動；抹塗之名動；泥塗之名動；混攪搗亂容樣；......。
