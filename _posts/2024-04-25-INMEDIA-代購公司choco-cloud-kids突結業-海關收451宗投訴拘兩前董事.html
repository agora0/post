（獨媒報導）嬰兒用品代購商Choco Cloud Kids早前突然宣布結業，指公司「嚴重資不抵債，在沒有新資金注入下，公司已無法繼續營運」，多名未能收貨的顧客設立苦主群組，並向海關。據指涉案代購網店由兩間公司營運﹐海關今日（25日）指昨日已拘捕公司兩名前董事，兩人正保釋候查，並指目前接獲451宗相關投訴，涉案金額共約40萬元，最大金額一宗涉約2萬元。

苦主群組逾二千人 前員工爆負責人呃貸款移民

2013年成立的嬰兒用品代購商Choco Cloud Kids 本月初於網站突然宣佈結業，現時網站內已無任何產品資訊，僅有一份結業公告，指公司嚴重資不抵債，無法繼續營運。公告中亦顯示，Choco Cloud Kids已於今年4月4日遣散員工，正安排清盤程序。

不少已在該商戶預購生活用品的消費者未能收到貨物，Facebook更設有「choco cloud mall 批發零售苦主」群組，目前已逾2,000人。不少苦主分享自己的情況，並呼籲「報警、報海關、報消委會」。有苦主質疑負責人移英置業，Choco Cloud Kids發聲明否認說法，指「突然結業主因是公司負責人身體健康原因」，而此前觀塘、荔枝角兩分店「一直正常運作」 ，加上負責人「一家早已移民」，反駁「走佬」說法。聲明中，該公司表示，香港經濟每況愈下，「像我們這樣一直由疫情到現今經濟危機仍苦苦支撐的中小企很需要政府再支持」，並稱「看不到未來」。

一位自稱2018年起任職Choco Cloud Kids會計的不具名人士亦向傳媒爆料，指公司在2021年參與「中小企融資擔保計劃」，並成功向滙豐銀行申請「百分百擔保特惠貸款」，總數逾280萬港元。爆料人士續指，公司負責人隨後將貸款由公司帳戶移至私人帳戶，並宣稱在內地賣樓、準備移民英國等。

Choco Cloud Kids再發公開信指，公司堅持按時還款至上月；因需要向境外收款人付款，而現時因人民幣管制，公司選擇在找換店兌換後透過私人戶口匯款，否認私吞貸款。聲明又指公司「錯判了疫情影響」，「做好老闆唔裁員蝕咗200多萬」，質疑爆料人前會計陳小姐未提及公司財政狀況。聲明最後稱該前員工「行為噁心」、「知道前東家失勢講大話來打落水狗」。

涉案約40萬 拘兩名前董事

香港海關今日舉行記者會，海關商品說明調查科第七組高級調查主任黃典俊指昨日（24日）拘捕公司兩名前男董事，年齡分別為33歲及38歲，涉嫌於銷售貨物時作出不當地接受付款的營業行為，違反《商品說明條例》，兩人正保釋候查。海關續稱，截至今日，共接獲451宗相關投訴，涉案生活用品包括服裝、化妝品、健康食品等，涉案金額共約40萬元，最大一宗涉約2萬元。

海關亦提醒商戶遵守《商品說明條例》規定，並提醒消費者網上購物時保持警惕，並保留交易單據作為紀錄，以便日後作出申訴。
