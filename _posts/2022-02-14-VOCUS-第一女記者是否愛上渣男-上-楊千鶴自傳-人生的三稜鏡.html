台灣第一位女記者，也是日治女高校生的厲害學姊之一 — — 楊千鶴，她的自傳《人生的三稜鏡》，愛情故事豐富到讓人忍不住捶心肝。​

​故事圍繞楊千鶴與文藝憂鬱青年 N 先生的愛情故事（還是遇到渣男？），但細究其他人物：憂鬱青年的前任、楊千鶴的女生朋友們，便會發現這個愛情故事一點都不單純。​​

與其說是愛情故事，更應該說，是質疑「為什麼女性的未來只有婚姻」的故事。

​

把自己放最後的母親​

楊千鶴在《人生的三稜鏡》裡面，自述自己是「差點被送走」的小女兒。母親異常地堅持把楊千鶴留在身邊照顧，才得以讓楊千鶴待在母親身邊安心長大。​

​楊千鶴非常喜愛母親，自傳描述的是一個溫柔的傳統女性，會照顧丈夫前妻的家人、喜歡位親戚朋友縫製衣服，凡事只為他人，總是把自己擺在最後。​

但是母親在她就讀靜修女學校的第二年去世。楊千鶴必須自己學習，如何作為一名女性在傳統與現代的價值衝突下生存下去。​

期待落空，總會讓人不小心長大。​

楊千鶴後來進去台北女子高等學院就讀，俗稱新娘學校。​

​雖說是新娘學校，但那裡提供的是十項全能的教育，楊千鶴會讀書，體育也不錯，擔當桌球部主將。新娘學校給予她的養分，反而使她長成敢追求不平凡的性格。​

這段成長的時光，還讓楊千鶴意識到，不只是女性會被人看輕，就連作為台灣人，在社會上也會受到不平等的待遇。​

​於是，楊千鶴決定推掉親事，畢業後進入職場工作。

​

頂著光環的女孩​

看在外人眼裡，楊千鶴頂著閃亮的新時代女性光環，但其實她沒有把自己看成新時代女性，因為她知道頂著光環的女孩是要付出代價的。​

楊千鶴先在磯永吉教授的徒弟底下當助理，在發現自己跟日本人同工不同酬後，萌生辭意。

後來西川滿挖角楊千鶴到臺灣日日新報當記者，答應她一定會跟日本人同工同酬。於是楊千鶴便跳槽，開始當起女記者。​

那時全報社只有三個台灣人，另外兩人的年紀可以當楊千鶴的爸爸。女性的話，全報社也只有三人，只有楊千鶴是台灣女性。​

楊千鶴簡直是臺灣日日新報的稀有生物。​

​偶爾也會發生仗勢楊千鶴是年輕女子，又是台灣人的身分，在一些地方被刁難的情況。​

不過楊千鶴的性格沒那麼脆弱，而且很幸運地有好同事相助，大致上在報社的日子，填補了楊千鶴剛畢業時的徬徨。​

也是在這段時期，她遇見了 N 先生。


信念之外的愛情來敲門​

若說楊千鶴始終環繞的信念是「平等」，想要平等地享受到不分男女、台日人都可以擁有的選擇權。​

那麼 N 先生像是楊千鶴信念之外，飛來的一個奇妙緣分。​

​N 先生是楊千鶴的同事，兩人第一次相遇，是因為 N 先生舉著楊千鶴的稿子，大聲問辦公室的人：誰的字跡這麼潦草啊？​

是個挺糟糕的第一印象。​

​兩人第二次相遇是在喫茶店，這時他們才發現彼此有如此多話題能聊。​

N 先生告訴楊千鶴，他出生自東京的望族，從小受到父親冷落，與家庭關係不好，有段時間躲在一名年長女性的家中。​

N 先生說，那名女子是他的初戀對象，兩人本來有婚約，但是女子突然想去當女演員，不願意被家庭束縛自由。N 先生在失戀的情況下，來到了台灣，然後跟楊千鶴相遇。​

​確實 N 先生舉手投足都是公子哥氣息，穿昂貴大衣、拿萊卡相機就算了，還會抱怨台灣喫茶店的咖啡超難喝。​

若是實力正常發揮的楊千鶴，這位公子哥應該不會全身而退才對，但可能楊千鶴曉得 N 先生受傷的過往，所以對他百般包容。​

尤其某次 N 先生看見楊千鶴母親的畫像，眼神閃過對家庭、母愛的羨慕。楊千鶴知道縱使 N 先生裝作清高的樣子，但其實是富有情感的人。​

後來，N 先生邀請楊千鶴到家中品嚐什麼是真正的好喝咖啡。（總感覺類似今天「要不要來家裡看 Netflix」）當時的女孩子光是走在男生身旁，就會被議論紛紛，何況是去男生家中。​

不過楊千鶴還是去了，兩人真的是純喝咖啡聊天，那個年代的曖昧大概就是如此樸實無華。​

兩人最親密的互動，頂多是某次在楊千鶴生日宴會上，N 先生喝了一些酒，試圖摟抱楊千鶴，但是楊千鶴拒絕了，N 先生也馬上道歉。​

當場兩人有些小尷尬……​

究竟兩人關係會怎麼發展？敬請期待下一篇。
