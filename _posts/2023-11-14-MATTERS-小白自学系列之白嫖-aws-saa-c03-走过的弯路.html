成为 AWS 入门级持证人员后，我又开始折腾 AWS 下一级证书 AWS Certified Solutions Architect Associate。只是在升级证书的过程中，小白如我这次走了不少弯路。

SAA-C03 和 SAA-C02 的区别

AWS 于 2022 年末更新了 SAA 考试的内容。对于当时刚接触 AWS 考证事业的我来说，两眼懵逼。没有细致了解 03 和 02 版本的区别，成为日后弯路的第一大阻碍……

数字增加说明升级，虽然还是一个级别的证书，但是对考试内容进行了一些调整。相对于 C02 来说，2022 年升级改版的 C03 增加了「安全架构」类别的比重。简而言之，考试范围还是那些，但考试侧重点有变化……

改版换代新晋学习资源少，鱼龙混杂

更新改版升级意味着市场需要一段时间适应新题型、新题库，找到靠谱的学习资源不容易。连 AWS 都没有 65 题版的官方题库，大伙还需上 Udemy 寻找第三方题库练习。

刚入门的我，就是在这样的背景下练了一些质量不过关的题，结果越练越糊涂……迷茫之中，某志同道合的题友点拨：

我曾经也是稀里糊涂做了一些不靠谱的题，上了一些不靠谱的培训班，浪费了好多时间，结果越学越糊涂……直到遇上了这款 Udemy 最高分良心课程后，如拨云见日茅塞顿开……

得到点拨的我已在错误的道路上浪费了很多折腾时间，才好不容易走上正确学习道路……

难度陡增，需要反复刷题强化

SAA-C03 助理云架构师考试题，当然比之前的 CLF-C01 的难度有升级。只是难度增长得有些陡峭，让大伙措手不及（当中自然也包括在考试路上走了弯路的我……）。

就知识点来说，助理级架构师考察的还是 AWS 那些服务。只是相对于之前的学徒级，不再是考察对 AWS 服务的认知和辨析，而是考察如何使用 AWS 服务们部署一个靠谱、安全、高效的云系统。

字面理解就是，题目变长了，不论题干还是选项。对于母语非英文的我，自然看得头疼。

除了题目变长，做选择题不再局限于可以辨析那种服务适用，而是在题目给定的场景下，选择最优解。简单理解，就是在理解 AWS 各项服务是干什么的后，还需要逻辑判断了……

考察点巨细，背诵量大增

AWS 的一大特点就是产品及其丰富，而且系列产品雷同，初学者很容易混淆。忽略细节囫囵吞枣，很容易又糊涂。

学徒级备考时，只要掌握 AWS 产品名称，服务范围，再找几套题练练就过关了。2 小时的考试，做熟后差不多 30 分钟就可以一套题。

到了助理级别，题目变长考察知识点徒增后。140 分钟的考试，刚开始头晕眼花做得极慢，且正确率极低。很多题友吐槽，做了 1 个小时 正确率仅 50% 不到，距离考试要求的最低通过率 72% 遥遥无期。这一点，对初学者，非常受打击。

需要持之以恒每天练习，才能不断提高刷题正确率……

综上所述，我在 SAA-C03 上走过弯路后，小白自学之路总结归纳如下：

  • 考试前，切勿心急气躁，需要静心认识考试范围，考试特征……

  • 考试级别增加，考试费用增加，考试时间增加，考试难度增加，意味着市场上能培训的有效资源减少。需要仔细挑选靠谱学习资源，在正确的道路上学习。在不靠谱的学习路上折腾，只能深陷泥潭事倍功半。

  • 证书越值钱，考试就越难过。125 美元一场的考试看价格就知道不是随随便便就能让你通过的，要认真准备反复刷题，才能发现考试精髓和重点——当你对 networking，操作系统，分布式系统的软硬件知识越深厚，SAA 考试需要记忆的知识点就越少，通过 SAA 考试就越容易。

价值 125 美元的考试意义：

  • 可以往简历上写，刷掉一波面试者。

  • 凭借助理级架构师当然不足以在业内找到 cloud engineer 的专业工作，但肯定比没有强。

  • 助理级架构师考试范围主要集中在 3-tire 或 2-tire 的系统部署和产品选择上。对于部署千万级点击量的复杂云系统有差距，但总的来说，对于分布式系统设计理论培训上，影响深远。
