本熊在一月十一日購入了方格子的 NFT ──非同質化代幣，為的不是炒賣，不是收入，而是為了它的背後意義。事實上，在去年十月左右本熊開始研究 NFT，一來是被那些發橫財的新聞引發出好奇心，二來是想要把自己認為有價值的內容放上一個可以「永久」保存及供人閱讀的地方，而 NFT 及 ISCN 會是一個好選擇。不過，這次我們不談 ISCN，只談 NFT。

NFT的真正價值

現在，傳媒主要以「收益」、「賺錢」作宣傳 NFT 的資本，加上很多新加入團體進行「收割行為」，導致區塊鏈在不太理解的大眾眼中，形成了一種印象不太好，甚至有欺詐成分的商業工具。至於 NFT、ISCN 的好處、加密貨幣的實際用途等，這方面的知識和新聞其實很少被傳統媒體「大力」介紹，因為他們認為大眾沒有興趣，所以不會主動報導，成為一種遺憾。

而接下來，本熊將會以個人見解討論 NFT 的真正價值。

一、去中心化

自香港經歷了二零一九年的社會運動後，尋找各類型「去中心化」的地方彷彿成了一個生活目標：銀行會因為純粹的政治原因而不合理地凍結某些反對派的戶口；社交媒體會因為政府政策及利益而把某些言論隱藏起來甚至刪除；通訊媒體更會因為政治因素及法律限制而把使用者的資料全數交給執法部門處理……不同平台基於各類型利益關係，而對使用者施加無形的自我審查壓力，這無疑是其中一個讓人們朝「去中心化」理念前進的原因之一。

本熊正是這個探索旅程的其中一員。

說到「去中心化」，就必須要討論現代藝術的價值體現。

很多人不明白 NFT 的世界中，為何一張畫風不被討好的作品可以賣上成百萬美元，甚至出現了不少梗圖，告訴人們想要的話，其實只需要按下右鍵儲存下來就好。但是，事實上購得 NFT 的人們並不害怕人們利用右鍵存檔，因為那張圖只代表了一個表面價值，而內裡的價值──例如一份深入玩某個遊戲後所獲得的成果、一個高級會員資格、一個永久的購物優惠、一個永久的諮詢服務或一個永久的儲存檔案方法等，這些附帶的價值將會依靠區塊鏈世界的技術，不會因為某些地方結業就變得不可被存取，或能夠輕易地被別人盜用、凍結或奪去──

除非當中涉及了駭客或中心化交易所的背後操控。

由於中心化，人們對於現實及價值的認知，會落在傳統教育、社會觀念及中心化市場規範上，這會導致藝術創作者的薪金，除非是十分有名的創作者，否則一般都會十分低廉。「為什麼你畫畫的薪金會等於我上班的薪金？」這個問題對於一般人來說是十分難以接受和理解的，同時人們會理所當然地認為醫生、律師、政客等擁有高薪厚職是「正常的」，而相對地，他們亦會認為作曲家、畫家、小說家除非很有名，否則乞食是「正常的」。

那麼，這個正常的點到底在哪裡？其實就落在於人們對於現實的價值認知。因此，當一件事能夠拋開現實、傳統、社會、教育等規範時，亦可以說是「創新」，那將會讓一件作品擁有另一個價格，而這就是「去中心化」的其中一個主要體現。若然你不拋棄以往或現在的那套舊有想法，應該永遠都無法理解當中的價值，及 Web 3.0 世界正在建構的實際運作模式。


二、對抗戰爭、政治及權力操控的另一個手段

近日烏克蘭與俄羅斯的戰爭得來了全球關注，俄羅斯對烏克蘭採取了眾多戰爭手段，而其中一項會令大眾十分擔憂的，正是中斷網絡及資金流通問題。有見及此，在政客們仍然猶豫要不要協助，甚至只想坐視不理的情況下，想要協助當地卻自問無能為力的他國市民便找上了區塊鏈世界。

面對俄羅斯的中斷網絡手法，Elon Musk──Tesla合夥創始人和總設計師在較早前已表示，將對烏克蘭開放星鏈衛星 Starlink 聯網服務，並寄送聯網終端設備。

而這個承諾在四十八小時內迅速兌現，烏克蘭表示現已收到 SpaceX 捐贈的 Starlink 衛星互聯網終端機，並向 Elon Musk 致謝。以上例子，跨過了政權的擺佈、限制，直接由一方到達另一方，不但證明「斷網」不足以讓區塊鏈世界停擺，更證明了大部分現代國家的政權質素有多差劣和無法被接受。

而在解決了網絡問題後，資金轉帳方法借由加密貨幣及 NFT 世界便能夠輕鬆抵達對方的手上。人們在這個過程中唯一要害怕的，只有電源、駭客及詐騙問題，而為了讓自己不會陷入被欺騙的危險之中，一些官方認證（如Opensea和Twitter都有的藍剔認證），以及他們直接發佈的消息將會成為關鍵，大眾亦必須理解到，什麼是「認錢包地址、合約地址，不認人」──

是的，這就是 NFT 的世界。

三、Play & Earn模式

作為半個動漫電玩及奇幻科幻迷，近年與 Web 3.0 世界理念──一個主要與基於區塊鏈的去中心化、加密貨幣以及 NFT 有關的概念──最接近的作品應該是《刀劍神域》。主角把儲存了遊戲資料的 SEED 發放出去，而後來這個種子會發芽成麼樣子，就是靠人們的想像力經營而成，而緊接著的就是「GGO（Gun Gale Online）」──一個可以把遊戲中的金錢直接轉換成真實世界可用的電子貨幣的遊戲世界，亦即現在開始在區塊鏈遊戲中流行的 Play & Earn 模式。

若然主角把 SEED 留在自己身上，不進行任何發佈或只作私人管理，也許故事中的世界不會加速發展成 Web 3.0，而「GGO」亦可能只是一個會停留在中心化管理或仍然處於開發階段的遊戲。同時間，要做到 Play & Earn，不讓自己的遊玩時間變成年長一輩口中的「浪費時間」，當中的認證方法正是採用了 NFT 技術。

本熊認為 Web 3.0 的理想狀態，將會實現一個現在仍未被達成的目標：幾乎每一個遊戲的資料，都可以與其他遊戲共用，同時數碼世界將會與現實世界掛鉤。現實中，以前有風靡全球的 Pokemon Go 成為 AR 遊戲中的榜樣，亦算是區塊鏈遊戲前身的重要例子，而在《刀劍神域》裡，則有《序列戰爭》（Ordinal Scale，OS），一個可以在現實裡透過 AR 技術遊玩的 RPG 遊戲，而這些遊戲的獎勵，將可以兌換成購物優惠、入場券或現金券。

《刀劍神域劇場版：序列爭戰》動畫內人們活於現實時所使用的科技技術，正是其中一個實現 Web 3.0 大眾化的其中一個要點，另一套曾經有提及過相關科技層面的作品是《Iron Man》，但那是只有有錢人可以享用的版本，而 Web 3.0 則是把這種技術達致大眾化為目標。

然而，Web 3.0 真正來臨時，人們又到底要如何認清數碼世界與現實世界呢？這將會是另一個大難題，同時亦似乎是一個也許未需要觸及的問題，因為 Web 3.0 還在起步當中。

購入的，是一份理念

在加密貨幣的路上，未改制的 LikeCoin 成為了創作者接觸加密貨幣的新手上路捷徑，讓本熊幸運地走上一條通往未知世界大門的路上；而在 NFT 的路上，由於方格子的 NFT 初期能夠用法幣購入，同時本熊對 NFT 的理解，以及曾經目睹過中心化危險性的經歷，讓本熊終於購入了第一個 NFT。

個人而言，只就表面價值來說，這兩張插圖確實不在自己的喜好範圍之中。不過以客觀的角度來看，亞洲區基本受到日系和寫實畫風影響，基本上大家的審美觀都是類近的，而方格子 NFT 的畫風，在亞洲區則是主要用在一些時事插畫或兒童繪本比較多，讓本熊能夠理解方格子的目標客戶及選取畫風的方向是什麼。

另一方面，在歐美區，這類畫風其實也很被大眾接受，他們的教育會令他們的眼光比較能接受到不同類型風格的東西──當然還是需要運氣。同時間，不再使用現有「眼光」來去看待一件事，而是去理解它的背後意義和價值，亦將會考驗參與 Web 3.0 人們的個人修為：

修身、齊家、治國、平天下。

接下來的日子，方格子將會邁向 Web 3.0，本熊對此有點期待，亦有點期許。作為 Web 3.0 的其中一份子，本熊認為方格子在未來應該要在「宣傳Web 3.0」及「創作者本身的魅力」當中取得平衡，同時亦要為自己所「打造」的 NFT ，定下更多的定義，而這些定義將能輔助創作者擁有更美好的生活，而不會成為單純的投資工具。

至於針對 Web 3.0 的宣傳手法，本熊認為其目的除了對外宣傳之外，還有要抵抗那些懷著惡劣及惡意的傳媒報導。本熊認為，正因為是踏足了 Web 3.0 的團體，宣傳方法應該有遠比傳統機構更加出色的方法，可惜本熊不是技術人員，所以亦不太清楚這方面知識可以如何協助到推廣。

不過，本熊亦因為未知，而擁抱了這個小小願望，希望 Web 3.0 的各個團體，都能找出可以積極推廣區塊鏈給大眾的方法，讓未知不再成為大眾的恐懼，而是一個屬於所有人的機會。
