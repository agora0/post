例如日本的AV ，如果直接複製 magnet 的話，有些在 Transmission 中會顯示檔案名稱過長，出錯無法下載
這時候，複製檔案的 hash 值，刪除重新以此方式添加，即可正常下載
其實這些只是 magnet 內的檔案名太長，torrent 直接打開的話不會。對於這種情況，可以使用此法解決

具體示例，
https://sukebei.nyaa.si/view/3827501
IPZZ-022 桜空もも 的作品，如果直接點擊 Magnet 用 Transmission 開啓，會出現
Error : Unable to save resume file : File name too long
這時要麼重新下載 Torrent 檔案，要麼直接複製出錯檔案 Details 內的 Hash 值
b1d1f1a2753c68d73c8872fa0a96576f6f4eb9ce
然後刪除重新添加即可下載
（桜空もも｜Momo Sakura Twitter @sakumomo1203

爲何我要如此做呢，原因大概有
- 我喜歡 Transmission ，不然的話，換成 FDM 或是 qBittorrent 並不會出現這種問題
- 日本類似這樣的 AV 在網路上標題名稱警察出現很長的，國內的環境與其多一步翻牆下載種子檔，不如 magnet 方便。所以每次會附帶上 AV女優的社交帳戶，希望至少能幫忙推薦一下

其實這種方法很有侷限性，只是適用於 Magnet 連結內檔案名過長、種子檔名稱不長
但，對於我接觸到的下載資源而言，已經足夠了
還有，看了 Transmission V4.0.0 的更新說明，好像也沒有提到解決了這個問題
