six = siks (實際發音) = si.k.s = 似介形 = 介 = 六，此時「似介形」是約略描述漢字「六」之形似漢字「介」的甲骨文或金文字體。其中 k 約略形聲台閩語「介」的起音。至於漢字「六」的初文爲什麼形似「介」狀呢？…… 有一說是源自上古形似「廬」的一種構架在陸地上的寮舍之象形；但是，筆者卻覺得也有可能是指「雙腳踩在兩根雪橇上」的象形，自然也和「雪中行」及「溜冰」的意像相關，所以漢字「六」本來也可能有「溜」甚至是「滑溜」、「溜走」、「開溜」、「暗中開溜」等含意，原因是漢字「六」也包含了「陰」的概念、甚至與「冰山」以及「一陽八陰」的概念有密切的關聯，本文稍後的推演便會提到。(P.S. 參考【附圖：六字演化】)
所以，我們會發現，June (六月) 一字隱約中也帶有漢字「介」的特徵，即 June = Jun.e = 介聯人.E = 介二𠆢.ㄈㄧ = 介II 𠆢.ㄇl = 介丨丨𠆢.⺆兩 = 介丿丨𠆢.⺆二 = 介介.月 = 介.月 = 六.月 = 六月。其中 Jun 轉換的漢字「介介」同時表達漢語「介」的起音之一及漢字「介」的字形，而漢字「六」的殷商甲骨文或金文字體之一又形似漢字「介」，所以此時也就等於是在轉換單一的漢字「六」。
此外，也可能有更精簡的一個橋接式：June = Jun.e = 一聯人.月 = 丨聯𠆢.月 = 丨丨𠆢.月 = 丿丨𠆢.月 = 介.月 = 介月 = 六月，此時 J  約略形聲台閩語「一」的起音之一，e 則同前述轉換漢字「月」。
不過，June 也能直接通過數字概念來轉換漢字「六月」，即 June = Jun.e = 一五聯.月 = 六.月 = 六月，其中 J 約略形聲台閩語「一」的起音之一，e 則同上式轉換漢字「月」。
另有一說，June 又源自 Ancient Roman 神話 (遠年昔時的人民的文化；古代的人民神話 / 古羅馬神話) 中「女性與婚姻女神」的名稱 Juno (結婚女，即 Juno = 結暈n偶 = 結昏nn = 結昏女女 = 結婚女，其中「暈」轉注「昏」；宙宇后；天后；天母)，她也是主神 Jupiter (主於家庭，即 Jupiter = 主於pig廷E广 = 主於豕廷ㄈㄧ广 = 主於豕廷ㄇ丶广 = 主於豕廷冖丶广 = 主於豕廷宀广 = 主於家庭，類似「一家之主」或「戶長」之意，但此「家」或「戶」是指此一神話世界的宇宙；宙宇神；眾神之王；眾神導領；眾小辟之導領，其中「小辟」意通「小上帝」或「小天神」；天王；天父) 之妻，戰神 Mars (戰神；戰鬥神；爭戰神，意通「戰爭神」；武戰神；馬司；天子，此「天子」帶有「宙宇神之子」的意義，與古漢字文化的政治術語「天子」無關) 之母。
由於 Juno 在此神話世界中主管女性與婚姻，可能影響到西方文化的六月份也與女人的婚事有關，據說「六月新娘」(June bride) 一說便是由此而來。按此看，June 除了有簡單的「六月」之意，還可能有「結婚女之月」的神話內涵。

six = sieks = 斜冫影鏡繫 = 斜二影鏡繫 = / \ 影鏡繫 = Λ = 六，其中「斜二影鏡繫」是表示「兩個斜線以鏡影的模式接在一起」而構成形似 Λ 的符號，這也是漢字「六」的金文字體之一，亦即漢語「六」的起音又可能與 Greek 字母 Λ (lambda) 相關。

東漢許慎《說文解字》記載「六，《易》之數，陰變於六，正於八。从入，从八。凡六之屬皆从六。」，也就是說「六」是表達漢字「陰」之不明顯、糢糊或暗示性質的變體符號，也許本來「陰」是指「山北謂之陰」、「六」是指「水南謂之陰」(即 six = sieks = south氵ek二 = 南水ekek = 南水ElㄑElㄑ = 南水曰連ㄥ丨三令冫 = 南水曰連ㄥ丨3今丶二 = 南水曰連ㄥ阝今丶二 = 南水曰陰) 也說不定，亦即「六」也有「陰」之意。
此外，其中「从入」的記載可能影響了漢語「六」的發音；而「正於八」指的應是「陰正於八」，所以這部份記載可能是說「『陰』的性質若以『六』這個漢字的意像來解釋，正好位在其中形似水平線的符號『一』之下方的符號『八』的位置」，就像冰山只露出水面上一小部份的一分量體，但水面下卻潛伏著表面上看不太清楚的、更大量的八分量體一樣。
所以，一般的船隻在海面上航行，若是看見漂亮的冰山之一角，千萬不要急著駛近去欣賞，否則很可能會發生 Titanic-sinking (鐵達尼號式的沉沒事件)。
換句話說，相對於漢字「九」也被古人用來表達「陽」的性質，漢字「六」這個符號所表達的「陰」其實可能是「一陽八陰」的感覺 (大約 11% 陽、其餘 88% 陰；另有 1% 可能處於現實面中的或陰或陽之陰陽交界狀態，例如冰山量體之處在水面附近而或浮或沉的那部份)，並非是相對於「全陽」或「九陽」的「九陰」或「全陰」之狀態。(P.S. 現代的冰山量體計算，通常以 2/21 為冰山露出海面的量體與總量體之比率，換算爲百分率則大約是 9.5238%，其誤差率可能受到冰山與海水的實際密度之影響。)

上述這個類似「冰山」(iceberg) 或「一陽八陰」(即 iceberg = aiceberg (實際發音) = 一1三昜be二今 = 一丨3昜bebe今 = 一β昜八EpE今 = 一阝昜八丨三pointL二今 = 一阝昜八丨3點ㄥ二今 = 一阝昜八β丶ㄥ二今 = 一阝昜八阝云今 = 一阝昜八阝侌 = 一陽八陰) 的概念，或許可以用來解釋爲何漢字「六」的異體字或大寫字要使用也可能意通「地」的「陸」字，因爲「地」在古代探討變化的經典《易經》的卦象中也有「陰」的性質，也因爲肉眼通常只能看到陸地表面上的東西，而陸地下方卻可能隱藏著龐大的未知。
按此概念，雖然古人也可能用「冰山」(iceberg) 來比喻漢字「六」(上一下八) 帶有的「一陽八陰」(iceberg；one yang & eight ying - 一陽卦及八陰卦) 之性質，但似乎不夠精簡、也不易隨處可見，於是纔使用也可能意通「地」的「陸」字來表達「六」的大寫字也說不定。
也就是說，雖然漢字「陸」與「露」發音有點相近，但性質上卻比較像是「露一隱八」 (大約 11% 露、其餘 88% 隱；另有 1% 可能處於若隱若現的狀態) 的感覺，不是全部都露出、也不是全部都隱藏，而是露的少、隱的多。
