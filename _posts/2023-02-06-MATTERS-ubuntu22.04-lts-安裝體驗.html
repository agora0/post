.7z 加密的檔案無法解壓，提示不支持，需要安裝完整版
install p7zip-full p7zip-rar

刪除 amazon 圖標及廣告
apt-get remove unity-lens-shopping

mpv 影片播放器
當使用觸摸板操作時，雙指調節播放進度或者音量速度太快
未解決，暫用鍵盤快捷鍵或者鼠標替代

最小化在 Dock 列的軟體，當快捷鍵再次啓用的時候，並不會顯示出來。只是在消息列有通知、圖標在 Dock 列晃動
這點尤其是使用 Fsearch 時候就很明顯不方便了。因 Fsearch 設定每次打開掃描更新一下 Database ，因爲我一般最小化，需要的時候直接按 F1 調出
現在 Ubuntu22.04 之下，他只是在 Dock 列掙扎着出不來

影片的縮略圖，如 ZorinOS 一般，需要安裝 ffmpeghumbnailer ，讓 Nautilus 顯示
sudo apt install ffmpegthumbnailer
sudo -H gedit /usr/share/thumbnailers/totem.thumbnailer

appimage 類軟體無法直接運行，即使已經給了作爲軟體的權限，需要安裝 libfuse2

Dock 列的軟體，如果已經最小化了，點擊一下會彈出，但是再次點擊並不會最小化（收回）

Gedit 編輯器內使用 Fcitx 輸入法，備選的字詞即使脫拽到下方，再次打字的時候依舊跑到左上方去

Dock 列的圖標顯示太過複雜了，例如 Ubuntu22.04 的安裝盤，插入筆電，Dock 多出了三個圖標

雖說是「問題」，也只是與 ZorinOS 對比，並且僅僅體驗了一天之後不習慣的地方。
Ubuntu 的界面，從不適應已經開始有點喜歡了。

因爲 ZorinOS 和 ElementaryOS 依然是基於 Ubuntu20.04 ，有點等不及了，所以想着先體驗下 22.04。Pop!OS 官網下載鏡像，被國內給屏蔽了，又需要翻牆下載，不然優先考慮他的。
