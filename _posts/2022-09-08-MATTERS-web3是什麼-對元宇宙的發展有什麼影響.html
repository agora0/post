什麼是Web 3.0?

Web 3.0 是網際網路的最新發展階段，主要能夠透過區塊鏈技術，讓網路系統運行達到去中心化的目的，所以只要任何人在 Web 3.0 使用應用服務，都會以去中心化的方式儲存資料數據，不再讓主導權掌握在少數企業上，將所有權還給使用者，使網路系統變得更加公開、公正、透明。

 

舉例來說，只要在Facebook上發佈貼文，都須經過 Facebook 的中心化審查機制，所有的保留數據的權力都掌握在官方手裡，如果哪天Facebook不幸倒閉或惡意關閉你的帳號，你所有的資料可能就會因此消失，但 Web 3.0 就能夠徹底解決數據所有權被特定企業或政府壟斷的問題。

 

Web 3.0 所具備的特性

1.去信任化

用戶間可直接進行交流，不須通過受信任的中介、把權力授予給其他人或組織。

 

2.分散式運算

於區塊鏈上的資料數據不在固定位置，其鏈上的資料數據將以 P2P 的方式分散同步資訊。

 

3.可驗證

因網路系統並無中央伺服器來管理用戶服務，所以使用時不需經過中心化許可。所謂驗證是指透過公開金鑰加密技術，確認使用者真的有上傳資料而且不可否認。

 

4.人工智慧與機器學習

在Web 3.0透過語意網(Semantic Web)和自然語言處理的人工智慧技術，使電腦能夠利用數據和演算法來模擬人類的學習方式來解讀資訊，提高資訊解讀速度和正確性來提供使用者所需要的數據。

 

5.不經許可

任何人都可以在區塊鏈上傳資料，無需經過第三方授權，維持元宇宙中的自由度。

 

Web 的演變階段

Web 3.0 主要的發展概念，是改進 Web 2.0 所產生的中心化問題，將數據的所有權轉向使用者，讓使用者在無需經過第三方的許可之下，就能自由傳遞和儲存，數據也保有安全性和隱私，並保留了原本 Web 2.0 的使用者互動式體驗。


Web 3.0 與元宇宙的關聯

元宇宙的基礎就是採用Web 3.0的技術，來建構出以區塊鏈技術運行以及去中心化的世界，可結合 AR、VR輔助設備，提供使用者在虛擬世界當中能夠互相交流的系統。

 

因此可將元宇宙視為虛擬的世界，Web3.0則是在虛擬世界中，建構使用者身份的存在方式。而當元宇宙的概念成功透過區塊鏈的技術實現，讓任何人都可以能夠自由地參與其中，則表示元宇宙將逐漸融入我們的日常生活。
