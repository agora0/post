完整標題：patrol 與「赴視繞行」、「赴於視查」或「赴於巡邏」或「赴於巡邏視查」或「巡邏」或「兵為於巡邏」(巡邏兵) 或「巡邏團人」、「巡邏隊伍」或「偵察」等的轉換密碼

【附圖：「巡、邏」字形演化】

1. patrol = p.at.t.r.ro.ol 省一個 t 省一個 r 省一個 o = 赴.A對.T.ㄦ.繞.偶L = 赴.目.T.ㄦ.繞.LL = 赴.目.示.ㄦ.繞.ㄔ = 赴.目示ㄦ.繞.行 = 赴.視.繞.行 = 赴視繞行，類似「赴視繞回圈行」，其中 at 通過「A對」轉換 AA 對接而象徵一目狀以轉換漢字符號「目」的象形文，t 通過大寫 T 象徵漢字「示」的早期象形文之一並意通「示」的偏旁符號「礻」，ro 則有 round (繞圓的) 之意而轉換漢字「繞」，ol 通過「偶L」即 LL 約略象徵由二個形似 L 形符號的旋轉組合而轉換漢字符號「ㄔ」的初文之一而代表漢字「行」。

2. patrol = p.a.at.t.r.r.o.l 省ㄧ個 a 省一個 t 省一個 r = 赴.於.A對.T.ㄦ.木.日.1 = 赴.於.目.示.ㄦ.木.日.ㄧ  = 赴於.目Tㄦ.木日一 = 赴於.視.查 = 赴於視查，其中第二個 r 通過 tree (樹木；植木葉葉、植木葉三，意指「植木葉多數」) 或 root (根) 等的組字符號之關聯轉換漢字符號「木」。

3. patrol = p.a.tr.o.l.tro.o.l 省一個 tro 省一個 o 省一個 l = 赴.於.tri-.和.L.辵.偶.羅 = 赴於.tri-和L.辵偶.羅 = 赴於.三和ㄑ.辵辵.羅 = 赴於.巛.辶辶.羅 = 赴於.巛.辶.辶.羅 = 赴於.巛辶.辶羅 = 赴於.巡.邏 = 赴於巡邏

4. 轉換式3 + 轉換式2 = 赴於巡邏 + 赴於視查 & 省一個「赴於」= 赴於巡邏視查，包含「巡查」之意。

5. patrol = plural.atro.l = 複atro.l = atroatro.l = atroat.rol = Atri-one1three.羅L = 夊三一丨三.羅辶 = 止彡𠃊三.邏 = 龰彡ㄑ三.邏 = 辵ㄑㄑㄑ.邏 = 辶巛.邏 = 巡.邏 = 巡邏

6. patrol = p.a.a + patrol (轉換式5) 省一個 p 省二個 a = 兵.為.於.巡邏 = 兵為於巡邏，類似「巡邏兵」之意。其中字首 p 約略形聲漢語「兵」，第一個 a 約略象徵漢字「為」並約略形聲漢字「為」的尾音，第二個 a 則通過 aA 轉換漢字「於」的金文字體。

7. patrol = patrol (轉換式5) + ol 省一個 ol = 巡邏 + o.l = 巡邏 + 團.人 = 巡邏.團人 = 巡邏團人。其中 o 約略象徵兼會意漢字「團」，字尾 l 約略形聲台閩語「人」的起音。

8. patrol = patrol (轉換式5) + trol 省一個 trol = 巡邏 + t.r.o.l = 巡邏 + 隊.連.五.人 = 巡邏 + 隊.連.五.亻= 巡邏 + 隊.連.五亻= 巡邏 + 隊.連.伍 = 巡邏 + 隊連伍 = 巡邏 + 隊伍 = 巡邏隊伍，包括「巡邏隊」之意。其中 trol 之 o 約略形聲台閩語「五」的 gô 式發音之尾音。

9. patrol = pa.tro.l =卜A.察.人 = 卜貝.察.亻= 貞.察.亻= 貞察亻= 偵察，其中 A 約略象徵一個貝殼狀而轉換漢文「貝」，不過漢字「貞」下部的符號「貝」在早期的象形文中是畫爲鼎器的形狀，而漢文「鼎」的起音爲 d，因此漢字「貞」、「偵」兩字和 detect (偵探；偵測；偵察)  一字可能比較直接相關；tro 這部份約略形聲漢語「察」。此義是筆者在英漢字典上纔看到的，所以試著橋接一下。
