腦中空白了幾秒，再慌張了幾秒，檢查細節發現錢包地址竟不是一直熟口熟面那個。原來 terrastation 更新了，不再支援 Ledger cosmos app 預設的 path，於是現在顯示的錢包是在另一條 path 中的錢包。以往用 Ledger 上的 cosmos app 再也不能操作 terrastation 上的 UST，因為 cosmos app 跟 terrastation 用的 path 不相同，我的 UST 雖仍是安全的，但慣用的工具介面再操作不了那錢包了。

不用怕，社群是我們的朋友！有人提供了解方，在 legacy.station.terra.money 仍可操作舊 path 的錢包地址。

在右上方點擊登入按鍵，選 “Access with Ledger”，hurray! 顯示出我的 UST 存款了！

Terra 官方建議 UST 還是不要再存在舊 path 下的錢包地址，應該轉移到 terrastation 的 path 下的錢包。直觀程序是先把存款在 legacy.station.terra.money 上 withdraw 出來，再發送到現在 terrastation 上在新 path 下的地址。但這樣一出一入會浪費手續費，所以我選擇直接從舊錢包發送 aUST 到新錢包。

aUST 是在 anchor protocol 上存款的憑證，legacy.station.terra.money 網站預設不會顯示 aUST，要手動加上。在 wallet 頁下方選 “Add token”，再輸入 aUST 的合約地址 terra1hzh9vpxhsk8253se0vv5jj6etdvxu3nv8z07zu，畫面會顯示出 aUST 的選項。點 “Add” 把 aUST 加到工具中，在 wallet 頁便能看到錢包中的 aUST 了。


直接在 wallet 頁把 aUST 發到 terrastation 上的新錢包地址。保險起見，可先試發出小額，確認收到後再把餘額發出。留意你的舊錢包必須有少量 Luna 或 UST 作 gas fee。


我成功發送了 1 aUST 到新的錢包地址，在 Anchor Protocol 網站以 terrastation 登入 earn 頁面，立即見到 1.171 UST 存款。接著只需用同樣方法便可安心把存款從舊錢包轉移到新錢包了。

說到這裡，若有人不知道 UST 是什麼：UST 是 Cosmos 生態中的美元穩定幣，目前在 Anchor Protocol 的存款年利率高達 19.4%。留意穩定幣不是全無風險，自己量力而為啊。
