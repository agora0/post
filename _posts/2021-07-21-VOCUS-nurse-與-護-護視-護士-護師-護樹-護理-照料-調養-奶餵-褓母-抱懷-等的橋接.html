2. nurse = nurse + se 省ㄧ個 se = 橋接式1 + 視 = 護 + 視 = 護視，類似「看護」之意。其中 se 約略形聲漢語「視」。

3. nurse = nurse + se 省ㄧ個 se = 橋接式1 + 士 or 師 = 護 + 士 or 師 = 護士 or 護師。其中 se 約略形聲漢語「士」或「師」。

4. nurse = nurse + se 省ㄧ個 se = 橋接式1 + se 省ㄧ個 se = 護 + 樹 = 護樹。此時爲農業用語。

5. nurse = nurse + rse 省 rse = 橋接式1 + 里雙E = 護 + 里EE = 護+ 里ⴺⴹ = 護 + 里𤣩 = 護 + 理 = 護理。其中 r 約略形聲漢語「里」的起音，se 通過「雙E」即「EE」即「ⴺⴹ」的重疊組合轉換漢字符號「𤣩」。此外若字尾 se 同時約略形聲漢語「士」或「師」則也有「護理士」或「護理師」之意。

6. nurse = n.ur.s.e = 刀.聯r.四.E = 刀.rr.四.日 = 刀.料round.灬.日 = 刀料〇灬日 = 刀料口灬日 = 照料，類似「照顧」之意。其中 s 約略形聲漢語「四」而代表漢字「照」中的四個點狀符號「灬」，e 則通過大寫 E 約略象徵漢字符號「日」。

7. nurse = nurse + se 省ㄧ個 se = 橋接式6 + se = 照料 + 細心 or 悉心 = 照料細心 or 照料悉心。意通「細心照料」或「悉心照料」。其中第二個 e 通過 eE 轉換漢字「心」，即以 e 形聲華語「ㄧ」轉換數字 1 且以 E 代表「L二」的組合而經由「1L二」的組合約略象徵漢字「心」。

8. nurse = n.ur.se = 冂u二.食羊 or 氵台 = 冂uu.養 or 治 = 冂聯曰.食羊 or 氵台 = 冂吉言.養 or 治 = 周言.養 or 治 = 調.養 or 治 = 調養or 調治。其中 uu 轉換「聯曰」且以「聯」旁通「吉」(有「聯結」之意)、以「曰」會意旁通「言」，e 約略形聲漢語「羊」的起音或約略形聲漢語「台」的 yí 式發音。

9. nurse = nurrse 省ㄧ個 r = nurse.u.round.立.soil.養 = 護.育.〇.立.土.養 = 護育口立土養 = 護育培養。此時意通「培育」、「培養」、「護育」、「養護」之類。其中字首 n 代表 nurse 而通過「橋接式1」轉換漢字「護」。

10. nurse = nurrse 省ㄧ個 r = 人育round木飼養 =亻育〇木飼養 =亻育口木飼養 = 保育飼養。此時意通「保育」、「保養」、「養育」、「育養」、「飼養」之類。其中 n 約略象徵漢字「人」的篆文字體而旁通漢字「人」的偏旁符號「亻」，r 則通過 tree (樹；植木葉葉) 轉換漢字「木」。

11. nurse = n.u.r.se = 奶.畏.連.食 = 奶.畏連食 = 奶.餵 = 奶餵。此時爲「餵奶」的同義倒裝詞。其中 u 約略形聲漢語「畏」的起音再轉音。

12. nurse = 奶與女雙ㄧ = 奶與女雙丶= 奶與母 = 奶母，意通「奶媽」。其中 r 約略形聲台閩語「女」的起音，rse 形聲轉換的「女雙ㄧ」或「女雙丶」則是在描述漢字「母」的象形字是在「女」的象形字中加入了雙點符號。(P.S. 參考【附圖：「女、母」字形演化】)

nurser：護視人；護樹人；護理者；照料者；照料細心之人；照料悉心之人；調養者；調治者；護育培養者；保育飼養者；奶餵者；奶母之人；褓母之人；奶瓶，此時 nurser 通過「nurser = n.u.r.s.e.r = 奶.瓦.人.雙.1.二 = 奶.瓦.人雙.｜二 = 奶.瓦.人人.干干 = 奶.瓦.人人干干 = 奶.瓦.并 = 奶.瓦并 = 奶.瓶 = 奶瓶」的橋接式轉換漢字「奶瓶」二字，其中 rser 通過「人雙｜二」會意轉換漢字「瓶」中之符號「并」的篆文字體之形似「人人干干」的組合。

nursery：護園；護視園；護樹園，或意通「苗圃」；護理園；照料園；調養園；調治園；調治療養；護育培養園；保育飼養園；褓母園，或意通「育兒室」、「幼兒園」、「托兒所」、「育幼院」之類。其中字尾 ry 轉換「round.袁」轉換「〇袁」轉換「囗袁」再組合轉換漢字「園」。

nurseling：奶母奶養子；護育培養領養子；保育飼養認養子；褓母認養子。意通 nursling ㄧ字。

nursery rythem：護育培養園歌；保育飼養園曲樂；褓母理幼律韻調和音樂。類似「育兒歌」、「童謠」、「兒歌」、「搖籃曲」之類。
