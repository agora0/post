上述由「航快」到「行快」這類含義源自 Old English 的 hulc，或許和憑藉風力推動而航行快速的帆船相關 (P.S. 證據在 hulk 也可能橋接於漢字「風力快」，即 hulk = hu.l.k = 風力快，其中 hu 約略形聲台閩語「風」再轉音)，但也許是因爲本來屬於輕舟快船的這種帆船，後來又被使用在載運大量貨物，hulk 一字到了 Middle English 時卻反而變成「負重而行」甚至有點「笨重難行」之意：

hulk = h聯而行 = hh而行 = 負heavy而行 = 負重而行，其中第一個 h 約略形聲漢語「負」的起音，第二個 h 代表 heavy 而轉換漢字「重」(heavy = 人EA與ㄧ = 亻EΛ一與一 = 亻曰土與一 = 亻里與一 = 重，其中符號「Λ一」的上下組合即漢字「土」的甲骨文字體之一）。

hulk = h聯難行 = hh難行 = ph.heavy難行 = f重難行 = fool重難行 = 笨重難行。其中 l 約略形聲台閩語「難」的起音，fool 轉換漢字「笨」(fool = fo.ol = F偶.woodㄧ = 亇偶.木ㄧ = 亇亇.本 = 竹.本 = 笨，其中 F 約略象徵漢文「亇」)。
