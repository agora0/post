完整標題：deposit 與「貯放」、「置放」、「貯放置」、「貯放存」、「置放存」、「貯存」、「貯」、「儲」、「存」、「儲存」、「貯放金錢」、「置放金錢」、「存錢」、「儲金」、「存款」、「儲蓄」、「帳戶存」或「抵押品于借貸」、「擔保之證」、「抵押保證」、「抵押保證金錢」或「訂金」、「下訂金」、「付錢以訂」、「訂銀付先」或「沉積之物於底」、「沉積」、「疊積物沉之堆」、「底部沉澱」、「沉澱」等的轉換密碼 



1. deposit 與「貯放」的橋接轉換：deposit = de.po.sit = 貯.方.手with卜 = 貯.方.手與卜 = 貯.方.攵 = 貯.方攵= 貯.放 = 眝放，其中 de 約略形聲台閩語「貯」；po 通過 pho 約略形聲漢語「方」；「攵」在象形文中原來大多是「手形符號」和符號「卜」的組合，而此處 s 約略形聲漢語「手」的起音之一，i 代表 with (與)，t 約略象徵漢字符號「卜」。

2. deposit 與「置放」的橋接轉換：deposit = de.po.sit = 置.方.手與T = 置.方.㐅與𠂉 = 置.方.攵 = 置.方攵 = 置.放 = 置放，其中 de 約略形聲台閩語「置」；po 通過 pho 約略形聲漢語「方」；「攵」在象形文中是「手形符號」和符號「卜」的組合，但後世字體中演化爲形似符號「㐅」和符號「𠂉」的組合，而此處 s 約略形聲漢語「手」的起音之一即意通此處符號「攵」中的「㐅」形符號，i 代表 with(與)，t 通過大寫 T 逆時針轉 90 度約略象徵此處的漢字符號「𠂉」。

3. deposit 與「貯放置」的橋接轉換：deposit = de.po.sit = 貯.放.四與直 = 貯.放.罒與直 = 貯.放.置 = 貯放置，其中 po 約略形聲台閩語「放」但省去尾音；s 約略形聲漢語「四」的起音而以「四」約略象徵漢字符號「罒」；t 約略形聲台閩語「直」。

4. deposit 與「貯放存」的橋接轉換：deposit = de.po.s.i.t = de.po.s.i.T  = 貯.放.子.ㄧ.亻= 貯.放.子ㄧ亻= 貯.放.存 = 貯放存

5. deposit 與「置放存」的橋接轉換：deposit = de.po.s.i.t = de.po.s.i.T  = 置.放.子.ㄧ.亻= 置.放.子ㄧ亻= 置.放.存 = 置放存

6. deposit 與「貯存」的橋接轉換：deposit = de.po.s.i.t = de.拼和.s.i.T = 貯.拼和.子.ㄧ.亻= 貯.拼合.子ㄧ亻= 貯.拼和.存 = 貯存，此處「拼和」ㄧ詞僅作爲構字的連結語意之用。

7. deposit 與「貯」的橋接轉換：deposit = de.p.o.si.t = 宁.貝.合.之.text = 宁.貝.合.之.文字 = 宁貝合之文字 = 宁貝 = 貯，此處的轉換方式是在描述漢字「貯」由漢字符號「宁」和「貝」所組合而橋接於單一的漢字「貯」。

8. deposit 與「儲」的橋接轉換：deposit = de.p.o.s.i.t = de(mo-).point.o.三.with.the = 人(民之).點.口.三.與.者 = 人.點口三.者 = 亻.、口三.者 =亻.言.者 = 亻言者 = 儲

9. deposit 與「存」的橋接轉換：deposit = d.ep.osit = 直.ㄧ複.和.o繫t = |.二.子 = 扌子 = 存，此處的轉換方式是約略描述漢字「存」的篆文字體 (形似「扌」和「子」的組合且「子」字在象形文中又部份略似字母 ot 或「o十」上下組合的樣子) 而橋接於單一的漢字「存」。

10. deposit 與「儲存」的橋接轉換：橋接式8 + 橋接式9 = 儲 + 存 = 儲存。

11. deposit 與「儲蓄」的橋接轉換：deposit = 橋接式8 + sit + sit 省兩個 sit = 儲 + 雙之t + 玄與田 = 儲 + tt + 畜 = 儲 + tt畜 = 儲 + 艹畜 = 儲 + 蓄 = 儲蓄。deposit 和 save ㄧ字在此有類似的意思，但是 save 比較偏向「蓄積」、「蓄維」、「存」或「省下」、「少付」及「節約」的漢字語意。

12. deposit 與「貯放金錢」或「置放金錢」、「存錢」、「儲金」、「存款」的橋接轉換：de.po.sit = 貯.放.金錢 = 貯放金錢，意通「置放金錢」、「存錢」、「儲金」、「存款」之類。其中 si 實際發音 zi 而約略形聲華語「金」或台閩語「錢」且 sit 又同時通過 s.it 轉換「雙.點ㄧt」轉換「雙.、ㄧt」轉換「雙.戈」轉換「雙戈」轉換「戈戈」而與「金」組合轉換爲漢字「錢」；此外 sit 也可通過 ssitt 轉換「士示點TT」轉換「士示´𠂉亻」轉換「士示´𠂉人」轉換「士示欠」的組合而橋接於漢字「款」。在當今一般銀行術語中，deposit 通常和 withdraw (回抽；倒抽；倒拉；回退；退回；退出；退還；回撤；出撤；收撤；收回；出款；退款；要求退還；為於提退款、出提管托物，類似「提款」或「退出托管物品」。ㄧ般銀行辦理提款時也多用此字) 是相反的操作行為。

13. deposit 與「抵押品關于借貸」的橋接轉換：deposit = d.e.p.o.si.t = d.e.p.of.si.T = 抵.押.品.關于.昔.亻貸 = 抵.押.品.關于.昔亻.貸 = 抵.押.品.關于.借貸 = 抵押品關于借貸，意通「抵押品于借貸」，其中 t 通通大寫 T 約略象徵漢字符號「亻」並同時約略形聲漢語「貸」的起音。

14. deposit 與「擔保之證」的橋接轉換：deposit = d.e.po.s.i.t = 擔.與.保.之.言.登 = 擔與保.之.言登 = 擔保.之.證 = 擔保之證，即類似「擔保的證明」或「保證」之意，似乎也含概了「保證書契」之類，因爲字尾 sit 也可能同時通過 s.i.t 轉換「書三.|.刀大」轉換「書.三|刀大」轉換「書.丰刀大」轉換「書.契」轉換「書契」二字。

15. deposit 與「抵押保證」的橋接轉換：deposit = d.e.po.s.i.t = 抵.押.保.之.言.登 = 抵.押.保.之.言登 = 抵.押.保.之.證 = 抵押保之證 = 抵押保證，包含「抵押品」、「押金」之類。

16. deposit 與「抵押保證金錢」的橋接轉換：deposit = deposit + sit 省一個 sit = 抵押保證 + 金錢 = 抵押保證金錢，類似「抵押保證金」或「保證金」、「訂金」之類，其中 si 約略形聲華語「金」或台閩語「錢」且 sit 又同時通過 s.it 轉換「雙.點ㄧt」轉換「雙.、ㄧt」轉換「雙.戈」轉換「雙戈」轉換「戈戈」而與「金」組合轉換爲漢字「錢」。

17. deposit 與「沉積物之於底」的橋接轉換：deposit = d.e.p.o.s.i.t = 沉.1E.貝.禾ob(ject).之.於.底 = 沉.丨三.貝.禾物(件).之.於.底 = 沉.龶.貝.禾物.之.於.底 = 沉.龶貝禾.物.之.於.底 = 沉.積.物.之.於.底 = 沉積物之於底，意即「沉積物在地部」之意，也就是「底部的沉積物」。此處 d 是約略形聲台閩語「沉」的 dîm 式發音之起音。此外，deposit 也可以直接轉換漢字「沉積」，即：deposit = de.posit = 沉.貝禾三with直 = 沉.貝禾三與丨= 沉.貝禾龶 = 沉.積 = 沉積。

18. deposit 與「疊積物沉之堆」的橋接轉換：deposit = d.e.p.o.s.i.t = 疊.1E.貝.禾ob(ject).sink.之.堆 = 疊.|三.貝.禾物.沉.之.堆 = 疊.龶.貝.禾物.沉.之.堆 = 疊.龶貝禾物.沉.之.堆 = 疊.積物.沉.之.堆 = 疊積物沉之堆，類似「沉積物的堆疊」之意，ㄧ般而言所謂「沉積物」的形成也都有一段堆疊的過程。

19. deposit 與「底部沉澱」的橋接轉換：deposit = de.po.s.i.t = t = 底.部.sink.with.澱 = 底.部.沉.with.澱 = 底.部.沉.與.澱 = 底.部.沉與澱 = 底.部.沉澱 = 底部沉澱，其中 po 約略形聲台閩語「部」或也與 part (部份、部段) ㄧ字相關，t 約略形聲漢語「澱」的起音。

20. deposit 與「沉澱」的橋接轉換：d.e.po.s.i.t = 沉.與.point.三.with.殿 = 沉.與.point三.與.殿 = 沉.與.點三.與.殿 = 沉.與.氵.與.殿 = 沉.與.氵與殿 = 沉.與.澱 = 沉澱，其中 d 同前述約略形聲台閩語「沉」之 dîm 式發音之起音，s 形聲漢語「三」的起音，此處 pos 即通過「point三」轉換「點三」再轉換所謂「三點水」的漢字符號「氵」。
