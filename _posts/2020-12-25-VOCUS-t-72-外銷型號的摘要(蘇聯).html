Object 172M為T-72A的外銷項目，在設計定制時，根據面向的客戶，分為兩種外銷子型。其中每一次升級的E/E3/E5 都為向華約國家出口型號，而同期推出的代號為E1/E4/E6的為向第三世界國家出口子型，兩者在技術水平上有一定的差異，特別是在172M-1-E4上，出現了嚴重技術水平縮水的情況。

由此在對T-72的外銷型號的討論中，經常出現猴版的說法，最主要是因為在海灣戰爭中，大量的均質裝甲砲塔的伊拉克T-72M型在與美軍的戰車對戰中顯得不堪一擊，以至於戰後美軍曾對嘲諷蘇式裝備不堪一擊的西媒略有自大的態度這樣嚴肅地表示：the US fought only against monkey models, real Soviet tanks were much stronger!

對此簡要梳理Object 172M整個項目的對外出口的型號概況，以圖片說起：


Object 172M-E/E1 "T-72 export" 

對應1976年塔吉爾廠下線的T-72 Ural Obr. 1975，T-72 export 其為該型的出口型號，或者一些資料將其稱為T-72G。其中172M-E 是對華約國家的出口型號，E1是對第三世界國家的出口型號。在1979年該型才第一次在東德公開照片。在E 和E1型兩者的技術水平是非常接近的，主要區別在於核生化防護系統，第三世界國家的戰鬥中基本不存在染核區域的通行情況，因此將會看到包括本型的第三世界版本都是在NBC上有所簡化。

東德陸軍T-72 Ural 主力戰車

T-72 Ural   砲塔圖

Object 172M-1-E3/E4 "T-72M"

該型雖然基於T-72A obr. 1979設計，但與T-72A obr.1979 型不同的是，E3型的砲塔是T-72 Ural的石英砂填充的複合裝甲，而不是T-72A obr.1979上更貴的燒結陶瓷管填充的複合的裝甲。但在E4上，卻是比E3技術更糟糕，在伊拉克戰爭中被爆打後;西方技術人員切開其砲塔裝甲發現，在E4的砲塔中均為均質鑄造砲塔，砲塔上的設備也停留T-72 Ural 的技術水平，甚至E4的核生化防護系統也使用的是E1的簡化版本。此外，底盤上的懸掛系統也是T-72 Ural的。

波蘭陸軍 T-72M 主力戰車

      ※T-72M 在1980年在波蘭和捷克斯洛伐克被授權和生產。

T-72M 砲塔圖

Object 172M-1--E5/E6 "T-72M1" 

在1982年，蘇聯得到中東戰場上敘利亞繳獲的馬加赫主力戰車;當時敘利亞陸軍裝備T-72戰車的是蘇聯自用型T-72A obr.1979，測試後發現T-72A obr.1979的首上難以防禦以色列的105mm APFSDS彈，因此在1983年開始給外銷國緊急發布了E5/E6的修改版本在裝甲上解決措施是對首上加裝了一層16mm裝甲板，以及把砲塔換成了T- 72A 該有的燒結陶瓷管填充的複合裝甲。

這個版本是基於E3/E4的升級，對懸掛、砲塔、車長觀瞄等進行了改動。當然與T-72M最大的區別是T-72M1增加了可以加裝附加裝甲的螺栓組，使得一般在照片中反而成區分T-72M和T-72M1的一個特徵。在出廠狀態上的E5/E6上也開始出現了7+5的煙霧榴彈發射器，但這一點不能足以區分E3與E5，因為在波蘭的生產線上，因為E3轉入E5比蘇聯慢，在他們生產的晚期E3上就出廠時就加裝了煙霧榴彈發射器。

T-72M1是在1983年被生產，E5型號為華約版本，E6為第三世界版本。

芬蘭陸軍T-72M1主力戰車

T-72M1 砲塔圖

Object 172M-E7/E8 "T-72M1M" 

T-72M1M是基於T-72M1的升級，達到了T-72B的水平。使用1A40型火控系統、接觸-5爆反等。但問題是，T-72M1M的外銷價格不菲，華約國家並沒有採購。E8是對第三世界國家外銷的T-72M1M，與E7相比，不具備發射砲射導彈，與T-72B1一樣。

而E8 額外有個型號是T-72M1M，由烏拉爾廠在2002推出沿用蘇修時期的T-72M1M的設計，不過在武器展中加裝了競技場主動防禦系統。貌似印度購買T-72M1是購買E8型，他們稱為T-72 Ajeya MK2。使用有接觸-1反應裝甲共有155塊。

俄羅斯T-72M1M 主力戰車

印度陸軍Ajeya MK2 主力戰車

Object 172M-E9 "T-72S"

比E8還簡化的T-72S。使用了比E7更便宜的晝夜觀瞄系統，T-72S型戰車同為T-72B戰車的外銷版本；掛有接觸-1反應裝甲，與T-72B ERA區別是反應裝甲數量為155片而已，而非T-72B ERA的227片；據資料表示，該戰車舊稱為“T-72M1M1”；防護力也稍差許多，很大程度還是維持在T-72M1水平，而非與T-72B同級；這批戰車在蘇聯解體後被賣到了伊朗。
