随着第一期 MAPO 社区大使招募进入尾声，我们顺势启动 Web3.0 开发者大使招募计划。我们希望通过这一计划连接更多区块链开发者，一起建设真正的去中心化的未来（文末有专属福利哟！）

成为 MAPO 开发者大使的好处：

  • 成为区块链技术布道者的重要一员
  • 启用你无穷的创造力，为技术社区创作优质内容
  • 第一时间获取区块链技术的最新进展
  • 和同道者一起建设去中心化的未来

🏋️‍♀️ 职责

  • 回答 Discord 社区关于开发相关的问题 
  • 给予 MAPO 开发者社区在 MAP Protocol 开发的支持和引导
  • 共同建设并夸大 MAP Protocol 开发者社区

💰 奖励

  • 试用期间每月可获得 10k 的 MAPO 奖励
  • 试用期结束，成功进入正式技术大使即可获得每月 100k MAPO奖励

🛠 要求

  • 有区块链项目开发经验
  • 能快速掌握 MAP Protocol 白皮书内容
  • 能快速掌握 MAP Protocol MOS 层的内容
  • 熟悉 MAP Protocol 跨链机制
  • 能够每天及时回复开发者的问题，解决开发者疑问，并推动 MAPO 开发者社区的发展

💎 如何申请

  1. 加入 Discord，完成人机认证，第一时间获取相关信息
  2. 填申请表（可以用中文填）

🤑 福利：现在加入 MAPO Discord 即可获得专属奖励！

开发者空投奖励🌟

  1. 加入 Discord，完成人机认证；
  2. 在 #dev-verify 频道中，按照置顶消息的指示，链接你的 GitHub 账号；
  3. 链接好后在 #dev-verify 频道中发送消息，告诉我们你已经成功链接账号；
  4. 我们会在当天下午6:00 前完成验证并通知你发放500 MAPO的空投奖励。

开发者推荐奖励 🌱

  1. 在 MAPO Discord 里建立你的邀请链接；
  2. 分享你的邀请链接给你的开发者朋友；
  3. 在 #dev-referral 频道中发送消息，告诉我们你已经邀请了你的开发者朋友，并注明邀请人数；
  4. 我们会在当天下午6:00 前完成验证并通知你发放200 MAPO的空投奖励，每个成功邀请都可获得200 空投奖励。
