Pi 參加頂尖大學黑客馬拉松！

從 2022年10 月 14 日到 16 日，Pi Core 團隊贊助並參加了在哈佛、康奈爾和加州大學伯克利分校舉辦的三場黑客馬拉松。這種外展是核心團隊在當前封閉網絡時期構建 Pi 的實用程序和基礎設施的努力之一——將 Pi 網絡介紹給熱情的建設者，教育有創造力的年輕開發人員了解 Pi 平台，並邀請和入職下一代開發人員幫助構建Pi 對下一代加密的願景。

為了進一步實現這一使命，參加這些黑客馬拉鬆的學生了解了 Pi Network、其社區和平台 SDK，並在 Pi Developer Platform 上探索和構建了各種 Pi 應用程序原型，以實現他們的創新想法。應用程序包括遊戲、健康應用程序、NFT 應用程序、具有環境影響的應用程序等，它們有助於拓寬可在 Pi 平台上構建的 Pi 應用程序的類型，並展示該平台實現真正實用性的潛力，從而使 Pi 生態系統走向成熟和繁榮。

此外，作為這些黑客馬拉松準備工作的一部分，核心團隊在關鍵 Pi 開發人員資源的開發方面取得了很大進展，並為 Pi 應用程序開發人員發布了更完整的工具。

例如，我們用許多新功能更新了開發人員門戶，例如更清晰、更具指導性的應用程序註冊流程和“應用程序清單”，以幫助開發人員啟動和運行他們的應用程序。您可以在 Pi 瀏覽器的 develop.pi 上親自體驗新的門戶更新。我們發布並改進了演示應用程序和我們的 SDK API，以便開發人員可以更輕鬆地設置和測試 Pi 應用程序。此外，為了改進開發人員的文檔，我們發布了（並不斷改進）新的開發人員入職指南，幫助開發人員了解 Pi 生態系統和在 Pi 上構建。

這些資源不僅可以幫助黑客馬拉松參與者，還可以幫助我們的先驅者和社區開發人員為網絡創建新的 Pi 應用程序和實用程序。

隨著我們繼續我們的倡議，我們希望在接下來的幾週內繼續開發和改進頂級 Hackathon Pi 應用程序。與此同時，以下是一些 Pi Core 團隊成員在這些活動中的表現：
