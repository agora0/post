Keplr 錢包的定位是「跨鏈錢包」，支援多種 Cosmos 生態上的密碼貨幣，包括 ATOM、OSMO、AKT、CRO、IRIS、KAVA 等等，用戶可以一個錢包管理多條不同區塊鏈上的貨幣。若區塊鏈支援 Inter-Blockchain Communication (IBC) 的話，還可實現跨鏈互動。實際案例如最近的 COSMOS 生態上的 Osmosis 去中心交易所的操作，雖然 ATOM 跟 OSMO 是於兩條不同的鏈上發行的幣，但用戶可在 Keplr 傳送 ATOM 到 Osmosis ，再直接換幣或建立流動池等動作，無須先把不同鏈上的幣進行「wrapping」才能操作，方便得很，都是拜 IBC 所帶來的便利所賜。

LikeCoin 目前雖然尚未支援 IBC，但在月內 LikeCoin chain 升級上 FoTan 版本後便會支援，開啟更多應用的可能性。

Liker 自行管理私鑰

過往只能以社交帳號或電郵地址註冊 Liker ID ，原因是降低用戶註冊的門檻，畢竟普羅讀者大多未接觸過區塊鏈。流暢體驗的代價是用戶的私鑰由系統的鑰匙管理系統託管，用戶必須信任託管公司的技術，不然無法使用 Liker ID 相關的服務。

熟悉密碼貨幣操作的進階用戶現在多了一個選項，可以 Keplr 錢包註冊 Liker ID。如此一來用戶可自行管有 LikeCoin 錢包的私鑰，也能使用 Liker Land 的大部份服務。

以 Keplr 註冊 Liker ID 的限制

不介意用戶體驗較䌓複的話，以 Keplr 註冊 Liker ID 是更自主的選項，適合進階用戶；然而以 Keplr 註冊 Liker ID 也有一些限制。

  • 由於 Keplr 是一個瀏覽器的外掛，只能在桌面電腦上的瀏覽器使用，因此無法在手機上使用。
  • Liker Land 手機應用程式也不支援以 Keplr 登入。
  • 以 Keplr 轉帳 LikeCoin 時必須填上收款方的錢包地址，不能只填 Liker ID

雖然 Keplr 用戶無法使用 Liker Land app 委託 LikeCoin，但可以使用 stake.like.co 工具來進行委託及相關操作。

個人資訊的保護

用戶如基於任何考慮，想盡量少在網上留下個人資料的足跡，但又想使用 Liker Land 的服務的話，以 Keplr 註冊 Liker ID 或是個更佳選項。雖然目前系統仍會要求填上電郵地址，但若連電郵也不想透露的話可在註冊時隨便填一個，並不作驗證。這樣，Liker ID 只會聯繫著一個錢包地址而已。然而請留意若不提供電郵地址的話，便無法收到包括轉帳通知、每月報告等系統服務資料，這是無法避免的取捨。
