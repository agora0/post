【附圖：夢字演化 / 其中第一系列爲甲骨文字體，第二列爲金文字體之一，第三系列爲篆文字體，第四列爲隸書字體之一】

мрія = mriya = mr冫與A = mr二與夕 = mmrr與夕 = 目ㄇᒥᒥ與夕 = 罒冖ᒣᒥ與夕 = 罒冖ㄧ丨丨ㄧ與夕 = 罒冖十十與夕 = 罒冖艹與夕 = 夢。此字為名詞，或意通 English 的 dream (夢；做夢；作夢；築夢；想夢；墜入於夢；沉入於夢；夢中；夢的；夢樣)。橋接式中出現的符號 ᒥᒥ 及 ᒣᒥ 也可能與漢字「夢」的甲骨文字體及金文字體中的構字符號相關。(P.S. 參考 【附圖：夢字演化 / ……】)

мрія = mriya = 夢木眼與heart = 夢木目與心 = 夢相與心 = 夢想。也有可能直接轉換自此式「мрія = 夢木眼心 = 夢木目心 = 夢相心 = 夢想」，此時即 м 約略形聲漢語「夢」的起音，р 約略象徵一棵樹木而轉換漢文「木」，i 約略形聲華語「眼」的起音再轉注漢文「目」，я 約略象徵一顆帶有血管的心臟以轉換漢文「心」。

P.S.

мріяти：夢動、夢想動，此時其中的字尾 ти = 重力 = 動，即 т 約略形聲台閩語「重」的起音，и = 丨Λ = 丿ᒣ = 丿㇆ = 力，此 ти 爲動詞的表示；夢作，此時即漢字「作夢」的同義倒裝詞，此時字尾 ти = ㄒI/I = 丅ㄧ㇀ㄧ = 乍 = 作，其中首先轉換的漢字「乍」是漢字「作」的本字，所以可能由「乍」再轉注「作」。此字是動詞或為作詞，字尾 -ти 類似 English 有時也以 -ate (為動；為作) 來表示動詞或為作詞。
