說起俄羅斯，大概絕大多數的人都能馬上聯想到俄羅斯的著名蒸餾酒——伏特加（vodka / 俄語 водка）。這款烈酒宛如已成為俄羅斯在世人心中的代名詞。它既象徵著俄羅斯人發展出來的酒文化，同時也勾起世人對俄羅斯人的負面印象：嚴重的酗酒問題、相對短的預期壽命。

然而，大家所較不知道的是，伏特加不只是俄羅斯的飲食文化、社會問題，它同時也是數百年來鑲嵌在俄羅斯政治中不可分割的一部分。多少俄羅斯的極權統治者用伏特加攏絡、分化屬下，並剝削斂財。在俄羅斯帝國窮盡大陸彼端的擴張行動中，伏特加為其貢獻了不知多少財富，卻也創造了不知多少悲劇。伏特加，俄羅斯人的毒藥，同時也是獨裁者的統治妙術。

網路上用喝醉的俄羅斯人製作的各種迷因（Source: Imgflip）

從「生命之水」到伏特加

現在我們覺得伏特加好像開天闢地以來就是俄羅斯的烈酒，但當我們回到俄羅斯帝國的前身——莫斯科大公國時代（1283-1547），那時並沒有什麼伏特加。雖然俄羅斯人還是好酒，但他們常喝的是啤酒或者其他發酵酒。不過，一個顛覆飲酒歷史的產物卻悄悄地在不知不覺間出現於俄羅斯。當十五、十六世紀初期，莫斯科大公們舉辦酒宴、接待外客時，除了各種本土發酵酒或名貴的葡萄酒以外，他們更可能取出一個透明無色的強烈酒精飲料：「生命之水」（拉丁語 Aqua Vitae / 俄語 живая вода, вода жизни）。

愛喝酒的人肯定會眼睛一亮：原來這就是伏特加的起點！生命之水不就是伏特加的別名嗎？
 
誠然，當今「生命之水」是伏特加的外號，同時也是伏特加中一種極烈的酒種。[1]但生命之水最開始並非伏特加，而是中世紀煉金術對蒸餾酒的代名詞。大概在十二世紀的時候，歐洲——尤其是義大利的煉金術士們——為了開發青春藥水，從希臘古籍和阿拉伯人的技術中復甦了一度存在於歐洲的蒸餾酒技術。歐洲的煉金術士相信這種新興的酒類能帶來數不盡的健康好處。所以，生命之水在當時的歐洲嚴格上來說並不是一種娛樂飲料，而是藥品。我們舉十三世紀的義大利醫師 Arnaldus de Villa Nova 的話來形容生命之水：「燃燒之水……是煉金術師口中的『黃金之水』，它可以治癒許多疾病、延年益壽，所以值得以『生命之水』為人所知。」換言之，生命之水就是當時人們眼中的萬能藥。

Arnaldus de Villa Nova 是十三世紀的義大利醫師，同時也被認為是一位煉金術師（Source: Unknown derivative work by JPS68 / Public domain / via Wikimedia Commons）

那麼，生命之水又是如何從義大利傳到俄羅斯的呢？這段歷史就模糊得多。但很有可能與義大利商人遍佈整個地中海沿岸的商業網路有關。其中一個商業巨城——熱那亞，其商業路線直達克里米亞半島，並在上頭建立了自己的貿易據點及城市。生命之水的技術也就一路從義大利透過熱那亞人的商船來到克里米亞半島，再從克里米亞半島隨著商人、使節，來到了莫斯科大公國的境內。當地人們發現，這種蒸餾酒技術不只可以用葡萄來製作，本地產的穀物也可以製作蒸餾酒。於是，用黑麥、小麥製作而成的伏特加，就悄悄地在歷史的一個陰暗角落中誕生於廣大的東歐地區。在大公與沙皇宮廷中原先頗有異國色彩的「生命之水」，也逐漸代換成了本土的象徵「伏特加」。
 

什麼？伏特加也可以割韭菜？

伏特加出現以後，於十六世紀在俄羅斯一帶迅速擴張。人們發現，伏特加的製作器具簡單、便宜，不在乎品質的話可以輕易製造，為各式各樣的休息日和節慶提供高酒精濃度的狂歡飲料。同時，伏特加的出現正可說是恰逢其時。此時俄羅斯一帶正在經歷農業技術的革新，大量的穀物囤積在倉庫裡面無處可去。於是，大地主、貴族們為了避免賤售穀物，便把大量多餘的穀物做成了伏特加，不但便宜、易保存、方便運送，重點是還很好賣！簡直就是完美解方。
 
不過，也就在此時，政府的手也開始伸進伏特加貿易之中。
 
一切可以說始於這位傳奇沙皇——伊凡四世，又被稱作恐怖伊凡或者伊凡雷帝（Иван Грозный）。他是俄羅斯沙皇國的創建者，擴地無數。當他消滅鄰近的喀山汗國時，他發現汗國建立了一種國營酒館制度：卡巴克（Кабак）。伊凡四世覺得這個制度對政府財政來說極為有利可圖，於是他返國以後做出了一個決定——俄羅斯也要有自己的卡巴克系統。因此，所有的私營酒館以後都為沙皇國禁止。
 
這些酒館的經營者為了要應付沙皇國的重稅、並取得巨額利潤，無所不用其極地鼓勵俄羅斯人喝酒，尤其是伏特加。而當酒客為了喝酒喝到一無所有，連衣服都拿去抵押時，酒保們就會立刻翻臉，無情地將渾身赤裸的酒客（衣服都抵押酒費了）轟出卡巴克，任由他在俄羅斯寒冷的冬天凍死在路邊。卡巴克的主人們因此又被當作是俄羅斯農村的吸血鬼、沙皇國的耳目兼「韭菜收割者」。酒精成癮的農人們對卡巴克主人又恨又怕、但同時又不能沒有伏特加。卡巴克主人儼然成為了小農村內的土皇帝。

俄羅斯人民在寒冷的冬天等待酒館開門（Source: Л.И. Соломаткин, Утро у трактира, 1873 / Public Domain）

卡巴克系統只是俄羅斯政府剝削伏特加利益的第一步。在十七世紀，俄羅斯政府正式決定把所有的私釀伏特加禁止——農人甚至不可以在家私釀伏特加慶祝婚禮。要在婚禮上喝伏特加，請往國營酒館卡巴克。這個壟斷法條出現在 1649 年的法典中，雖然有時候會鬆弛禁令，但大體上在隨後的俄羅斯帝國時期，伏特加都是政府壟斷的商品。

你以為這樣俄羅斯政府就滿意了嗎？太沒效率了！於是政府又進一步引入包稅制系統，將伏特加生意的經營權標給商人，形成官商相護，壓榨百姓的局面。對商人來說，只要完成每年的定額，剩餘的伏特加利潤都可以納入荷包中。對政府來說，這個做法省心省力，再也不用煩惱如何經營伏特加買賣，需要錢的時候，只要調高定額就好。至於商人怎麼樣壓榨顧客，那是他們的事情。
 
隨著莫斯科大公國到俄羅斯帝國的擴張，帝國自然需要數不盡的人力資源和金錢。在十七世紀開始出現的伏特加壟斷制度，為俄羅斯帝國的擴張貢獻良多。在 1680 年，伏特加和鹽產生的稅金竟為俄羅斯帝國貢獻百分之五十三的收入。此後伏特加長期占據俄羅斯帝國、甚至蘇聯時代的政府收入約百分之二十五的份額。如果說英國人靠販賣鴉片給中國人賺取財富，俄國的沙皇們則是讓國民酒精中毒來累積軍費。

十七世紀的俄羅斯出現「酒是人類的敵人」的聲浪（Source: Unknown author / Public Domain）

俄羅斯統御術之「你把我灌醉」

伏特加不僅是俄羅斯國庫的好乳牛（儘管產的牛奶有毒），對於俄羅斯的獨裁者們來說，他同時也是一項控制部屬的政治武器。就這一點而言，大概沒有哪一份紀錄，會比眾多獨裁者之一，蘇聯總書記赫魯雪夫的親口敘述，來得更有說服力了。

赫魯雪夫是史達林的親信、以及接替史達林的總書記。他政治生涯的最後，因為「被退休」而在後任政府的監視下度過老年餘生。不過，赫魯雪夫的退休人生不只釣魚種花，還口述錄下了自己的回憶錄，成為研究蘇聯史的寶貴語音資料。在他的回憶錄中，研究者驚訝地發現赫魯雪夫是如何深深涉入伏特加獨裁政治中——作為被害者。是的，即使是權勢薰天的赫魯雪夫，在史達林還坐鎮克里姆林宮的時代，也只能任由史達林魚肉，而史達林魚肉部屬的方式之一，就是以伏特加為首的酒精飲料。

史達林（右一）熱衷於用灌酒控制部屬，就連親信赫魯雪夫（右二）都無法避免（Source: Unknown author / Public domain / via Wikimedia Commons）

在赫魯雪夫用蒼老的聲音講述的故事中，史達林曾經是一個好老闆。許多他的側近，可以在任何時候，懷著重要的國政問題去拜訪史達林，與之討論。但是，好景不常，很快地史達林成為了一個猜忌部屬、難以接近的領袖。其中一個控制部屬的方式之一，乃是酒宴。無論是小規模的討論會、或者是大規模的酒宴，史達林熱衷於對部屬灌酒。圍繞著史達林的掌權者們常常被灌到不省人事、不得不由他人抬回家裡。
 
史達林自己呢？他是個酒鬼，但在這些灌酒場合中自己卻喝得不多。因為他的眼睛、耳朵，正看著、聽著這些蘇聯的掌權者們在酒醉後的一言一行。在酒精的魔力下，即使是潛藏在內心最深處的小小不滿，乃至陰謀，似乎都可能酒後吐真言。即使赫魯雪夫等人有時想要使些小手段偷偷不喝，但在史達林嚴厲的監視，還有政敵告發的威脅下，即使喝到肝癌暴斃，也不得不繼續喝（實際上真的有人把肝喝壞然後暴斃）。於是，說來有些荒謬，伏特加竟成為獨裁者分而治之的手段。
 
史達林並非利用伏特加統治帝國的特例。早在伊凡四世的時代，第一個沙皇就對這種酒後吐真言的統治術行雲流水。而伏特加妙用不止於此，俄羅斯的歷代女皇，都需要在政變之時，發放伏特加於軍中，以獲取軍隊的衷心擁護。把來訪的外交使節灌醉，更是俄羅斯的傳統「外交禮儀」。史達林本人就強灌過德國、英國等國的使節。至於俄羅斯帝國最為人知的彼得大帝，更是熟練運用其酒豪的能力：他在女兒結婚時，竟把夫婿庫爾蘭大公活生生灌死（他在回去的路上神祕暴斃），讓女兒取得了庫爾蘭公國的統治權。

成也伏特加，敗也伏特加

俄羅斯歷代政府用伏特加剝削人民、充盈國庫、給養軍隊。伏特加宛如是俄羅斯治術的基石。但是，打從最開始，這就是像魔鬼出賣靈魂的交易。俄羅斯為了伏特加治術付出了巨大的代價。遠近皆知的酗酒、壽命問題不談，即使是俄羅斯帝國最熱衷的帝國擴張，也慘遭伏特加反噬：日俄戰爭（1904-1905）時，每當日軍勝利、走入俄軍堡壘或軍營之時，滿地的伏特加酒瓶總是十分顯眼。俄羅斯的軍人們在酒醉的狀態下，迎接了一場又一場的敗北。

醉倒在卡巴克酒館的人（Source: Andrei Petrovich Ryabushkin, Кабак, 1891 / Public domain / via Wikimedia Commons）

俄羅斯歷任政府自然知道伏特加之毒，但卻欲罷不能：四分之一的政府收入，這不是說放棄就能放棄的。即使如此，在數百年的伏特加歷史中，還是有幾個勇者挑戰伏特加問題，其中最著名的兩位是末代沙皇尼古拉二世，以及末代蘇聯總書記戈巴契夫。但包含這兩位名人在內，他們無一例外，慘敗於伏特加手上。

不只是因為財政需求的緣故，依賴伏特加所衍生的問題已經深入俄羅斯政治、經濟和社會的臟腑之中，每當頒布極段的禁酒法令時，就出現大量的私釀商人，甚至出現喝工業酒精的瘋狂民眾，故難以透過極端禁令等「震盪療法」逆轉。然而，徐圖緩進的改革，又撐不過強大的伏特加既得利益者反對。
 
於是，一直到二十一世紀，伏特加的幽靈仍在俄羅斯無邊的大地上，徘徊不去。
