什麼是 Pi Chain Mall ？ - Pi Network 最大的市場。

派鏈商城Pi Chain Mall 電商平台

派鏈商城 Pi Chain Mall (PCM) 是 Pi Network 中最活躍的去中心化應用程序 (Dapps) 之一。 PCM 是一個分散的市場，允許全球用戶網絡使用 Pi 代幣作為貨幣購買、銷售和交易商品和服務。

派鏈商城 Pi Chain Mall 從 Pi Network 開始就一直在建設。 PCM 是 2021 年 10 月首屆 Pi Network 黑客馬拉松的獲勝者。PCM 在黑客馬拉松中被選為排名第二的商業應用程序，僅次於就業招聘市場 Pi Workforce Pool。 PCM 團隊最近通過向社區發布的推文慶祝他們在Twitter獲得了30萬追蹤人數。

Pi Chain Mall 的運作非常簡單。 用戶可以建立個人資料，將其鏈接到他們的 Pi 錢包，然後開始買賣產品。 而商家加入商城亦無需完全KYC； 然而，隨著 Pi Network 走向開放主網，所有成員都將進行 KYC。

PCM 網絡是全球性的，成員從台灣，越南，美國，香港到尼日利亞都有登錄。 根據 Pi Blockexplorer 上的 PCM 支付智能合約，用戶幾乎每時每刻都在交易和從事業務。

“基本上，我們每分鐘都會在區塊鏈上進行交易。 根據每天的交易數量，我們也是最活躍的 Pi 應用程序，”PCM 全球社區經理Edwin告訴 BSC 新聞。 “儘管我們仍處於封閉的主網中，而且大多數先驅者還沒有獲得 KYC。”

電子商務市場存在於 Pi 網絡的封閉主網中。 這對 Pi Chain Mall 用戶造成了一些限制。 Pi Network 用戶不能為任何法令提取他們的 Pi 代幣。 用戶不能鏈接任何其他金融服務並在應用程序中交換除 Pi 以外的任何東西。

“目前，PCM 用戶只能通過銷售產品來賺取 Pi。 但是他們賺取的 Pi 不能提取或用於購買其他東西。 這限制了 Pi 在我們平台上的流通，”PCM 全球社區經理 Edwin 繼續說道。 “不過，PCM將於下月開放π提現。 用戶可以將其從 PCM 應用程序中提取到他們的 pi 主網錢包中。 這對我們來說將是一個具有歷史意義的時刻，pi生態應用的發展將邁上一個新的台階。 隨著 Pi 更好地流通，它們將產生更多價值。”


Pi Chain Mall 團隊與 Pi 生態系統中的許多項目保持著良好的關係。 該團隊還與 Pi 核心團隊的成員保持密切聯繫，可以提出更新請求以改善整體 Pi Network 體驗。

Pi Chain Mall 可通過手機和瀏覽器訪問。 PCM 也可以通過 Pi Network App 和 Pi Browser 訪問。


什麼是 Pi Network：

Pi Network 是一個移動區塊鏈挖礦項目，其使命是讓每個人都能接觸到加密貨幣革命。 該項目由斯坦福大學博士團隊創立。 設計了適合移動設備的區塊鏈挖掘算法的候選人。

Pi Network 經過多年的發展，組織了一次黑客馬拉松，構建了幾個可用的 Pi 區塊鏈應用程序，現在正處於向公共主網過渡的最後階段。
