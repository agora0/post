舉例來說，HOWL 是一個建立在 Juno 上面的微網誌服務，若是 Desmos 更加完善之後，這類應用將會很快在 Desmos 鏈上如雨後春筍出現。

當然，任何有智能合約的鏈理論上都能做到承載社交 dapp 的功能，但就跟 web2 一樣，一個特化的基礎建設能夠省去很多不必要的肥大程式碼，也少了很多需要維護的技術債。

技術上來說，特化的小鏈是比較合理的設計，但大鏈通常有規模效應以及資金優勢，至於以後是大的泛用智能合約鏈會通吃，還是特化鏈會比較受到青睞，我想也只能等待市場驗證了。

驗證人技術更新

首先在日常維護方面搬遷了 Desmos 驗證主機，硬碟改為 NVMe SSD，讀取與寫入更快，同步更順暢，一兩個月以前偶爾會掉 block，現在幾乎是 100% 能出塊的狀態。

再來一個比較大的技術更新是進一步研究 cosmprund，這是個能夠幫助驗證人維護節點的工具。

這次的改變除了打開原本就有的 prune Tendermint SDK 功能之外，更重要的是能針對 Desmos 以及 LikeCoin 等鏈客製化（刪除它們的 key）更有效率客家地使用硬碟空間，也發了 PR 回饋 cosmprund 並 merge 了，感謝 Cosmos TW 群組的各位技術支援，完全是靠大家的幫助才能完成 🙏。

這是你在找的聖杯嗎？

基督教相傳在公元33年，耶穌吩咐11個門徒喝下杯子裡面象徵他的血的紅葡萄酒，這裡使用的杯子就被稱為聖杯

這篇文章講的「聖杯」是基於《Fate》系列作品中，由奈須蘑菇從基督教發想所衍生出來的虛構設定，其俗稱是「萬能的願望機」。

對這個遊戲作品不熟悉不夠宅的讀者，一言以蔽之，這是一個「以為自己在爭奪一個可以實現願望的機器，殊不知都在犧牲自己的靈魂幫別人實現願望」的故事。

它巧妙地反應了現實世界的規則，特別是在投資理財方面，非常有趣。

區塊鏈的敘事總是讓相信它的人們充滿希望、並前仆後繼的吸引人來爭奪，包含什麼都不懂只想跟著賺錢的韭菜、想要來割這些韭菜的人、以及像我一樣，以為韭菜不是自己的人。

這是萬能的願望機，給了人們實現任何願望的可能性，吸引了擁有強烈願望的人前仆後繼地前來奉獻一切。

每當一個區塊鏈式微，不管是分岔了、跑路了，還是被監管了，很多人都會問相同的問題：「這個 token 還會漲嗎？」、「這個 token 還能買嗎？」ETC 如此、LUNA 如此，現在 ETHW 也是如此。

要回答這個問題，得先了解 token 的本質是什麼。

token 就是一個憑證，只要區塊鏈還存在，有人願意投入金錢，它就會有價格，所以才會有很多迷因幣即使毫無用處，仍然有很多人趨之若騖，希望可以透過它達成財富自由的夢想。

能完成什麼事情不重要，這是一個構築極為精巧的系統，只要作為一個稱職的載體，越多人相信，這個系統的力量就越強大。

不論代幣叫做什麼名字，由什麼團隊、用何種技術打造都是其次，這只是用來實現願望的「魔術迴路」，魔力來源需要有人供給，而我們自己就是那些提供「魔力」的人。

也因此，雖然大家都想透過 token 賺錢挖礦，多數人的結果卻都是賠了自己的錢當礦。

追求它的信徒以為存在一個魔法可以違反等價交換的真理，卻不知道自己的靈魂才是驅動願望機的燃料。

聖杯的反身性

可以承載價值的任何東西都具有反身性。反身性是索羅斯著名的理論，指「因果關係之間相互影響，此時兩者都不能被簡單歸類為原因或結果。」

最有名的就是投資市場本身具有反身性：市場趨勢與投資者的心理預期會循環影響，常見的現象諸如「踩著自己飛上天」以及「死亡螺旋」都是反身性的實例，一個往上、一個往下。

反身性使得投資者無法簡單地知道市場是因為價格驅動了預期，還是預期驅動了價格；就像我們不會知道，聖杯是因為有龐大魔力才驅使人們鬥爭，還是因為人們參與了鬥爭，才使得聖杯有龐大魔力。

從另一個角度看，或許它們根本上就是同一件事情。

金鏟子就是聖杯

在區塊鏈的世界中，有個名詞叫做「金鏟子」，顧名思義就是只要買了這把金鏟子挖礦，它就會帶給你財富，而大部分代幣都只是被挖出來的「礦幣」，除了賣掉變現之外毫無用處。

金鏟子的出現

金鏟子的出現，通常都是因為生態中的第一個先行者專案獲取了大量的關注，而不論這個專案實際上成功與否，因為接下來類似的專案都必定會「趨附」在它之上，就搖身一變成了金鏟子。

譬如 Raydium 是提供 Solana 流動性的早期專案，任何 swap 需求都會聚集到 Raydium，因此「上架到 Raydium 與 RAY 做交易對」便成為多數專案的目標，曾經也有一段時間，許多人認為或許 RAY 就是 Solana 上的金鏟子。

然而這樣的榮景在熊市資金撤離的時候一下就崩壞了：RAY 跟其他幣價格一起暴跌、購買 RAY 以及提供流動性的使用者自己變成了礦。

又或者 Solanium 也曾有一小段時間被當做金鏟子，因為持有它才能參加 IDO，當時 Solanium 是做得最好的先行者，新的專案幾乎都會選擇它，因此 Solanium 的價格就水漲船高，然而當越來越多專案做 Launch pad、再加上熊市到來、沒有專案想要 IDO，Solanium 持有者就被套牢，下一波牛市的專案選擇在 Solanium IDO 的機會也不高了。

曇花一現的先行者？

這跟 Web2 的先行者優勢乍看很像，卻又不同。

先行者優勢是建立了護城河，以及培養使用者習慣、建立了網路效應，使得後進者難以與其競爭；然而 Web3 的金鏟子現象更多是來自於吸血鬼攻擊。

吸血鬼攻擊除了使用新的代幣給予大量的早期報酬之外，有時候也藉由空投手法，使得注意力同時疊加在雙方上，暫時性地使得先行者 token 有了優勢。

在 Web2 取得優勢的先行者公司，一直到被打敗之前或許能維持數年以上的領先，而區塊鏈的金鏟子效應未必能存活超過數個月。因為注意力轉移非常容易、使用者可以輕易離開平台；也因為哪裡收益更高就往哪裡去，多數使用者並沒有黏著性。

當然，吸血鬼自己能否存活也是另一個問題。

EVMOS 可能是 Cosmos 生態的下一把金鏟子，理由也很類似：它是 Cosmos 上面 EVM 的先驅者，即使停鏈超過一個月的技術能力受到質疑，它仍然承載了大量的「願望」。

而這些願望就使得新進的 EVM fork 專案們必須空投給 EVMOS 持有者，進一步強化了這樣的「願望」，因此目前持有 EVM 就像是永動機一般，不僅幣價神奇地在熊市上漲、還能期望一直收到其他鏈的空投。

金鏟子的耐久度

牛市的時候大家只談論哪個幣是金鏟子，諸如 BSC 鏈上的CAKE、Solana 上的 RAY，以及後來許多「自稱」是金鏟子的幣，或許 Cosmos 上的 EVMOS 也是。

但在風頭上的時候，從來沒人討論這些金鏟子的「耐久度」，道理很簡單，如果我說這個鏟子不知道何時會斷掉，那敢來買金鏟子的人就變少了，我的金鏟子價值就變低了；所以為了要讓我的金鏟子能活久一點、價值貴一點，持有金鏟子的人通常都不會去提這件事情，只會說「這是金鏟子」吸引大家進場。

區塊鏈的金鏟子之所以看起來會穩賺不賠，就是有源源不絕的韭菜把錢丟了進去相信它是金鏟子，並非金鏟子本身真的有能力「挖」出什麼東西，我們得到的收益全都來自於其他人的投入。

這就跟《Fate》的聖杯如出一轍：人們只被宣傳聖杯是萬能的願望機，而背後沒有說出來的是，聖杯之所以可以實現願望，就是因為它從參與聖杯戰爭的人與從者中吸收了大量的靈魂作為魔力來源。

虛構的設定有助於用中性的角度理解世界

金鏟子也好、聖杯也罷，都完美呼應了 DeFi 第三定律：「當你不知道收益從哪裡來，你就是提供收益的那個人」。

我在《在座的各位都是資金盤》提過一樣的概念，不過這次是用另一個角度切入，希望「聖杯」的比喻有助於讓更多人認為這可以是一個中性的解讀。

華人社會系統都教導我們要用一刀兩斷的黑白模式去思考，不但統治者較容易維持整體系統的安定、人民也比較不需要思考；隨著社群媒體無限制的發展，原本應該存在的光譜也逐漸被極化，兩極就像磁鐵一樣牢牢地吸著，無法抵抗。

而我相信虛構的設定能使我們暫時得以脫離現有的框架、用更純粹的眼光觀察系統，再把這樣的思考模式帶回到我們生活的現實世界，看看關心的事物是趨向光譜哪一邊更多、甚至跳到不同維度去做比較，或許有助於稍微緩解極化的現況。
