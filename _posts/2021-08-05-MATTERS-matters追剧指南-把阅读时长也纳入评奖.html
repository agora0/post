奖金1：人人有奖之多劳多得奖

维持原状，保持不变

X = 前期奖金 + 前传 Matty 多发的 3350
N = 活动结束时收到的投稿数量
每篇文章可得奖金 = X / N


奖金 2：5——5分账之拍手奖

改版后，大家收获的拍手数应该减少了。活动结束时，获得这个奖的用户很可能是改版前的占多数。

但是又想不出什么改进的办法，所以还是以不变应万变，维持原状。如果最后获奖的都是改版前的用户占主流，就当他们积极参加活动，早到早得。

拍手数前50%的文章平分
如果一位用户投稿2+，则用获得拍手数最多的那一篇计算


奖金3：全新积分算法 3612 LikeCoin

这个奖是按照参数计算的。积分最高的20篇文章平分3612。来稿 2+ 的用户，以获得积分最高的那篇文章算。

活动发起时的计算公式，只涉及拍手，支持，讨论，追踪参数。

这个奖改起来很容易，把阅读时长也作为新参数评奖。下调拍手所占比重，增加阅读所占比重，希望可以对改版后的参加者有利：

积分 = 拍手数 X 0.2 + 支持笔数 + 讨论数 X 0.5 + 阅读小时 - 追踪数 X 0.1

积分算法举例：Matters追剧指南：大家一起来破案目前获得

拍手数：211
支持笔数：0
讨论数：35
阅读时长：2.4 小时
作者被追踪：205

当前积分 = 211 X 0.2 + 0 + 35 X 0.5 + 2.4 - 205 X 0.1 = 41.6

参数我随意瞎编的，不一定科学。

奖金123可以重复获得。
