First , update & upgrade Zorin OS system
Zorin OS 安裝檔 .iso 現時的版本是 16.2，更新升級之後會比初始安裝時多出一兩個軟體。
所以，如果要解除安裝，待更新後。

uninstall apps list : ( ternimal : apt remove program_name)
aisleriot cheese deja-dup evolution gnome-mahjongg gnome-mines gnome-sudoku libreoffice* gnome-maps quadrapassel remmina gnome-todo* totem* gnome-weather ibus* brasero
包含有遊戲、辦公室軟體套件、備份、郵箱、遠端控制、影片播放等等。

這種命令列全部貼上的方式解除安裝最快速，不然軟體商店需要點擊每一個而且每次都需要輸入密碼。。。
再三次不斷重新安裝 Zorin OS 之後，膩煩之下，終於找到每一個不需要的軟體的名字。下次直接貼上，一次搞定。

install apps list ：
輸入法，fcitx-rime/anthy
BT軟體，transmission
影片播放，mpv
翻牆，v2rat-desktop ( AppImage version )
電子書閱讀，foliate ( .deb )
本機搜尋，fsearch （ PPA : christian-boxdoerfer/fsearch-stable ）
縮圖，ffmpegthumbnailer

期待着，Zorin OS 17 的來到~
