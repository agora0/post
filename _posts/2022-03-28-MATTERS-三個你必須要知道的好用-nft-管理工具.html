從 2021 年年底持續到今年的 NFT 熱潮，由於有許多名人（例如杰倫熊、余文樂的 Zombie）陸續加入，這波名人效應有效地帶進許多從未交易過加密貨幣的新手，也就是幣圈常說的「出圈」，市場多頭下各式 NFT 項目都一陣瘋搶，不管是努力肝白名單還是快手搶，現在普遍幣圈每人手上應該都會持有好幾個 NFT，管理自己的NFT資產並時時追蹤市場行情就越來越困難。

常見做法是每天到 Opensea 自己的 profile 裡，一個一個點開自己寶貝的 NFT，查看今天項目的地板價（floor price)，才有辦法知道 NFT 當天的漲幅，以及是不是出售的時機點。然而當你擁有的 NFT 數量越變越多，幾乎不可能再每天逐個點擊，此時就很希望 NFT 也能夠有像 Blockfolio (現更名為 FTX) 一樣的 app ，可以方便一個地方快速檢視所有 NFT 資產的稀有度、漲幅、歷史地板價、屬性等，下面是幾個最近發現的好用小工具：

  1. TraitSniper

TraitSniper 作為瀏覽器外掛可以方便直接在 Opensea 網頁上看到每一個 NFT 的稀有度排名（rarity ranking)，同時下方還會顯示這個項目目前的地板價，因此每天可以利用這個工具，打開 Opensea 上自己的 Profile，就能一目瞭然自己有哪個 NFT 目前價格是低於地板價的，即時就能快速調整售價策略。

這個 extension 還有一個好處就是也可以直接到想追蹤的 NFT 項目或帳戶的Opensea 頁面，馬上就可以清楚知道哪些很稀有的NFT售價跟地板價差不太多，這就是一個可以購買的很強訊號，能幫助你快速做判斷，而不用花時間在 rarity.tools 的網站上來回查找。

  • 優點：完全免費且瀏覽器外掛即可使用，不用改變原先網頁瀏覽的行為。
  • 缺點：只能看到稀有度、地板價的資訊，其他額外資訊還是要到項目主頁查看。


2.KryptoGO Wallet

KryptoGO Wallet 主打跨鏈資產管理，最近更納入了 NFT 詳細資料，例如地板價、歷史成交價、屬性等，直接在錢包裡頭就能輕鬆查看，甚至 Etherscan 合約地址、 Opensea 頁面等連結也都有列出，完全可以一個錢包就檢視完所有加密貨幣資產的資訊，相當方便省時。

使用 KryptoGO 錢包還有一個好處，它除了統整 NFT 資訊外，其他加密貨幣資產（目前支援 BTC、ETH、Solana、Tron、Polygon）的漲幅也在錢包首頁可以全部一併追蹤，搭配 DApp 瀏覽器就能即時買賣幣、Swap 甚至是投資 DeFi，算是能滿足多種基本需要的一個 app。

  • 優點：完全免費，介面清楚易懂，只要錢包裡的 NFT 都可以查看詳情。
  • 缺點：目前只支援 ETH 和 Polygon 兩條鏈上的 NFT。

3. Floor

Floor 主打統整你所有 NFT 資產「資訊」在一個 app，良好的資產成效追蹤介面，在首頁就立即知道 NFT 資產今天的總漲跌，點入各個 NFT 則會看到詳細的資訊（Portfolio)，例如地板價、不同期間的平均價格、屬性等，未來應該會再多增加過往交易的紀錄（Activity)。

  • 優點：良好的介面且資訊清楚好閱讀，未來還會提供過往交易歷史的資訊。
  • 缺點：須先購買 mint access token (價格 0.2 ETH) 才能使用 app 功能。


總結來說，TraitSnipper 會非常適合需要「稀有度」的資訊來幫助買賣 NFT的朋友，不但免費好用而且還不用額外學習使用方式。若你是需要一個地方除了可以快速總覽加密貨幣、NFT 資產漲幅，還可以立即透過 DApp 瀏覽器進行買賣，KryptoGO錢包就是最好的選擇，目前使用完全免費，都可以下載來試用看看。最後，若你有一些預算並且很明確是要找能統整 NFT 資產漲幅的工具，就可以考慮使用 Floor。希望這幾個分享的小工具能夠幫助大家更輕鬆管理自己的 NFT，若有其他好用的小工具也歡迎與我分享，也別忘了幫我讚賞或Like喔。
