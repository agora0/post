加密貨幣真正的星辰大海：（也許）是作為真人數據的表示符號。

前信息時代，註意力原始豐饒；信息時代後，註意力進入稀缺螺旋。註意力指的是用於生產數據的人類勞動力。

AIGC 普及後，AI 生產的數據不斷膨脹，然而真人註意力總量存在基本恒定的固有上限，於是，真人註意力生產的數據相對地將愈發稀缺，價值愈發珍貴，這種珍貴會大大超乎當代人的想象。

這可能引發“數據的符號學革命”。

真人數據太過貴重，至少帶來兩個效應：一，需要以精確的價值單位來衡量其交換價值，二，通行的以字符符號表示和傳遞的數據形式無法承載，太容易被截獲、盜取。

貨幣在它是一切之前，首先是符號，貨幣是比字符更優質的符號。人們將會以貨幣形態表示、存儲和傳遞真人數據。

上述只是理論。具體的形態，個人設想，貨幣符號數據也許是長這樣的：

在某一條通用公鏈上存在若幹側鏈，每一條側鏈的 token 被人們約定俗成地視作代表 ASCII 碼的一個字符，然後，存在一種類似 erc-998 的 NFT 標準，允許將多條側鏈上的多種資產以特定順序組合為一枚 composable NFT（cNFT）。

這樣表示的一枚 cNFT，至少具備五種特性：

第一，它可以同時既表意又承載價值；

第二，它具備生產的供應鏈，具有可以核算得非常精確的生產成本，可以自然盈利或虧損；

第三，它可以分解，分解後，所承載的數據信息會徹底消失；

第四，它分解後能以成本價回收，能克服“最後一名接盤者”的龐氏困局；

第五，它無法簡單複製。

這種貨幣符號數據如果作為真人數據的表示符號，會改變非常多的東西。

現在我們上網，向服務器發送字符符號數據，數據可能瞬間繞地球走一圈；如果發送的是貨幣符號數據，這個發送的動作本質上是這樣一個過程：

1、買入字符 token 原料 →

2、用私鑰簽署一個 cNFT 鑄幣合約 →

3、把 cNFT 轉賬到請求你的真人數據的目標地址。

由於數據的生產具有了可精確衡量的貨幣成本（而不像現在，雖然人們會說 data as labour，但數據的成本精算在生產端仍是糊塗賬）那麽，其他的人和公司請求獲取這些真人數據，就需要支付一個精確的價格，於是，真人從事的數據勞動會自然地成長為一種真正的資本主義雇傭勞動（而現在平台經濟中的數據勞動實質上是農奴制經濟）。

這種貨幣符號數據也許能帶來一些奇妙的應用，譬如加密硬件、加密媒體等。隨便擧一下例子：一台加密手機，除了基本的通訊功能外，不可如普通手機那般盡情地連接網路，只能接收和發送貨幣以及請求合約；用這部手機拍攝一張照片，手機在本地根據照片的像素組成，生成若干原料 token 的采購訂單，買入足量原料 token 后，發起合約鑄造此照片的 cNFT。這種照片 NFT 就不是現在”加密藝術“領域常見的那種把圖片鏈接寫在 metadata 裏的 NFT，而是在生產端原生的、可直接表意的、天然具備智慧產權的 NFT，這種媒介形態可稱作真正的”加密媒體“。

除此之外，還可能改變很多，這顯然與許多加密貨幣社區成員能直覺到但又給不出具體圖景的那些宏大展望在體量上是相稱的。當然，這設想還較爲初級，就不意淫太多了，歡迎感興趣的朋友探討、指正。
