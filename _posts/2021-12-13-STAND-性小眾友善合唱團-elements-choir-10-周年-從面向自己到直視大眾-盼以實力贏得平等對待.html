從小童群益會鑽石山中心一個僅 10 人的義工興趣班，到出國公演、與本地 YouTuber 合作，更在疫情下完成公開售票的 10 週年音樂會，Elements Choir（同調合唱團）作為香港少數全男班性小眾友善合唱團，過去 10 年達成了很多個突破舒適圈的「第一次」，伴隨香港同運歷程的高山低谷，亦見證一眾團員的成長和覺醒。正如合唱團踏出鑽石山小小的活動室，團員亦由很多年前的遮遮掩掩、害怕身份曝光變相「出櫃」，到越來越擁抱自己的身份、自信驕傲地參與各種公開表演和宣傳活動，勇於迎向公眾和世界。

很多人聽過 Elements，知道它是小童群益會「性向無限計劃」旗下一個義工小組，卻未必知道 Elements 旗下原來有一隊合唱團叫 Elements Choir。入團八年、在團裡擔任男高音的 Enoch 憶述，2013 年他剛加入 Elements Choir 時，這裡只是一個小小的義工興趣班，十多名初始成員窩在鑽石山蟠龍苑中心的一個小房間練習唱歌，不公開招收團員，亦不會對外演出，是個很隱密低調的小團體。性格害羞又沒有音樂底子的 Enoch 被朋友帶到中心，起初只想一圓中小學未能加入合唱團的遺憾，但聽過全男班合唱團的和聲效果後，他很快被這種「大家做同一件事」的震撼深深吸引，慢慢和其他團員打成一片，越來越投入 Elements Choir 的活動。最令 Enoch 難忘的一次演出，要數 2015 年 Elements Choir 在九龍佑寧堂的表演，那是合唱團成立四年來，首次面對二百多名觀眾、可以邀請親戚朋友「買飛入場」的公開表演；與過去只能在小童群益會內部舉行、團員才能出席的小型表演相比，2015 年是 Elements Choir 踏出龍蟠苑的一小步，亦是開啟日後他們在香港嘗試不同形式的公開表演、走出香港參加海外音樂節的一大步。

歷練間長勇氣　「格仔成員」主動露面

從起初被動的「怕醜仔」，到慢慢認同並為合唱團成員的身份驕傲，Enoch 形容「合唱團一直成長，我都一直成長」。近年他更「上莊」協助合唱團的行政事務，希望讓更多人認識 Elements Choir 這個在香港深耕多年的性小眾友善合唱團。對 Enoch 來說，Elements Choir 團員是個令他驕傲的身份，但回想入團最初，仍是中六學生的他其實未準備好在其他人面前做自己，但在 Elements Choir 的歷練則給了他勇氣：「當時我未完全出櫃，雖然同學老師知，但未去到好 out（公開）。慢慢上台表演，又上過同志遊行嘅台，相都影埋……克服咗心理障礙，就覺得唔係咁可怕！」 Enoch 說，團裡有個資料庫記錄了哪些團員可以露面、哪些團員不想露面，以前 Elements Choir 的一些活動照片、影片甚至要「打格」，以免暴露團員的身份。但創團至今 10 年，他眼見不願露面的團員一年比一年少，大家顯然都有所轉變，更勇敢擁抱自己的身份，甚至曾有團員主動找他，說希望「踏出一步」，拍攝 Elements Choir 的海報照，令他又驚又喜。

2018 年才加入 Elements Choir 的男低音「190」雖然團資較淺，但也曾跟隨合唱團在 2019 年到東京參加同志合唱盛會「手牽手音樂節」，與來自亞洲不同性／別友善的合唱團同台演唱和聯誼。他特別珍惜在出國旅行時，與 Elements Choir 其他團員分房同住，白天唱歌、晚上一起玩，大家更創立了一個名為「家庭」的 WhatsApp 群組，至今仍保持互動。他認為香港雖然有不少合唱團，但很多都與學校或特定團體有連繫，如果不是校友或會員很難加入，而年青、面向性小眾的合唱團就更少了。「190」記得初入團時自己比較內向，不會特別和陌生人聊天，但慢慢熟悉其他團員，明白他們的「眉頭眼額」，大家有了默契就越來越自在。

談到社會上對性小眾的觀感，「190」說自己身邊也有異性戀（直）的朋友，他們不一定對性小眾抱持惡意，但可能會在言談間、調侃別人時不自覺用了「死基佬」、「搞啲咁嘅嘢」等字眼，即使言者無心，聽者也不好受。「190」說自己雖然已經向父親「出櫃」，但父親起初看到 Elements Choir 的全男班陣容，也會有點不理解，甚至問他：「點解全部都係男仔？成班男仔一齊要做咩，點解冇女仔嘅？」「190」憶述這段對話時，自己都覺得好笑：「對我來講，Elements Choir 都係一個 choir 啫，咪唱歌囉！唯有慢慢同佢（父親）講，雖然佢可能都會唔理解。」在百花齊放的網絡時代，性小眾多了很多新鮮的渠道和平台結識同路人，那 Elements Choir 的獨特價值又是甚麼？「190」說坊間確實有很多活動可以聚集一班人，可以一齊玩，但在 Elements Choir 他更能感受自己和夥伴的進步，為共同的目標努力，即使那是微小如「唱到以前唱不到的歌」這種目標：「我們啲人有價值，個 choir 未有價值囉！我們享受件事，件事就會好。」

除了 Enoch 和「190」等一眾團員的多年付出，Elements Choir 能從鑽石山走向世界，發展成一個有三十多名活躍團員、能公開演出的合唱團，幕後推手非指揮暨總監 Albert 莫屬。起初 Albert 只是替朋友「頂更」，自 2013 年開始不時為 Elements Choir 伴奏，慢慢看到合唱團有潛質「可以向前行快少少」，於是越來越多參與伴奏、編曲和指揮的工作，直至 2015 年正式擔任合唱團的指揮。Albert 笑說在朋友邀請他前，他從未聽過 Elements Choir，只知道 Elements 在鑽石山「原來有個咁嘅中心」。當年他第一次來幫手時，Elements Choir 與其說是一個要表演、要走出去的合唱團，反而更像是參加者自助的活動小組，甚至團員會自嘲這裡是一個「rehab center」（復康中心）。Albert 說當時合唱團主力為團員提供一個私隱度高、安全放心的環境聚腳：「無論你有幾脆弱，你都可以投入唱歌呢件事」，但也正因如此，Elements Choir 在 2015 年前從未走出鑽石山龍蟠苑，團員只能在小童群益會內尋找零散的演出機會，非常低調。

「不論性傾向是甚麼　做得到的事不應有不同」

從無到有，從來都是最困難的。過去 10 年，Elements Choir 不斷迎接和挑戰不同的「第一次」：第一次在公眾面前表演、第一次公開招團員、第一次出國表演、第一次和管弦樂團合作、第一次和 YouTuber 合作……各個突破 Albert 都有份參與，那麼他對 Elements Choir 潛質和能力的確信，到底從何而來？他說自己一直對合唱團充滿信心，其實早在 2015 年 Elements Choir 第一次在佑寧堂對着二百多名觀眾公開表演，他已經知道團員能做到他心中對合唱團的理想和期望：「其他人唔會因為你係性小眾，所以你唱得唔好，但我原諒你啦」，沒有歧視，沒有優待，不用卑躬屈膝求別人給予機會，而是憑歌唱實力贏得別人平等的對待和認同。他承認每個團員都帶着各自的理想和期望來到合唱團，有人純粹喜歡唱歌，想精進技巧；有人想尋求突破，組一個不走傳統路線的樂團；有人只是想被認同，想建立聯繫……但懷着甚麼想法不重要，最重要是大家可以聚在一起「一齊做好一件事」。

Albert 的信念很簡單，就是想透過合唱團的形式，證明一個人的性／別身份不是用肉眼就能看出來，亦與一個人的能力、表現無關：「不論性傾向是甚麼，你都值得好的對待，做得到的事不應有不同，能唱的歌不應有分別」。他舉例香港 18 區都有地區合唱團，Elements Choir 不需要是全港最優秀，但至少「唔應該渣過人」。相比起公眾經常標籤 Elements Choir 是一個男同志合唱團，他比較喜歡大家視之為一個性小眾友善的合唱團：「你係乜都唔重要，更遑論你稱呼自己係乜！」他相信若 Elements Choir 能做出成績，完成一場場好的表演，就是讓社會明白性小眾與其他人並無分別的最有力證明。
