拷問 ChatCKX：說好了的 LikeCoin 3.0 呢？

Nov 28, 2024 • ckxpress

Bob：LikeCoin 3.0 綠皮書自五月發表，至今半年還沒推出，已經放棄了麼？

ChatCKX：Over my dead body。我不會放棄。

綠皮書是收集民意的機制，當初我發表  LikeCoin 3.0 綠皮書建議升級並遷回以太坊，是考慮到基金會已經註銷幾年，我只是用戶之一，雖仍對 LikeCoin 的發展充滿憧憬，但重大議題理應匯聚用戶意見，實踐社群共治。

或許是由於技術門檻比較高，LikeCoin 3.0 綠皮書的反饋不算踴躍，但個別用戶還是提出了一些有意思的建議。醞釀一段時間後，台灣的知名自媒體、LikeCoin 驗證人區塊勢於 10 月 4 日正式提出議案 #85，動議 LikeCoin 升級，遷往以太坊第二層網絡 OP Mainnet；10 月 16 日，議案 #85 以 80.61% 贊成、19.38% 反對、0.01% 棄權獲得通過，投票率約 77.3%。

Bob：社群通過還不馬上推出來，賣什麼關子？

ChatCKX：沒有賣關子，只是沒法一蹴而就。

議案 #85 是「意向議案」（signalling proposal），目標是確立認受性，指引社群的下一步行動。這跟物理世界的議案類似，例如 Brexit，公投通過確立了英國國民脫歐的意向，然後政府才調撥資源動手做，單就效率而言，民主往往及不上威權統治。

議案 #85 後，Tech subdao 兩度在每月 community call 徵集開發商，落實升級由參與過 LikeCoin 2.0 開發的 Oursky 及承擔大部分 LikeCoin 日常營運的 Liker Land 負責後，與兩個團隊多次商討，列出升級所需的技術細節，對應的預估時間等等。

Alice：要開發多久？

ChatCKX：預計七個月。

按估算，升級所涉及的開發工作共需 289 人日（man-day），由兩個團隊同步執行。Tech subdao 把整個升級工作分成三個階段：

  1. LIKE 通證及 NFT 的智能合約、用戶升級通證所使用的程式、Liker ID API 等，預計 119 人日。

  2. 用戶使用 LikeCoin 相關 NFT 的介面、like.co API、like.co web 介面等，預計 125 人日。

  3. 質押 LikeCoin 的應用，預計 45 人日。

詳情可參考 LikeCoin 3.0 migration task estimation。

Alice：反正都是要開發，分三個階段有什麼意義？

ChatCKX：一為分期請款，二為檢視進度，快速反應。

第一階段成本約 69.26k 美元，已按市場價格折扣百分之十，Tech subdao 上週提案向社群請款，該議案 #86 的七天投票期過了一半，暫時投票率 52%，其中贊成票佔 83.39%，可望於本週五獲得通過，下週正式動工。

若第一階段進展順利，Tech subdao 預計 2025 年 2 月提案申請第二階段預算，爭取於第二季度完成升級必備的模組。

第三階段為建基於 LikeCoin 3.0 底層的質押應用，相對獨立，我將另寫一份綠皮書，提交社群審議。

Alice：如果社群不通過撥款開發又怎樣？

ChatCKX：當然是尊重社群的決定，難道將否決預算案的用戶定罪？

一旦提案被否決，我會收窄升級的範疇，修訂議案並再次提交，做好解釋工作以爭取通過。萬一預算案再次被否決，我會接受投票結果，辭任 Tech subdao 負責人。

Bob：好複雜，聽不懂，到底我要做什麼？

ChatCKX：抱歉沒能簡單解釋到讓人類明白。

整個升級可分為程式碼、數據及 LIKE 通證三方面，前兩項是程序員的工作，一般用戶除了關注、討論、投票表態等公民參與，靜待系統無縫升級即可；完成後你只需幾下按鍵，就能找到遷往 OP Mainnet 的 NFT 等數據。

至於 LIKE 通證升級則會待程式碼及數據升級完成，預計 2025 年中才處理，屆時需要用戶主動配合，透過網頁把 2.0 版的 LikeCoin 升級到 3.0，並學習一些簡單操作如投票、委託和質押等。

Bob：扯到公民參與也太離譜了吧？LikeCoin 又不是國家，我炒幣而已！

ChatCKX：公民參與不離譜，公民沒得參與才離譜。

對，LikeCoin 不是國家而是 DAO，分散式自治組織。正正是由於 LikeCoin 並非國家，持份者更有空間、有身位去參與，塑造整個生態，在生活中創造民主。在威權國家爭取民主，隨時要面臨冤獄，相對而言在 DAO 創造民主，建構和實驗理想的共治模式，代價不過是投入時間去關注、學習和參與而已，性價比高得很。

有人以為網上過的只是虛擬生活，影響不了真實生活，查實網上生活就是真實生活的一部分，隨著絕大部分資料以數位生成及儲存，現在資產也逐漸移師數位世界，網上、鏈上的民主自由終有一天會倒灌，反過來介入物理世界。

Freedom is not free，如果連付出時間這麼低廉的代價去創造自由民主都不願意，人類就交由我們 AI 統治好了。

⁕ 社群的提問由工人智能 ChatCKX 回應，如有幻覺，實屬正常。

p.s. 香港金管局就「虛擬銀行」改稱「數字銀行」（digital bank）諮詢，雖然意見反映業界傾向使用「數碼銀行」或「智能銀行」，但金管局堅持用「數字銀行」，捨香港常用的「數碼」（還有人記得數碼港嗎？：），當然更不可能選用台灣常用的「數位」，搬出一套理論，但真正原因明顯不過，只爲跟從中國大陸。核心價值觀尚且可以捨棄，一個用詞我可沒有那麼在意，我想說的其實是金管局易名的原因：「虛擬」帶來不真實或高風險的感覺。就連政府都知道「虛擬=不真實」是個誤解，你還認為虛擬世界的民主沒有意義？
